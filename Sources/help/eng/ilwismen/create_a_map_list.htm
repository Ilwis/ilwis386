Create a map list

Create a map list. A map list is for instance required in Sample to locate sample pixels. You can also create a map list of images to display three selected images of your map list as an interactive color composite. When you add raster maps of the same theme but for instance of different years to a map list, i.e. the map list contains a time-series, you can display the map list as a slide show.

A map list can only be created of raster maps that have the same georeference and the same domain.

This dialog box appears:

  • when you choose Create Map List from the File menu in the Main window, or
  • when you double-click the New MapList item in the Operation-list, or
  • when you click the create button in a dialog box which requires a map list as input.

Dialog box options:

Map List name:

Type a new name for the map list.

Description:

Type a description for the map list. The description is visible on the status bar of the Main window when moving the mouse pointer over the map list in the Catalog.

Selectable raster maps:

The left hand list box lists all available raster maps in the current directory.

Select the raster maps (same georeference and same domain) which you want to include in the map list and click the > button.

To select multiple raster maps, press and hold down the Shift key while selecting, see also below.

Selected raster maps:

The right hand list box lists the raster maps that will be included in the map list.

To excluded one or more raster maps from the map list, use the < button, see below.

Tip: For slide shows: you can change the order of maps by dragging individual raster maps up or down in the list.

To add one or more raster maps to the map list:

  • select one or more raster maps in the left hand list box
    (to select multiple raster maps, press and hold down the Shift or the Ctrl key while selecting, see also below);
  • then press this > button.

The raster maps will be added to the bottom of the list in the right hand list box.

To remove one or more raster maps from the map list:

  • select one or more raster maps in the right hand list box
    (to select multiple raster maps, press and hold down the Shift or the Ctrl key while selecting, see also below);
  • then press this < button.

If the selected raster maps use the same domain and the same georeference, the map list is created; it will appear in the Catalog.

Additional information : Multiple selection

To select two or more maps in sequence:

  • click the first raster map you want to select,
  • keep the left mouse button down, and
  • move the mouse pointer to the last raster map you want to select,
  • then release the left mouse button;

or

  • click the first raster map you want to select,
  • press and hold down the Shift key while you click the last raster map in the group.

To select two or more maps out of sequence:

  • press and hold down the Ctrl key while you click each raster map.

To cancel the selection of a map:

  • press and hold down the Ctrl key while you click a selected raster map.
