Layout editor

 File menu

Print

 

Print the contents of the current layout. All map views and annotation that were added to the layout will be sent to the printer.

The options that can be selected in the dialog box depend on the available installed printer(s) and on the selected default printer for your layouts. For more information, refer to the Page Setup dialog box.

Dialog box options:

Printer:

 

Name:

Select the printer that you wish to use to print this layout.

For your information, the status, type and server name of the printer are shown.

Print to file:

Select this check box if you wish to print this layout to a file, e.g. a Postscript file.

Clear this check box if you wish to print this layout directly on the selected printer.

Properties button:

Inspect the properties of the selected printer.

Print range:

All:

You can only print the entire layout.

Copies:

Nr. of copies:

Specify the number of copies you wish to obtain.

Collate:

This check box currently has no function.

Tip:

  • To choose a default printer, select paper size and orientation of the paper, etc., see the Page Setup dialog box.

Printing to a Postscript file (.PS):

  1. To 'export' a layout to Postscript format:
    • From the Windows Start menu, choose Settings, Printers. Double-click the Add Printer icon.
    • Select Local Printer.
    • Choose any Postscript printer.
    • Instead of choosing a printer port, choose FILE:
    • Type a name by which you want to recognize this 'printer'.
    • Do not ask for a test page.
  2. Create a layout to which you add your map views, and annotation, and in the layout editor :
    • Choose File Print,
    • Select the Postscript 'printer' to File.
    • Print.
