Point editor

 File menu

Save   
 

 

Save all changes in the point map until this moment, so that they cannot be undone anymore by using the Undo All Changes command.

Using this Save command is optional, as any change is directly and immediately stored. Thus, you can also simply close the editor and not use the Save command.

When using the Save command:

  • the point map will be saved as it is at the moment;
  • any changes you made to this point map prior to using this Save command cannot be undone anymore through the Undo All Changes command;
  • only changes you make to the point map after using the Save command can be undone by using the Undo All Changes command.
