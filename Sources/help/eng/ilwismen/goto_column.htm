Table window

 Columns menu

Goto Column

 

Goto a specific column in the table.

Dialog box options:

Column:

Select a column in the table (open the list box by clicking it).

 

  • The selected column appears at as the first column in the table.
  • The order of the columns in the table remains the same.

Tip:

To change the order of columns, use Column Management, or drag a column header to another position in the table.
