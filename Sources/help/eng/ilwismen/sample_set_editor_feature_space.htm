Feature Space
 

Show a feature space of two bands. A feature space gives a visual overview of the separation of classes of training pixels. The spectral values of one map are put along an horizontal axis, and the spectral values of another map are put along the vertical axis. At the position of a sampled pixel, the color of the assigned class appears.

You can open as many feature spaces as you like, but as each of them is updated after a new selection of training pixels, speed may decrease. Currently selected pixels appear black in a feature space. Feature spaces can be shown with or without additional grid lines, can be copied to clipboard and printed.

Dialog box options:

Band 1:

Select the map to be used for the horizontal axis of the feature space.

Band 2:

Select the map to be used for the vertical axis of the feature space.

The feature space appears in a separate window; you can size the window by dragging its borders.
