SMCE window

 Generate menu

Histogram   
 

 

Show the histogram(s) of the currently selected SMCE output map(s).

 

  • When you select the main goal and press the Histogram button, the histograms of the SMCE output maps will be calculated and shown.
  • When you select a sub goal with named output maps and press the Histogram button, the histograms of the sub goal output maps will be calculated and shown.
  • When you select a single output map and press the Histogram button, the histogram of the selected map will be calculated and shown.

By inspecting the histograms of all alternative output maps, you can get an idea of the distribution of values in the SMCE output maps, and on desired class boundary values for Slicing.

After selecting the main goal, a sub goal or a single map in the criteria tree viewer, you can use:

  • the Histogram button in the toolbar,
  • the Histogram command from the Generate menu, or
  • the Histogram command from the context-sensitive menu.

Tips:

  • By inspecting the histograms, you may be able to decide on the values that should be used as boundary values in Slicing.
  • In the histograms, you can inspect the minimum, maximum, sum and average (mean) value for each SMCE output map individually.
  • By using Aggregate Values, you can inspect the minimum, maximum, sum or (weighted) average value for all alternative SMCE output maps in one go.

Notes:

  • Histogram is not available when the SMCE output maps have not been generated.
  • Histogram will only calculate the histograms of SMCE output maps that are already generated.
  • Histogram will only calculate the histogram of an SMCE output map if the histogram is not available yet.
