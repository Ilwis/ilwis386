Representation Class editor

Column Width

 View menu
 

Change the width used to display colors and class names in the Representation Class editor.

Dialog box option:

Column width:

Type a value for the column width used to display colors and class names.

The smaller you choose the column width, the more classes are shown in the Representation Class editor, but class names may become unreadable.

The larger you choose the column width, the more space is used to display colors and class names, but you might end up with very few classes in the window.

Minimum column width is 20, maximum column width is 250.

The column width is stored in the representation.
