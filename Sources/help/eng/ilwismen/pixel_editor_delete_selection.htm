Pixel editor

Edit menu

Delete/Clear
 

Clear the class names, IDs or values of selected defined pixels, so that the selected pixels become undefined pixels.

To delete/clear:

Select one or more pixels, and:

  • choose Delete from the Edit menu, or
  • choose Delete from the context-sensitive menu, or
  • press Del on the keyboard.

A message box will appear asking you to confirm clearing of the selected pixels.

Answer with:

  • yes: to clear the selection;
  • no: to return to the editor without changes.

 

Mind: There is no possibility to recover the class names, IDs, or values of any cleared pixels.
