Graph window

 File menu

Open table

 

Directly opens the table from which this graph has been created.

Tip:

To open an existing graph from within a table window:

  • open the Properties of the table via the File menu of the table window;
  • go to the Used by tab in the table properties sheet;
  • in the Used by list box, select the graph you wish to view with the right mouse button, and choose Open from the context-sensitive menu.
