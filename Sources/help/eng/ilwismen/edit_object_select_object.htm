Select object to be edited

Select a map, or table or any other object, open it and edit it.

Dialog box options:

Select an object:

Select the object which you want to edit.

Select object type:

Select the type of object you want to edit: a map, a table, a map list, a domain, a representation, a georeference, a coordinate system, etc.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

Depending on the selected object, an editor or a dialog box appears:

Raster map:

The Pixel editor is started.

Polygon map:

The Polygon editor is started.

Segment map:

The Segment editor is started.

Point map:

The Point editor is started.

Table:

The table is displayed in a table window. Fields of source columns can directly be edited.

Two-dimensional table:

The two-dimensional table is displayed in a table window. Fields in the table can directly be edited.

Map list:

The map list is opened in a kind of Catalog which has some extra possibilities. See How to edit a map list.

Map view:

The map view is displayed in a map window. You can add data and annotation layers as usual.

Domain:

For a class or identifier domain, the Domain Class/ID editor is started.

For a value domain, the Edit Domain Value dialog box appears.

Representation:

For a representation linked to a class domain, the Representation Class editor is started.

For a representation linked to a value domain: the Representation Value/Gradual editor is started.

Georeference:

For a georeference corners, the Edit Georeference Corners dialog box appears.

For a georeference tiepoints, the Georeference Tiepoints editor is started.

For a georeference 3D, the Georeference 3D editor is opened.

Coordinate system:

For a coordinate system boundary only, the Edit Coordinate System Boundary Only dialog box will appear.

For a coordinate system projection, the Edit Coordinate System Projection dialog box will appear.

For a coordinate system latlon, the Edit Coordinate System LatLon dialog box will appear.

For a coordinate system formula, the Edit Coordinate System Formula dialog box will appear.

For a coordinate system tiepoints, the Coordinate system Tiepoints editor will be opened.

Histogram:

The histogram is displayed in a table window. Histogram columns cannot be edited, but you can perform calculations using the histogram columns.

Sample set:

The Sample set editor is started.

Matrix:

The matrix is displayed. You cannot edit the matrix.

Filter:

The Filter editor is started.

Function:

The Function editor is started.

Script:

The Script editor is started.

Tips:

  1. To edit a map or a map list, you can also click the right mouse button on the map or map list in the Catalog, and choose Edit from the context-sensitive menu.
  2. In a map window, you can also choose Edit Layer from the Edit menu and select the layer which you want to edit; the appropriate map editor is opened.
  3. To edit a table, domain, a representation, a georeference or a coordinate system, it is generally easier to double-click it in the Catalog. The object is opened and you can directly edit it.
  4. To edit a table, domain, representation, georeference, coordinate system, map view, sample set, filter, function or script, you can also click it with the right mouse button in the Catalog, and then choose Open from the context-sensitive menu.
  5. To select the type of objects that are displayed in the Catalog, click the Customize Catalog button in the toolbar of the Main window.
  6. Dependent data objects and objects marked as read-only cannot be edited.
