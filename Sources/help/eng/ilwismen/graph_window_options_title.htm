Graph window

 Graph Management

Title
 

Modify the title of the graph as displayed at the top of the graph window.

This dialog box appears:

  • when you double-click the Graph Title item in the Graph Management pane,
  • when you double-click the title of the graph as displayed at the top of the graph in the graph window itself.

Dialog box options:

Title:

Type the text that should at the top of your graph.

Font name:

Select the font in which the title of the graph should be displayed. You can select any installed True Type font.

Font size:

Type a value for the font size (pt) in which the title of the graph should appear.

Bold:

Select this check box to have the title of the graph appear in bold.

Italic:

Select this check box to have the title of the graph appear in italics.
