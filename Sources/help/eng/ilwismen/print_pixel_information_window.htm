Pixel info window

Print Options

 File menu
 

Prints the contents of the pixel information window on the default printer.

Dialog box options:

Header on all pages:

Select this check box to obtain a header on each page.

Clear this check box when you do not want to obtain a header on the print out(s).

Leftmost column on all pages:

Select this check box to print the name of the maps, and attribute columns as appearing in the leftmost part of the pixel info window. This will appear to the left of any printed columns and fields.

Clear this check box when you only wish to print fields.

Font Name:

Select one of the fonts from the list of installed fonts.

Font Size:

Type a font size for printing.

Bold:

Select this check box when you wish to print columns/fields in Bold.

Italics:

Select this check box when you wish to print columns/fields in Italics.

All:

When you made a selection in the pixel info window, select this option to print the entire pixel info window.

Current selection:

When you made a selection in the pixel info window, select this option to print the selected part of the pixel info window only.

When you click OK, a standard Windows Print dialog box will follow in which you can select a printer and specify some settings for the selected printer.

When you click OK in the Windows Print dialog box, the contents of the pixel info window will be printed.

 

Tip: To install a printer and choose a default printer, see your Windows documentation.
