Edit fields in multiple columns
 

Edit the values of one or more selected records, or edit a block of selected fields in multiple columns in a table.

To edit one or more fields in a single column, see Edit fields in a single column.

Notes:

  • The contents of fields of dependent columns cannot be edited, as these 'values' are the result of a calculation or a column operation.
  • The contents of fields of read-only columns and table-owned columns cannot be edited.
  • All changes are directly stored on disk. You CANNOT UNDO an edit action.

To edit the fields of a single record:

  1. Click the record you wish to edit: the record view window will appear.

    2. You can drag the record view window to another position.

  2. Then, either:
    • choose Edit from the Edit menu, or
    • click the selection with the right mouse button, and choose Edit from the context-sensitive menu.
  3. This Edit dialog box will appear:

    4. Edit record ...

    Specifies the selected record.

    Column name:

    For the field of the first column in the record, type a value, choose a class, or type an ID.

    Column name:

    For the field of the second column in the record, choose a class, or type an ID. Etc.

    Column name:

    For the field of the next column in the record, choose a class, or type an ID. Etc.

    Fields which are not editable are automatically grayed out.

To edit the fields of multiple records:

  1. Select the sequence of records that you wish to edit e.g. by keeping the left mouse button down while moving over the record buttons of the records that you wish to edit.

    2. The record view window will appear. You can drag the record view window to another position.

  2. Then, either:
    • choose Edit from the Edit menu, or
    • click the selection with the right mouse button, and choose Edit from the context-sensitive menu.
  3. The Edit dialog box as described above will appear for the first record that is selected.

    4. Type a value, select a class, or type an ID for each field in the record.

  4. After you click OK in the first Edit dialog box, the Edit dialog box for the next selected record will appear.
  5. When you click OK in the Edit dialog box of the last record that is selected, you will return to the table window.

To edit a block of fields in multiple columns in multiple selected records:

  1. Select the block of fields, in multiple columns and in multiple records, that you wish to edit.

    2. The record view window will appear. You can drag the record view window to another position.

  2. Follow steps 2-5 described in the previous section. The Edit dialog box will appear record by record, and it will only contain the column names of the selected fields.

Note:

In the record view window, you can only edit fields one by one.
