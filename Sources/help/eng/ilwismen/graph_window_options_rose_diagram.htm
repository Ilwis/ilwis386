Graph Options - Rose Diagram
 

Modify the name and/or the appearance (i.e. display options) of the selected rose diagram.

The option Rose Diagram is designed to display the output of the Segment direction histogram operation in graphical format.

This dialog box appears:

  • when you double-click an existing rose diagram in the Graph Management pane,
  • when you click an existing rose diagram with the right mouse button in the Graph Management pane, and select Graph Options from the context-sensitive menu.

Dialog box options:

Name:

Optionally, modify the name of this rose diagram. The name of the graph layer is visible in the Graph Management pane and in the legend of the graph window.

Line:

Select this option to connect the value of a record (Y values) with the value of the next record through a straight line.

Step:

Select this option to connect the value of a record (Y values) with the value of the next record through a step-wise line.

Needle:

Select this option to obtain a separate line in the rose diagram for every record (Y values) in the table. Every line will at 0,0.

Color:

Select a color in which the rose diagram should be displayed.

To define your own color, click the little create button or select the 'Custom' item at the bottom of the list; then, the Windows Color dialog box will appear.

Tips:

  • To adapt the title, X-axis, Y-axis of the rose diagram, double-click the appriopriate item in the Graph Management pane or in the graph window itself: the Title, X-axis (rose diagram), or the Y-axis (rose diagram) dialog box will appear.

    • The grid lines of the rose diagram can be modified in the X-axis and Y-axis dialog boxes.

  • To add another rose diagram to the current graph window, choose Add Graph from the Edit menu in the graph window or click the Add Graph button in the toolbar of the graph window; the Add Rose Diagram dialog box will appear.
  • To create a new graph in a new graph window, choose Create Graph from the File menu in the graph window: the Create Graph dialog box will appear.
  • To save the contents of a graph window as a graph, choose Save or Save As from the File menu in the graph window.
