Point editor

 Edit menu

Cut
 

Copy selected points with their coordinates and class names, identifiers or values to the clipboard, and then delete the selected points from the current point map.

Subsequently, the copied points can be pasted as points into another point editor, as one segment into a segment editor, and/or into any other application which uses the text format (e.g. a word processor or a data base).

To cut:

Select one or more points, and:

  • choose Cut from the Edit menu, or
  • press Ctrl+X on the keyboard.

For more information on selecting points, see Select Mode.

A message box will appear asking you to confirm the deletion of the selected points. Answer with:

  • yes: to delete the selection;
  • no: to not delete the selected points.

Tips:

  • To recover points which were accidentally deleted, you can use:
    • Undo all changes: to restore all changes made in the point editor until the last time you used Save.
  • You can use Windows' Clipboard/Clipbook Viewer to inspect the copied points on the clipboard in text format. For more information on the Clipboard Viewer, see the Windows Help.
  • An example of points copied to the clipboard is presented in Point editor : Copy.
