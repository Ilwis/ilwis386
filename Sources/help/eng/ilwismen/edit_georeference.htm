View/Edit Georeference
 

View and/or edit the georeference of the raster map that is currently displayed in the map window.

Depending on the type of georeference which is used by the raster map in the map window, a dialog box and/or an editor will be opened:

  • georeference corners (rasterized vector maps): the Edit Georeference Corners dialog box will appear;
  • georeference tiepoints (satellite images or scanned photograph without DTM), or
    georeference direct linear (scanned photograph without fiducial marks), or
    georeference orthophoto (scanned photograph with fiducial marks):
    the Georeference Tiepoints editor will be opened.
    When the raster map uses a georeference orthophoto, the Locate Fiducial Marks dialog box is opened first;
  • georeference 3D: the Georeference 3D editor is opened in the form of an Georeference 3D : Edit dialog box in which you can specify parameters for your 3D view.

This option is not available when the map window only displays polygon, segment and/or point maps. Further, this option is not available for raster maps that have a georeference which is marked read-only, nor for raster maps that use a georeference submap, georeference mirrorrotate, or a georeference factor.

Tip:

To view and/or edit a georeference, you can also double-click it in the Catalog. To make georeferences visible in the Catalog, refer to How to customize the Catalog. For more information, see also How to edit a georeference.
