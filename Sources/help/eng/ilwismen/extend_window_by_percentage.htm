Map window

 Options menu

Extend Window by Percentage
 

Increase or decrease the size of the current map window. The original height and width of the map window are considered 100%.

 

  • If you increase the size of the map window, extra space will be added to the map window.
  • If you decrease the size of the map window, only part of your map will be shown.

Only the size of the current map window will be changed, i.e. display; the size of map(s) in the map window will remain the same. You can save the new situation of the map window, by saving the map window as a map view.

Dialog box options:

Percentage at top:

Type a value for the percentage that should be added to the top of the map window.

Percentage at bottom:

Type a value for the percentage that should be added to the bottom of the map window.

Percentage at left:

Type a value for the percentage that should be added to the left of the map window.

Percentage at right:

Type a value for the percentage that should be added to the right of the map window.

 

Tip: To permanently change the size or outer coordinates of a map, use the Sub-map operations.
