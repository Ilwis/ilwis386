 Edit menu

Enter the number of alternatives

 

Enter or edit the name(s) of the alternative(s) and/or change the number of alternatives. Number of alternatives is one when performing Problem Analysis or Design of Alternatives, and more than one when performing Decision Making.

The Enter the number of alternatives dialog box appears:

  • when you choose Alternatives from the Edit menu in the SMCE window;
  • when you click the Alternatives button in the toolbar of the SMCE window;
  • when you click one of the grey column headers at the upper part of the criteria tree editor.

Dialog box options:

Number of alternatives:

Select the number of alternatives [1,9]. By default the number of alternatives is 1 when performing Problem Analysis or Design of Alternatives and 2 when performing Decision Making.

Alternative 1:

Type a name for Alternative 1.

Alternative 2..9:

Optional. Type a name for Alternative 2..9.

Note:

  • Changing the number and names of alternatives is only possible in the Problem Definition mode.
  • Typing a name for an alternative is only possible if the number of alternatives is more than one.
