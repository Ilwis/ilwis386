Select mode   
 

 

Choose Select mode to select one or more points with the mouse.

After the selection, you can edit the code of the point(s), delete the point(s), or cut or copy the point(s) to clipboard.

When Select mode is active:

  • the mouse pointer appears as a hand,
  • an option bullet appears in front of this menu command, and
  • the Select mode button in the toolbar appears down.

 

Tip: You can also press the Esc key on the keyboard to go to Select mode.

To select a single point:

  • click the point you wish to select.

 

Tip: When you double-click a point, you can directly edit the value, class name or ID of the point.

To select all points within a rectangular area:

  • drag a box around the points of interest.

To add points to a selection:

  • press and hold down the Ctrl key on the keyboard,
  • click the points you wish to select.

When finished, release the Ctrl key.

To clear part of a selection:

  • press and hold down the Ctrl+Shift keys on the keyboard,
  • click the selected points that should not be part of the selection.

To toggle between selecting and deselecting:

  • press and hold down the Shift key on the keyboard,
  • when you click a non-selected point, it will be selected,
  • when you click a selected point, it will be cleared from the selection.

To clear the whole selection:

  • press Esc on the keyboard, or
  • click in the white space between points.

Tips:

  • To select all points, choose Select All from the Edit menu.
  • To easily recognize selected points, you can increase the symbol size or change the symbol of all points in the Customize dialog box.
