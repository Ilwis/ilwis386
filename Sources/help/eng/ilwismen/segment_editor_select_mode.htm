Segment editor

 Edit menu

Select mode   
 

 

Choose Select mode to select one or more segments with the mouse.

After the selection, you can edit the code of the segment(s), delete the segment(s), or cut or copy the segment(s) to clipboard.

When Select mode is active:

  • the mouse pointer appears as a hand,
  • an option bullet appears in front of this menu command, and
  • the Select mode button in the toolbar appears down.

 

Tip: You can also press the Esc key on the keyboard to go to Select mode.

To select a single segment:

  • click the segment you wish to select.

 

Tip: When you double-click a segment, you can directly edit the value, class name or ID of the segment.

To add segments to a selection:

  • press and hold down the Ctrl key on the keyboard,
  • click the segments you wish to select.

When finished, release the Ctrl key.

To clear part of a selection:

  • press and hold down the Ctrl+Shift keys on the keyboard,
  • click the selected segments that should not be part of the selection.

To toggle between selecting and deselecting:

  • press and hold down the Shift key on the keyboard,
  • when you click a non-selected segment, it will be selected,
  • when you click a selected segment, it will be cleared from the selection.

To clear the whole selection:

  • click in the white space between segments.

 

Tip: To select all segments, choose Select All from the Edit menu.
