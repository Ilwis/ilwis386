Layout editor

 context-sensitive menu

Save View As ...
 

Save the contents of the currently selected map view in this layout with another name.

All layers in the map view and their display settings as well as the scale and size of the map view are stored.

You can use this command:

  • when you added or removed layers to a map view in the layout editor, or
  • when you changed display options of layers in a map view in a layout.

This dialog box appears:

  • when you select a changed map view in the layout, and choose Save View As from the context-sensitive menu.

Dialog box options:

Map View Name:

Type a name for the map view.

Title:

Type a title (=description) for the map view.

Tip:

To save a layout, for instance when you replaced a map view in a layout with another map view (see Replace Map View):

  • click somewhere in the white space of the layout, and
  • click the Save button in the layout toolbar, or
  • choose Save or Save As from the File menu of the layout editor.
