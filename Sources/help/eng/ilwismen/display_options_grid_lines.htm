Display Options - Grid Lines
 

Specify display options for grid lines to be displayed in a map window.

Grid lines are created based on the georeference that is used by a displayed raster map, or when no raster map is displayed, on the coordinate system used by the displayed vector map(s).

Dialog box options:

Grid distance:

Type a value for the spacing of grid lines.

It is not recommended to use a grid spacing less than the pixel size of a raster map.

Line type:

Select a line type in which the grid lines should be drawn: Single, Dot, Dash, Dash dot, or Dash dot dot. Grid lines will always be drawn as fine as possible, i.e. using one pixel on the screen and using one dot on the printer.

Color:

Select a color for the grid lines. To define your own color, select the 'Custom' item at the bottom of the list or click the create button; the Custom Colors dialog box will appear.

Scale limits:

Select this check box when the grid lines should be displayed or hidden depending on the scale specified in the scale control of the map window; specify the scale range within which the grid lines should be displayed. The grid lines will be automatically hidden if the scale of the map window is outside this range.

With this option you can achieve for instance, that certain grid lines are only shown when using a large scale in a map window, and/or that other grid lines (other spacing) are only shown when using a small scales in a map window, etc.

Clear this check box when you wish to display the grid lines regardless of the scale of the map window.

Tip:

  • To store the contents of a map window, you need to save the map window as a map view. Open the File menu of the map window and choose the Save View or the Save View As command.
