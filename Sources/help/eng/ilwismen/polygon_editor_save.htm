Polygon editor

 File menu

Save   
 

 

Save all changes in the polygon map until this moment, so that they cannot be undone anymore by using the Undo all changes command.

Using this Save command is optional, as any change is directly and immediately stored. Thus, you can also simply close the editor and not use the Save command.

When using the Save command:

  • the polygon map will be saved as it is at the moment;
  • any changes you made to this polygon map prior to using this Save command cannot be undone anymore through the Undo all changes command;
  • only changes you make to the polygon map after using the Save command can be undone by using the Undo all changes command.
