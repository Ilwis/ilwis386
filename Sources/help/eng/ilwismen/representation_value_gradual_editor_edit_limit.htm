Representation Value/Gradual editor

Edit Limit

 Edit menu
 

Edit the value of a selected limit (or break point) and/or the color of this selected limit (or break point) in this representation value or representation gradual.

 

  • When you are editing a representation value (i.e. the representation is linked to any value domain except system domain Value), the editor works in values.
  • When you are editing a representation gradual (i.e. the representation is linked to system domain Value), the editor works in percentages.

This dialog box appears:

  • when you double-click a value or percentage limit in the representation window, or
  • when you select a value or percentage limit in the representation window with the mouse pointer and select Edit Limit from the Edit menu, or
  • when you select a value or percentage limit in the representation window by using the Arrow Up and Arrow Down keys on the keyboard, and then press Enter.

Dialog box options:

Value:

For a representation value, type another value between the minimum and maximum value of this domain.

For a representation gradual, type another value between 0 and 100%.

Color:

Select another color for this limit (break point); open the list box by clicking it. To create a custom color, you can click the create button or click the 'Custom' item at the bottom of the color list.

The color bar at the bottom of the representation editor shows the colors of the entire representation.

Tip 1:

For the colors in between two limits (or break points), there are three possiblities:

  • values in between two limits are stretched between the colors of the upper and lower limit (Stretch), or
  • values in between two limits get the same color as the upper limit (Upper), or
  • values in between two limits get the same color as the lower limit (Lower).

 

To edit the colors of values in between two limits:

  • double-click on the words Stretch, Upper, Lower that appear in between two limits in the representation editor window, or.
  • select one of the words Stretch, Upper, Lower as they appear in between two limits in the representation editor window and press the Enter key.

When the color method in between limits was Stretch, then Upper is assigned;

when it was Upper, then Lower is assigned;

when it was Lower, then Stretch is assigned. This is a cyclic choice.

 

To change the number of colors used for values in between two limits in case of Stretch, choose the Color Steps command from the Edit menu.

Tip 2:

To interactively edit the colors of a value map,

  1. Display the value map in a map window,
  2. Open the map's representation by selecting Representation from the Edit menu in the map window,
  3. In the Representation Value/Gradual editor, edit colors, insert limits, etc.
  4. Keep the editor open and switch to the map window,
  5. Click the Redraw button in the map window to apply the changes.
  6. Switch to the Representation editor to continue to insert limits and changing colors and switch to the map window and click the Redraw button to apply the changes.

 

When you displayed a value map in the map window and you selected a system representation, like Gray, Pseudo, etc., you cannot edit the representation. Instead, in the map's Display Options dialog box, click the create button next to the Representation list box to create a new representation. After the Representation editor has opened, you can interactively edit the new representation of the map.
