Table window

 Edit menu

Add Record

 

Add a single new record to a table which uses system domain None.

The new record will appear at the bottom of the table.

Tips:

  • To add a record to an attribute table, you should add a new class name or ID to the Class or ID domain that is used by the table and by the map. The new domain item is then directly shown in the table.
  • When pasting into a table with system domain None, and when nothing is selected in the table, then the clipboard contents will be appended in new records at the bottom of the table. For more information, refer to Table window : Paste.

Command line syntax:

If you like, you can also add one or more records to a table with system domain None, by typing a command on the command line of a table window:

 

addrecord

to add a single record to the table.

addrecord nr

to add a specified number of records to the table.
