Map list as slide show
 

Specify how fast all raster maps in a map list should be displayed in the map window as a slide show. Showing a map list as a slide show is especially suitable when a map list contains a time series.

This dialog box appears:

  • when you started a slide show and subsequently specified the Display Options of one of the raster maps in the map list.

Dialog box options:

Refresh Rate (/min):

Select the number of raster maps you want to see per minute.

Display Options:

Click this button to return to the Display Options - Raster Map dialog box to specify or change the display options for all raster maps in the map list. In the Display Options Raster dialog box, the information is shown about the first or the currently displayed map.

When everything is OK, the slide show is started.

Tips:

  • When the refresh rate is set to zero, it means that the maps will not be refreshed automatically. Press the Redraw button in the toolbar of the map window (or Ctrl+R) to show the next slide.
  • When the map list contains value raster maps and you are not satisfied with the selected stretch range that is applied on all maps in the map list when running the slide show, you can easily adjust this stretch range by pressing the Display Options button in this Display Options - Map List as Slide Show dialog box.
