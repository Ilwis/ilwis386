Annotation Text editor

Add Text / Edit Text

 Edit menu
 

Add a new text item to the Annotation Text object, or edit the text and settings of one selected text item.

This dialog box appears:

  • when you choose the Add Text command from the Edit menu,
  • when you click the right mouse button in the editor window and choose Add Text from the context-sensitive menu,
  • when you double-click at a new position in the editor window,
  • when you double-click a text item in the editor window,
  • when you select one text item and choose Edit from the Edit menu or from the context-sensitive menu.

When you select an existing text item in the editor window, the selected item will appear highlighted.

Dialog box options:

Text box:

Type a new text or modify the existing text. The text box has no limitation on the number of characters that you can enter.

X Y:

When working with XY-coordinates, type X- and Y-coordinates where the text should appear or modify the XY-coordinate of the existing text.

Row Col:

When working with RowCols, type Row and Column numbers where the text should appear or modify the RowCols of the existing text.

Font name:

Select a font in which the text should appear. Open the list box by clicking it. You can select any installed True Type font.

Font size:

Type a value for the font size in which the text should appear.

Bold:

Select this check box when the text should appear in Bold.

Italic:

Select this check box when the text should appear in Italics.

Underline:

Select this check box when the text should appear Underlined.

Justification:

Select a position for the justification mark of this text. Justification Baseline Center gives a justification mark located at the center of the base line of the text; justification Top Left gives a justification mark located at the top left of the rectangle which surrounds the text.

Text color:

Select a color in which the text should appear. Open the list box by clicking it. To create your own color, select Custom from the bottom of the list or click the create button.

Transparent:

Select this check box when the rectangle in which this text appears should be transparent; you will see the text directly on the contents of the map window.

Clear this check box when a rectangle around the text should have the background color as defined for the map window.

Rotation:

Type a value for the rotation of this text. Rotation is specified in degrees: rotation 0 gives text in direction left to right, rotation 90 gives text in direction bottom to top. Maximum rotation is 360.

Tips:

  1. To move a text item: select a text item and drag it to another position in the editor window.
  2. To make a duplicate of a text item: press and hold the Ctrl key down, select a text item, and drag it to another position.
  3. You can also edit an annotation text object in a table window:
    • double-click an annotation text object in the Catalog, or
    • in the Main window, open the File menu, choose Open as Table, and select the desired annotation text object.
  4. To install a Windows True Type font:
    • In the Windows Settings, choose Control Panel, and choose Fonts. A list of installed fonts will appear.
    • When the desired font is not yet in this list, choose File, Install New Font. The Add Font dialog box will appear.
    • Specify the directory where fonts are available and select the font(s) you want to install. Then click OK.

    Now that the fonts are registered by Windows, you can use them in ILWIS as well.
