Representation Value/Gradual editor

Stretch Steps

 Edit menu
 

Type a value for the number of colors that should be used for values or percentages in between two limits.

When you use stretch for the values or percentages in between two limits, the values or percentages in between these two limits are shown with a number of gradual colors that range from the color of the lower limit to the color of the upper limit.

Dialog box options:

Stretch steps:

Type a value for the number of colors that should be used for values/percentages in between two limits in case of stretching.
When you select a large value, the more gradual colors will change from one limit into the other. The maximum number of colors between to limits is 30.
When you type a small value, you can simulate a kind of 'classification'.

The color step value applies for all values in between limits that are stretched.

The color bar at the bottom of the representation editor shows the colors of the entire representation.

Additional information:

For the colors in between two limits (or break points), there are three possiblities:

  • values in between two limits are stretched between the colors of the upper and lower limit (Stretch), or
  • values in between two limits get the same color as the upper limit (Upper), or
  • values in between two limits get the same color as the lower limit (Lower).

To edit the colors of values in between two limits:

  • double-click on the words Stretch, Upper, Lower that appear in between two limits in the representation editor window, or.
  • select one of the words Stretch, Upper, Lower as they appear in between two limits in the representation editor window and press the Enter key.

When the color method in between limits was Stretch, then Upper is assigned;

when it was Upper, then Lower is assigned;

when it was Lower, then Stretch is assigned. This is a cyclic choice.

Hardware limitation:

Mind that in total only 256 colors are available. It is therefore advised not to use a large value for the color steps, especially when you use many stretch ranges.
