Annotation Text editor

Edit Texts

 Edit menu
 

Edit the settings of multiple selected text items.

This dialog box appears:

  • when you select multiple texts in the editor window and choose the Edit command from the Edit menu or from the context-sensitive menu.

For more information on multiple selection, see below.

Dialog box options:

Font name:

Select a font in which the selected texts should appear. Open the list box by clicking it. You can select any installed True Type font.

Font size:

Type a value for the font size in which the texts should appear.

Bold:

Select this check box when the selected texts should appear in Bold.

Italic:

Select this check box when the selected texts should appear in Italics.

Underline:

Select this check box when the selected texts should appear Underlined.

Justification:

Select a position for the justification mark of the selected texts. Justification Baseline Center gives justification marks located at the center of the base line of each text; justification Top Left gives justification marks located at the top left of the rectangle which surrounds each text.

Text color:

Select a color in which the selected texts should appear. Open the list box by clicking it. To create your own color, select Custom from the bottom of the list or click the create button.

Transparent:

Select this check box when the rectangles in which the texts appear should be transparent; you will see the texts directly on the contents of the map window.

Clear this check box when rectangles around texts should have the background color as defined for the map window.

Rotation:

Type a value for the rotation of selected texts. Rotation is specified in degrees: rotation 0 gives text in direction left to right, rotation 90 gives text in direction bottom to top. Maximum rotation is 360.

Multiple selection

To select two or more text items:

  • choose the Select All command from the Edit menu; all texts in the annotation text object will be selected;
  • drag a rectangle with the mouse pointer in the editor window; all texts within this rectangle will be selected;
  • press and hold the Shift key down and click individual text items.

To deselect a selected item:

  • press and hold the Shift key down and click the selected text item(s) which you want to deselect.

Tips:

  1. To move multiple text items: select multiple text items and drag them to another position in the editor window.
  2. To make duplicates of multiple text items: press and hold the Ctrl key down, select multiple text items, and drag them to another position.
  3. To change the text of a text item, double-click a text in the editor window.
  4. You can also edit an annotation text object in a table window:
    • double-click an annotation text object in the Catalog, or
    • open the File menu in the Main window, choose Open as Table, and select the desired annotation text object.
  5. To install a Windows True Type font:
    • In the Windows Settings, choose Control Panel, and choose Fonts. A list of installed fonts will appear.
    • When the desired font is not yet in this list, choose File, Install New Font. The Add Font dialog box will appear.
    • Specify the directory where fonts are available and select the font(s) you want to install. Then click OK.

    Now that the fonts are registered by Windows, you can use them in ILWIS as well.
