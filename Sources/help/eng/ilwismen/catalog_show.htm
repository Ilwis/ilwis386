Catalog - Show ...
 

Show only certain groups of object types in a Catalog or show all objects in a Catalog. The Show commands work both in List View and in Details View.

Tip:

To specify visibility settings for individual object types, you can also use:

  • the Object Selection toolbar, or
  • the Customize Catalog command on the View menu, Selection tab.

With the Show command, you can choose to show:

 

Maps and Tables: Shows:

raster maps

polygon maps

segment maps

point maps

tables

 

All Data Objects: Shows the objects listed above and:

map lists

object collections

map views

layouts

annotation text objects

graphs

 

All Objects: Shows all objects listed above, and:

domains

representations

georeferences

coordinates systems

histograms of raster maps

histograms of polygon maps

histograms of segment maps

histograms of point maps

sample sets

stereo pairs

two-dimensional tables

matrices

(user-defined) filters

user-defined functions

scripts

Tips:

  • To include directories or drives, use the Object Selection toolbar.
  • To include system objects or non-ILWIS objects in a Catalog, e.g. DBF files and/or TIF files, see Customize Catalog, Selection tab.
  • To sort objects in a Catalog, see Catalog - Sort By ....
  • To search for certain objects, see Catalog - Predefined queries, or Customize Catalog, Query tab.

Mind:

  • Settings on visible object types, sorting/ordering method and queries on objects are either applied to all new Catalogs that will be opened, or to the currently active Catalog only. This depends on whether you click the Use current settings for new Catalogs button or the Use default settings for new Catalogs button on the Customize Catalog, Details View page.
  • Only those objects will be shown which match the selected object types and a used query.
