Map window

 Options menu

Extend Window by Coordinates
 

Increase or decrease the size of the current map window. The current boundaries of the map window are given as defaults.

 

  • If you increase the coordinates boundaries of the map window, extra space will be added to the map window.
  • If you decrease the coordinates boundaries of the map window, only part of your map will be shown.

Only the coordinate boundaries of the current map window will be changed, i.e. display; the coordinate boundaries of map(s) in the map window will remain the same. You can save the new situation of the map window, by saving the map window as a map view.

Dialog box options:

Minimum X, Y:

Type a value for the new minimum X-coordinate and for the new minimum Y-coordinate of the map window.

With minimum XY-coordinates, the coordinates of the lower left corner are meant.

Maximum X, Y:

Type a value for the new maximum X-coordinate and for the new maximum Y-coordinate of the map window.

With maximum XY-coordinates, the coordinates of the upper right corner are meant.

 

Tip: To permanently change the size or outer coordinates of a map, use the Sub-map operations.
