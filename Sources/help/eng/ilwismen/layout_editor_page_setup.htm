Page Setup

 

Before adding map views and annotation to a layout, you should first specify the size, orientation, and margins of the paper on which you wish to create your layout and on which you wish to print, and the printer that you wish to use.

For your information, the current paper size is listed in the status bar of the layout editor. The orientation of the paper is reflected by shape of the 'paper' in the layout editor itself.

Note:

If you change the printer, paper size, or orientation of the paper etc. after adding map views and annotation to the layout, you may need to rearrange and resize the map view(s) and the annotation elements that were previously inserted in the layout.

Dialog box options:

Paper:

Select the paper size and the source of the paper.

Size:

Select the paper size for the selected printer, for example A4 or A3, or another paper size appearing in the drop-down list box. Which paper sizes appear in the drop-down list box depends on the driver of the currently selected printer.

Source:

Select the paper source for the selected printer. Which paper sources that appear in the drop-down list box depends on the driver of the currently selected printer.

 

 

Orientation:

Select the orientation of the paper.

Portrait:

Vertical paper orientation.

Landscape:

Horizontal paper orientation.

  

  

Margins:

Specify the margins of the paper. Any (parts of) map views or annotation placed in these margins will not be printed.

Left:

Type a value in mm for the margin on the left.

Right:

Type a value in mm for the margin on the left.

Top:

Type a value in mm for the margin on the top.

Bottom:

Type a value in mm for the margin on the bottom.

  

  

Printer button:

When you click this button, another dialog box will appear. In that dialog box, the name of the currently selected printer is displayed and the status, type and server name of the printer are shown.

You can select the printer you wish to use. ILWIS will only list the printers that have been properly installed and which appear in Windows Control Panel. For information about installing printers, see your Windows documentation.
