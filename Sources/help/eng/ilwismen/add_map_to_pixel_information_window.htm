Pixel information window

Add Map

 File menu
 

Add a raster, polygon, segment or point map to the pixel information window. If the map has an attribute table, also the columns of the attribute table are displayed in the pixel info window. You can add as many maps to the pixel information window as you like.

The pixel information window shows the class names, identifiers and/or values of all maps and tables that are added to the pixel info window, at the coordinate received from the mouse pointer in a map window or from the digitizer cursor. This provides you with a kind of cross-section at a specific coordinate through all the maps and tables that you added to the pixel info window. The class names, identifiers and/or values found in the different maps and tables at that coordinate are shown in the pixel info window.

In case a map window only displays a raster map with georeference None, the pixel info window displays the Row and Column numbers of the actual mouse position.

Dialog box options:

Select map:

Select a raster, polygon, segment or point map which you want to add to the pixel information window.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

Tips:

  • You can also drag maps from the Catalog to the pixel info window with the mouse.
  • To hide or show maps or columns of an attribute table in the pixel info window, select the Customize command from the Options menu.
