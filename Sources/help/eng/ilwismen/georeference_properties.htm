Georeference Properties

The Georeference Properties sheet lists all kind of important information on a selected georeference. In fact, all relevant information that is stored in the object definition file of the georeference is displayed. For more information, refer to Basic concepts : properties of objects.

The Georeference Properties sheet appears:

  • when you click a georeference in a Catalog with the right mouse button, and select Properties from the context-sensitive menu;
  • when you select a georeference in a Catalog, and choose Properties from the Edit menu in the Main window.

Georeference tab:

Object type and name:

GeoReference type "Georeference name". When you click this flat button, the georeference will be (re)opened.

Nr. of Lines and Columns:

For your information, the number of lines and columns of this georeference is listed.

Pixel size:

For your information, and if available, the pixel size of this georeference is listed.

Coordinate System:

The Name, Description and Coordinate Boundaries of the coordinate system used by this georeference are listed.

Background map:

For a georef of type tiepoints, linear, orthophoto, or parallel projective: the name of background raster map is listed. The background raster map is the raster map which will be opened when you open this georeference.

You may also use a map list as a background map. When the georeference is opened, the map list will be shown as an interactive color composite.

DTM:

For a georef of type direct linear, orthophoto, parallel projective, or 3D: the Digital Terrain Model (DTM) that is used by this georeference is listed. Also, the description, the number of lines and columns and the size of the DTM is listed.

General tab:

Object type and name:

GeoReference type "Georeference name". When you click this flat button, the georeference will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this georeference was last changed.

Object size:

Total size of the georeference on disk in bytes (.GRF; for a georeference tiepoints, a georef direct linear or a georef orthophoto .GRF and .GR#).

Read only:

Select this check box if this georeference should be read only. You will then not be allowed to edit or to delete the georeference. Clear this check box if you want to be able to edit the georeference.

Description:

Optionally, change the description of this georeference. The description of a georeference appears in the title bar when the georeference is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the georeference in the Catalog.

Used By tab:

Object type and name:

GeoReference type "Georeference name". When you click this flat button, the georeference will be (re)opened.

Used by:

In a tree view, it is listed by which other objects this georeference is used.

Additional Info tab:

Object type and name:

GeoReference type "Georeference name". When you click this flat button, the georeference will be (re)opened.

Additional Info:

For a georef tiepoints, a georef direct linear and a georef orthophoto, information on transformation or orientation results is listed. For more information on the Additional Info of a georeference direct linear, refer to ILWIS object : georeference direct linear.

Georeferences are never dependent objects.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
