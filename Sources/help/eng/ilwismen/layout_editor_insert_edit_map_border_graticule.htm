Insert/Edit Map Border

Insert or edit a map border, and/or ticks and/or coordinates for grid lines or a graticule for a selected map view in your layout.

You can insert or edit:

  • a neatline which exactly fits the selected map view;
  • grid ticks and/or coordinates along the map view in meters;
  • graticule ticks and/or coordinates along the map view in degrees, minutes, seconds;
  • coordinates for the corners of a map view;
  • an outline at a certain distance around a map view.

Use the tabs to go to a specific page.

On this Grid page, you can add or edit ticks and/or coordinates at a certain interval in degrees, minutes and seconds along all sides of the selected map view.

Graticule page options:

Graticule ticks:

Select this check box to obtain ticks in degrees, minutes, seconds along the selected map view.

Clear this check box if you do not wish to obtain ticks.

Interval:

Type values for the distance between the ticks in degrees, minutes, seconds.

Tick length (mm):

Type a value for the length of the ticks.

Note: If you used a blocked border (i.e. a neat line with graduation) with a width of e.g. 2 mm, then the tick length should be larger than 2 mm, otherwise the ticks will fall below the blocked border.

Color:

Select a color for the ticks. To define your own color, select the 'Custom' item at the bottom of the list or click the create button; the Windows Color dialog box will appear.

 

 

Graticule Coordinates:

 

Select this check box to obtain coordinate information in degrees, minutes, seconds along the selected map view. Latitudes will appear along the left and the right of your map view; Longitudes will appear along the top and the bottom of the map view.

Clear this check box if you do not wish to obtain coordinate information in degrees, minutes, seconds.

Border texts:

Select the Left, Right, Top, Bottom check boxes to have coordinates appear around the map view along the selected sides.

Rotate horizontal border texts:

 

Type a value for an optional rotation of the coordinates along horizontal borders (degrees, anti-clockwise).

Rotate vertical border texts:

 

Type a value for an optional rotation of the coordinates along vertical borders (degrees, anti-clockwise).

Interval:

Type values (degrees, minutes, seconds) for the interval at which coordinate information should be shown.

Full coordinates:

Choose this option button to obtain complete coordinates along the sides of the map view.

Shortened coords:

Choose this option button to obtain short coordinates (i.e. less digits) along the sides of the map view.

Text offset (mm):

Type a value for the space in mm between the border of your map view and the coordinates.

Color:

Select a color for the coordinates. To create your own color, select 'Custom' at the bottom of the drop-down list, or click the create button ; the Windows Color dialog box will appear.

Font name:

Select the font in which the coordinates should appear; you can select an installed font from the list.

Font size:

Type a value for the font size (pt) in which the coordinates should appear.

Bold:

Select this check box to have coordinates appear in bold.

Italic:

Select this check box to have coordinates appear in italics.

Tips:

  1. To add a graticule to a map view in the layout editor, select the map view and click the Graticule button in the Layout toolbar. If you double-click a map view which contains a graticule in the layout editor, you can also edit the spacing, color etc. of the graticule, or remove the graticule. Click the Redraw button to apply any changes.
  2. To save a changed map view in a layout:
    • select the changed map view in the layout, and
    • click the Save button in the standard toolbar, or
    • click the selected map view with the right mouse button and choose Save View or Save View As from the context-sensitive menu.
  3. If you size a selected map border in the Layout editor, by dragging one of the size handles of a selected map border, then the size or the scale of your map view will change. This depends on whether you used the Keep Scale button.
  4. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
