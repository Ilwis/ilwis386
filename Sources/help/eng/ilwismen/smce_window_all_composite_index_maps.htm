SMCE window

 Generate menu

Generate All Maps   
 

 

Generate all composite index maps, i.e. the SMCE output maps for all alternatives, including all intermediate maps. A composite index map contains the accumulated suitability for all criteria, standardized and weighed as specified in the criteria tree.

Composite index maps have values between 0 and 1:

  • pixels with values near 0 represent unsuitable areas,
  • pixels with values near 1 represent suitable areas.

This dialog box appears:

  • when you choose All Maps from the Generate menu in the SMCE window;
  • when you click the Generate All Maps button in the toolbar of the SMCE window.

Dialog box options:

Show after calculation:

Select this check box if you wish to show all intermediate maps and all output Composite Index maps immediately after calculation. By default the check box is selected.
Clear this check box to only calculate the maps.

 

Note: To calculate a single output map or to calculate the intermediate maps of a certain subgoal, use Generate Selected Maps.

Composite Index maps (SMCE output maps) use system domain Nilto1 and a matching representation from red (0, unsuitable) to green (1, suitable).
