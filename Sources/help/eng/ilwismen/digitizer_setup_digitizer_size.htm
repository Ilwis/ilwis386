Setup Digitizer wizard

Size, Resolution and Final Check

 < Back | Finish >
 

This page allows you to check the behaviour of received digitizer coordinates, to specify the resolution of digitizer tablet (as configured), to specify the size of your digitizer tablet and to check size and resolution feedback.

Check behaviour of digitizer coordinates and digitizer buttons:

At the top of this page:

  • check whether all digitizer buttons work (press buttons 0,1,2,3):
    after Button: you should see the number of the button that you press on the digitizer cursor,
  • check whether the left hand X-value increases by moving the digitizer cursor to the right,
  • check whether the left hand Y-value increases by moving the digitizer cursor up.

If this is the case, you have communication and the Automatic Detection has succeeded. You may proceed to fill out remaining options on this page, see below.

Mind:

If you get any unexpected behaviour, you don't have a proper communication between digitizer tablet and your computer. The following combination of problems may occur:

  • the digitizer settings for (Baud Rate, Parity, Data Bits, Stop Bits) and the specified port settings may not match;
  • the digitizer may not send coordinates in Run Mode;
  • the digitizer may not be configured to send coordinates in a proper ASCII format.

To fix your problems, press the Back button and read the Trouble shooting section in Setup Digitizer - Automatic Detection.

Page options:

Metric Coordinates:
 

Select this check box if during the configuration of the digitizer tablet, you chose a certain number of lines per mm for the Resolution of the tablet.

Clear this check box if you chose to use certain number of lines per inch for the Resolution of the tablet.

Lines per mm/inch:
 

Type a value for the number of lines per mm (or per inch) as used for the Resolution during the configuration of the digitizer tablet. Only when you use the correct Resolution, you will see correct feedback on the Scale of the map on the digitizer in the Map Reference - Digitizer Control Points dialog box.

If you used 1000 lines per inch on the digitizer tablet, you can clear the Metric System check box, fill out 1000 for the resolution, then select the Metric check box again: the value 1000 lines per inch will be automatically converted into 39.37079 lines per mm.

X-size:

Specify the size of the digitizer tablet in X-direction (in mm or in inches).

Y-size:

Specify the size of the digitizer tablet in Y-direction (in mm or in inches).

Check tablet size and tablet resolution feedback:

Check the behaviour of the X Y values on the right hand side at the top of this page:

  • Locate the digitizer cursor in the lower left corner of the digitizer tablet:
    the right hand X and Y values should be similar to 0, 0.
  • Locate the digitizer cursor in the upper right corner of the digitizer tablet:
    the right hand X and Y values should be similar to the size of the digitizer.

 

When the feedback is correct, click the Finish button. You have successfully completed the digitizer setup. Your digitizer is ready for use. On the File menu of the Main window, a check mark will appear in front of the Setup Digitizer command.

Starting to digitize a map:

The next step is to establish a relationship between the coordinates sent by the digitizer and the coordinates of your paper map on the digitizer. This process is called Map Reference. In the Main window, open the File menu and choose Map Reference.

First, select a coordinate system in the Map Reference - Coordinate System dialog box; then establish a relation between the digitizer and the map on the tablet in the Map Reference - Digitizer Control Points dialog box.
