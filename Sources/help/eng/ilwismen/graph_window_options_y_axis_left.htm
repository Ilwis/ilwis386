Graph window

 Graph Management

Y-axis (left)
 

Modify the Y-axis as displayed on the left side of the graph window.

This dialog box appears:

  • when you double-click the Y-axis Left item in the Graph Management pane,
  • when you double-click the Y-axis as displayed on the left side of the graph in the graph window itself.

Dialog box options:

Axis text:

Type explanatory text that should appear along the Y-axis of your graph.

Show grid:

Select this check box when you wish to have grid lines in your graph.

The grid lines will automatically be drawn for all ticks along the Y-axis, i.e. as specified for Interval.

Axis type:

When using a value column for the Y-axis:

  • Select Linear when you wish to use a normal Y-axis.
  • Select Logarithmic when you wish to use a logarithmic Y-axis.
  • Select Normal Probability when you wish to show cumulative frequencies (which range from 0 - 1 or from 0% to 100%) on the Y-axis, and (classified) observed values on the X-axis. If the sample data are in fact normally distributed, the resultant cumulative frequencies will appear as a straight line.

For examples of different types of axes, see X-axis, section Examples.

Min  Max:

When using a Linear or a Logarithmic Y-axis:

Specify the minimum and maximum value that should be used on the Y-axis.

Interval:

When using a Linear Y-axis:

Specify the interval at which values should be shown along the Y-axis.

Text rotation:

Type a value for the rotation of the values along the Y-axis. Rotation is expressed in degrees (anti-clock wise).

Font name:

Select the font in which the explanatory text along the Y-axis should be displayed. You can select any installed True Type font.

Font size:

Type a value for the font size (pt) in which the explanatory text along the Y-axis should appear.

Bold:

Select this check box to have the explanatory text along the Y-axis appear in bold.

Italic:

Select this check box to have the explanatory text along the Y-axis appear in italics.

Tips:

  • To change the text along the Y-axis, or the minimum and maximum values used by the Y-axis, you can also double-click the text, minimum or maximum value in the Graph Management pane.
  • To change the font settings of the values displayed along graph axes:
    • choose Preferences from the File menu of the Main window,
    • then open the Font item and choose Graphs Font.
