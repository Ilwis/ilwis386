Segment editor

 Edit menu

Insert code
 

Select a class name or type a value that will be assigned by default to all new segments that you insert from this moment onwards.

You can set the Insert code when you intend to digitize a couple of segments which should obtain the same class name or the same value. When finished, you can reset the Insert code to another class name or value, and digitize segments with another class name or value than before.

For more information on inserting segments, see Insert mode.

 

Note: The Insert code option is grayed out when editing a segment map with an ID domain or a Unique ID domain, because in such maps, each segment is supposed to have a unique code.

Dialog box options:

Class name/Value:

Select a class name, or type a value that should be assigned by default to all new segments that you will insert from this moment onwards.

Class names, values, etc.

  • For a map using a class domain: select an existing class name from the drop-down list.

    • To enter a new class name: select <new> from the list, then type the new class name, and, optionally, type a class code and choose a color for the new class.

    A new class name will be automatically added to the class domain.

     

  • For a map using a value domain: type a value.

    • The value should fall inside the minimum and maximum value of the value domain used by the map.

     

  • For a map using a Bool domain: select True, False, or ? (= undefined) from the drop-down list.

 

When you choose the '?' question mark, the selected segment(s) will not get a class name or value; they will have the undefined value.

Tips:

  • For maps with a class domain: the class name selected for the Insert Code will be assigned automatically to each newly inserted segment (i.e. no questions).
  • For maps with an ID, a value, or a Bool domain: an ID or a value etc. will be asked for each newly inserted segment. For more information, see Insert mode.
  • For maps with a system UniqueID domain, each newly inserted segment will automatically obtain a unique ID like seg 1, seg 2, seg 3, etc.
  • You can also Edit the class names, values etc. of segments after you finished digitizing. For more information, see Edit.
  • The width of class names, IDs, or values as shown in the Edit dialog box, depends on the width of the domain. You can change the width of a domain by editing the Domain Properties sheet.
