Polygon editor

 File menu

Create labels
 

Create for each polygon a label point, and store the labels as points in a point map.

This operation can be used for instance to rename polygons in a structured way:

  • create labels;
  • open the point map as a table,
  • edit the labels in the table;
  • apply the new label points to the polygon map.

Dialog box option:

Output point map:

Type a name for the output point map which will contain a label point for each polygon.

Tip:

To create polygon labels, you can also use the Polygons to Points operation.
