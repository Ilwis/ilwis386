Segment editor

 File menu

Boundaries of map
 

Change the coordinate boundaries of the segment map you are editing.

The coordinate boundaries define the outer boundaries of the segment coordinates that can be added to your segment map.

 

  • By extending the coordinate boundaries of the segment map, you can also digitize segments in the extended part of the segment map.

  • When accidentally the coordinate boundaries are reduced too much, so that part of the segments would fall outside the map, the program will recalculate the outer bounding box of all segments that are already in the map and use this as the coordinate boundaries of your map; so none of your existing segments will get lost.

Dialog box options:

Min X, Y:

Type new values for the minimum X-coordinate and for the minimum Y-coordinate of the segment map. With the minimum XY-coordinate, the XY-coordinate of the lower left corner of the map is meant.

Max X, Y:

Type new values for the maximum X-coordinate and for the maximum Y-coordinate of the segment map. With the maximum XY-coordinate, the XY-coordinate of the upper right corner of the map is meant.

Only click the Default button when the new boundaries should exactly fit the outer coordinates of the existing segments in the current map. The Min X, Y and the Max X, Y coordinates will change in the text boxes above.

Note: You cannot return to the original map boundaries once you clicked the Default button and the OK button of this dialog box.

Adapt window:

Select this check box when you wish to show the map with its new boundary coordinates (i.e. new size) in the editor.

Clear this check box if you wish to show the map with its original boundary coordinates.

After clicking the OK button, you can directly continue to digitize new segments within the new coordinate boundaries; you can also do a Map Reference again using the new corners.

When the new coordinate boundaries exceed the coordinates of the coordinate system used by the map, it is not necessary to extend the boundaries of the coordinate system used by the map.

Tips:

  • To change the coordinate boundaries of a segment map, you can also use the Sub-map of segment map operation.
  • To merge two or more segment maps, use the Glue segment maps operation.
