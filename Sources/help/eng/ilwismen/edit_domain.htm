View/Edit Domain
 

View and/or edit the domain of a map that is currently displayed in the map window:

  • edit the class names or their codes of a class domain, or
  • edit identifiers or their codes of an identifier domain, or
  • edit the minimum and maximum values and the precision of a user-defined value domain.

When you select:

  • a map using a class or identifier domain, the Domain Class/ID editor will be opened;
  • a map using a user-defined value domain, the Edit Value Domain dialog box will appear.

This option is not available for maps using a system domain (Bit, Bool, Color, ColorCmp, Count, Distance, Image, Min1to1, NILto1, Perc, Value, Yes/No domain). This option is also not available for maps using a Picture domain. Furthermore, this option is not available for read-only domains.

Tip:

To edit a domain, you can also double-click it in the Catalog. To make domains visible in the Catalog, refer to How to customize the Catalog.
