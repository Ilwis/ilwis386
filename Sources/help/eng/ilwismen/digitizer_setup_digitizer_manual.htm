Unravel the received ASCII string yourself.

Received digitizer coordinates (check whether you have communication):

You have communication between the digitizer and the computer:

  • when the first line of this page reads:

    • Received: coordinates as readable numbers;

  • when coordinates change from about 0,0 (when moving the digitizer cursor into the lower left corner of the tablet) to greater positive figures (when moving the digitizer cursor into upper right corner of the tablet).

You definitely don't have a proper communication between your digitizer and your computer:

  • when the first line of this page reads:

    Received: Nothing

    Received: coordinates as unreadable characters

  • when coordinates do not change when you move the digitizer cursor over the tablet,
  • when coordinates only appear after pressing a button on the digitizer cursor.

Read the Trouble shooting section on the Automatic Setup page.

Page options:

Received:

Try to interpret the ASCII string after 'Received' yourself. You should be able to determine an X-coordinate, a Y-coordinate, and a value for the digitizer buttons (flags).

Nr chars:

Specify a value for total numbers of characters in the string after 'Received'. Count all characters in the string, including possible spaces, commas, etc. which may be used as separators.

X Pos:

Specify a value for the position where the first character of the X-coordinate is found in the string; this could be 1.

X Length:

Specify a value for the total number of characters used for the X-coordinate in the string. Do not include spaces or commas that are used as separators.

Y Pos:

Specify a value for the position where the first character of the Y-coordinate is found in the string.

Y Length:

Specify a value for the total number of characters used for the Y-coordinate in the string. Do not include spaces or commas that are used as separators.

Flag Pos:

Specify a value for the position where digitizer cursor button flags (sent when digitizer cursor buttons are pressed) are found in the string; this a single position.

Flag Codes:

Specify values for the flag codes that are received by the computer when digitizer cursor buttons are pressed; e.g. 1248

Example:

Suppose a received string reads: 07287,07980,0. When pressing digitizer cursor buttons 0,1,2,3, you obtain flag codes 1,2,4,8. Then, you should specify the following values:

 

Nr chars 13    
X Pos 1 X Length 5
Y Pos 7 Y Length 5
Flag Pos 13 Flag Codes 1248

When finished, click the Next button on this page; the Setup Digitizer - Size, Resolution, Final Check page will appear.
