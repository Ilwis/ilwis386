Point editor

 Edit menu

Add Point
 

Add points to a point map by entering XY-coordinates from the keyboard; then assign a class name, ID, or value.

Dialog box options:

X:

Type a value for the X-coordinate of the new point.

Y:

Type a value for the Y-coordinate of the new point.

Value:

Depending on the domain of the point map that you are editing, select a class name from the list, type an identifier, or type a value that should be assigned to the new point.

Tip:

It might be easier to add points to a point map:

  • with the digitizer, see Point editor : functionality or Point editor : digitizer, or
  • with the mouse, see Point editor : Insert mode.
