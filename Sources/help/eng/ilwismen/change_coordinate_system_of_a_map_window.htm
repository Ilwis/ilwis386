Map window

 Options menu

Change Coordinate System
 

Show the point, segment and polygon maps displayed in this map window with another coordinate system and/or georeference.

The newly selected coordinate system or georeference is only temporarily used by the map window, it is not added to the vector map(s).

 

  • This option can be used to quickly display vector maps in different coordinate systems, for example with different projections.
  • You can also use this option to show the vector maps with a previously created georeference 3D, or with a georeference tiepoints as used by images of the same area.

Dialog box options:

Select coordinate system or georeference:

Select a coordinate system or a georeference to be used by the current map window.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

If ILWIS is able to convert the coordinate system of the map into the 'new' coordinate system, the map window displays the maps using the 'new' coordinate system. The resulting map window can be saved as a map view.

Tips:

  • Instead of choosing the menu command Change Coordinate System, you can also drag a coordinate system or a georeference from the Catalog to the map window. This has the same effect.
  • Another way to set the georeference of a map window is to add a raster map to the map window. The map window then automatically gets the georeference of that raster map.
  • To permanently change the coordinate system of a vector map, use one of the Transform operations.
  • To show raster maps with a different coordinate system or georeference, the Resample operation should be used.
