Edit Raster Map
 

Select a raster map that you want to edit while using the current contents of the map window as a background. This is also called screen digitizing.

This dialog box appears when from the Edit menu in a map window, you choose Edit Other Map, Pixel editor.

Dialog box options:

Raster map:

Select the raster map you wish to edit. Only maps with the same georeference as the raster map currently displayed in the map window can be selected.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The Pixel editor is started; you can start editing the selected raster map.

When the map window already displayed a raster map, the existing raster map remains displayed. The Pixel editor is opened:

  • if you zoom in only a little, the pixels of the map you want to edit are displayed on top of the raster map that was already displayed.
  • if you zoom in very much, the class names, IDs or values of the map you want to edit are displayed as text on top of the raster map that was already displayed.

Tips:

  1. Dependent maps and maps marked as read-only cannot be edited. First break the dependency link or remove the read-only flag in the Raster Map - Properties sheet.
  2. To directly edit a raster map:
    • in the Catalog, select an editable raster map, then open the Edit menu in the Main window and choose Edit Object, or
    • in the Catalog, click an editable raster map with the right mouse button and choose Edit from the context-sensitive menu, or
    • in a map window where the raster map is displayed, open the Edit menu and select Edit Layer.
