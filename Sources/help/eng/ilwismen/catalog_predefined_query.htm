Catalog - Predefined queries ...

 

Show only the objects in a (directory) Catalog that are the result of one of the predefined queries. The Catalog Queries command works both in List View and in Details View.

In short, you can use the predefined queries to see:

  • All objects,
  • All objects made today,
  • Maps with a class or ID domain, the image domain or a value domain,
  • Maps with a georeference corners or a georeference tiepoints,
  • Any queries created by yourself using the Customize Catalog, Query tab and which were saved.

For more information on predefined queries and on query syntax to create queries yourself, see Customize Catalog, Query tab.

The selected query will be preceded by an option bullet . Furthermore, the query that is currently in use is shown in the status bar of the Main window.

Tips:

  • To view only certain types of objects in a Catalog, see Catalog - Show ..., Main window : Object selection toolbar, or Customize Catalog, Object Selection tab.
  • To sort objects in a Catalog, see Catalog - Sort By ....

Mind:

  • Settings on visible object types, sorting/ordering method and queries on objects are either applied to all new Catalogs that will be opened, or to the currently active Catalog only. This depends on whether you click the Use current settings for new Catalogs button or the Use default settings for new Catalogs button on the Customize Catalog, Details View page.
  • Only those objects will be shown which match the selected object types and a used query.
