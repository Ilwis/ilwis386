Map View Properties

The Map View Properties sheet lists all kind of important information on a selected map view. In fact, all relevant information that is stored in the object definition file of the map view is displayed. For more information, refer Basic concepts : Properties of objects.

The Map View Properties sheet appears:

  • when you click a map view in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a map view in a Catalog, and choose Properties from the Edit menu in the Main window.

Map View tab:

Object type and name:

Map View "Name". When you click this flat button, the map view will be (re)opened.

Georeference:

The name , the type and the description of the georeference used by this map view are listed.

Coordinate system:

The name, the type and the description of the coordinate system used by the georeference of the raster maps in this map view are listed.

General tab:

Object type and name:

Map View "Name". When you click this flat button, the map view will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this map view was last changed.

Object size:

Total size of this map view (.MPV) on disk in bytes. This does not include any maps or annotation text that may have been added to the map view.

Read only:

Select this check box if this map view should be read only. You are then not allowed to edit or delete the map view. Clear this check box if you want to be able to edit the map view.

Description:

Optionally, change the description of this map view. The description of a map view appears in the title bar when the map view is displayed. Furthermore, the description is visible on the status bar of the Main window when moving the mouse pointer over the map view in a Catalog.

Contains tab:

Object type and name:

Map View "Name". When you click this flat button, the map view will be (re)opened.

Objects in container:

All objects that are part of this map view are listed; this includes point, segment, and polygon maps, a raster map or a map list, and annotation text layers.

Used By tab:

Object type and name:

Map View "Name". When you click this flat button, the map view will be (re)opened.

Used By:

In a tree view, it is listed by which other objects this map view is used, e.g. layouts.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
