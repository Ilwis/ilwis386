Coordinate System Properties

The Coordinate System Properties sheet lists all kind of important information on a selected coordinate system. In fact, all relevant information that is stored in the object definition file of the coordinate system is displayed. For more information, refer to Basic concepts : properties of objects.

The Coordinate System Properties sheet appears:

  • when you click a coordinate system in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a coordinate system in a Catalog, and choose Properties from the Edit menu in the Main window.

CoordinateSystem tab:

Object type and name:

Coordinate system type "Coordinate system name". When you click this flat button, the coordinate system will be (re)opened.

Boundaries:

For your information, the coordinate boundaries are listed as (MinX, MinY) and (MaxX,MaxY).

When this coordinate system is a coordsys of type latlon, the boundaries will appear in minutes, degrees, seconds.

Sphere radius:

For a coordinate system of type projection to which no projection information has been added yet: the default Sphere radius is listed.

Projection:

For a coordinate system of type projection to which projection information has been added: the name of the Projection is listed.

Datum, Ellipsoid:

For a coordinate system of type projection, when a datum and/or an ellipsoid has been specified for the projection: the name of the Datum, the Datum Area, and/or the name of the Ellipsoid are listed as well.

For a coordinate system of type latlon, when a datum and/or an ellipsoid has been specified: the Datum and/or the Ellipsoid are listed as well.

Related coordsys:

For a coordinate system of type formula, tiepoints, direct linear or orthophoto: the name of the related coordinate system is listed as well. The related coordinate system is usually the coordinate system with correct coordinates.

Background map:

For a coordinate system of type tiepoints, direct linear or orthophoto: the name of background vector map is listed. The background vector map is the vector map which will be opened when this coordinate system is opened.

General tab:

Object type and name:

Coordinate system type "Coordinate system name". When you click this flat button, the coordinate system will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this coordinate system was last changed.

Object size:

Total size of the coordinate system on disk in bytes (.CSY; for a coordinate system tiepoints, direct linear, orthophoto .CSY and .CS#).

Read only:

Select this check box if this coordinate system should be read only. You will then not be allowed to edit or delete the coordinate system. Clear this check box if you want to be able to edit the coordinate system.

Description:

Optionally, change the description of this coordinate system. The description of a coordinate system appears in the title bar when the coordinate system is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the coordinate system in the Catalog.

Used By tab:

Object type and name:

Coordinate system type "Coordinate system name". When you click this flat button, the coordinate system will be (re)opened.

Used by:

In a tree view, it is listed by which other objects this coordinate system is used.

Additional Info tab:

Object type and name:

Coordinate system type "Coordinate system name". When you click this flat button, the coordinate system will be (re)opened.

Additional Info:

For a coordsys tiepoints, direct linear or orthophoto: information on transformation or orientation results is listed.

Coordinate systems are never dependent objects.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
