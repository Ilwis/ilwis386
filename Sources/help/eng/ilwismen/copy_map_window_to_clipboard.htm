Map window

 Edit menu

Copy to clipboard

 

Copies the current contents of the map window as a picture to the clipboard.

If you copy for instance a map window which displays multiple layers (i.e. a map view) to the clipboard, you can paste the picture later into a word processor, a desk top publishing system, etc.

 

  • All layers that are displayed in the map window will be copied to the clipboard.
  • When you zoomed in on a part of the map window, only that part will be copied.

Tips:

  • You can also use Ctrl+C to copy the currently displayed part of the map window to the clipboard.
  • To copy the whole map window including the window borders, title bar, menu bar, toolbar, status bar, etc., press Alt+PrintScrn on the keyboard. The map window will be copied as a bitmap to clipboard.
  • You can use Windows' Clipboard Viewer to inspect the current contents of the clipboard.
  • To copy (part of) a map as ILWIS data or as text (coordinates and class names, IDs or values) to the clipboard, use the Copy command on the Edit menu of the Pixel editor, Polygon editor, Segment editor or Point editor.
  • To add maps to a layout, it is advised to first save the contents of a map window as a map view, and then add the map view to the layout in the layout editor.
