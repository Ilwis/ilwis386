Segment editor

 Edit menu

Move Points mode   
 

 

Choose Move Points mode to change the position of a coordinate within a segment with the mouse.

You can move:

  • a node (the start or end coordinate of a segment), or
  • an intermediate point within a segment.

When Move Points mode is active:

  • the mouse pointer appears as a pair of tweezers,
  • all segment nodes appear as boxes and all intermediate coordinates appear as crosses,
  • an option bullet appears in front of this menu command,
  • the Move Points mode button in the toolbar appears down, and
  • the floating digitizer window is titled Move Points.

Three ways to move a node or an intermediate point:

  1. Click a begin or end node or an intermediate point;
    the selected point and its segment change their color;
    • move the point to another position;
    • click again to drop the point.

     

  2. Press and hold down the left mouse button on a node or an intermediate point;
    the selected point and its segment change their color;
    • move the point to another position;
    • release the left mouse button to drop the point.

     

  3. To pick up a node to which multiple segments are connected: click the node;
    the node is selected and one segment changes its color;
    • if this is not the segment you want, do not move the mouse and click again, until the correct segment is selected;
    • then, move the node to another position.
    • click again to drop the point.

To snap the selected point to an existing node (snap):

  • position the selected point on the node to which you want to snap;
  • press and hold down the Shift key and click or release the left mouse button on the node;
  • when AutoSnap is active, you can simply click or release the left mouse button on the node.

To split a segment and snap to the selected point to the created node (snap/split):

  • position the selected point somewhere on a segment;
  • press and hold down the Ctrl key and click somewhere on the segment;
  • a message box asks for confirmation to split the segment to which you want to snap to.

 

Tips:

  • When you have picked up a point, moved the mouse pointer, and when the left mouse button is still down (you did not yet drop the point on a new position), you can press Esc to cancel the move.
  • You can also move points within segments with the digitizer.
