Script Properties

The Script Properties sheet lists all kind of important information on a selected script. In fact, all relevant information that is stored in the object definition file of the script is displayed. For more information, refer to Basic concepts : properties of objects.

The Script Properties sheet appears:

  • when you click a script in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a script in a Catalog, and choose Properties from the Edit menu in the Main window;
  • when you open a script, and choose Properties from the File menu in the script editor;
  • when you open a script, and click the Properties button in the toolbar of the script editor.

General tab:

Object type and name:

Script "script name". When you click this flat button, the script will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this script was last changed.

Object size:

Total size of the script on disk in bytes (.ISL and .ISF).

Read only:

Select this check box if this script should be read only. You are then not allowed to edit or delete the script. Clear this check box if you want to be able to edit the script.

Description:

Optionally, change the description of this script. The description of a script appears in the description bar when the script is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the script in the Catalog.

Scripts are never dependent objects.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
