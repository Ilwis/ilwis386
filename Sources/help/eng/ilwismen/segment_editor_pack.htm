Segment editor

 File menu

Pack
 

Pack permanently removes from your segment map any segments that are marked for deletion.

Pack will thus reduce:

  • the number of segments in your map, and
  • the total file size of your segment map.

Segments are marked for deletion:

  • when you deleted segments with the Delete command,
  • when you merged segments: through merging a new segment will be created out of two merged segments, and both original segments will be deleted.

When, after you used Pack, the segment editor is still open, you can use Undo all changes to recover any changes (including the changes resulting from Pack) until the last time you used Save.

Once you used Pack, you cannot use Undelete Segments anymore to recover any specific segments which were accidentally deleted.

When, after using Pack, the segment editor is closed, all deleted segments are truly deleted and cannot be restored in any way.
