Create Pyramid Layers
 

Create pyramid layers for one or more raster maps.

Pyramid layers speed up the display of large raster maps in a map window, for instance when raster maps are larger than a few thousand lines and columns.

 

  • If no selection is made in the current Catalog, then pyramid layers will be created for every raster map in the Catalog;

    •  

  • If one or more raster maps have been selected in the Catalog, then pyramid layers will only be created for the selected raster maps;

    •  

  • If one or more map lists have been selected in the Catalog, then pyramid layers will only be created for the raster maps that are part of the selected raster map lists.

Tip:

You can also create Pyramid layers:

  • through the Preferences command on the File menu of the Main window: choose Map window, Display, Raster maps, and select the check box Create Pyramid Layers by default.
  • through the Properties of a raster map, or through the Properties of a map list,
  • through the Display Options dialog box of a raster map,
  • by typing a CreatePyramidLayers command on the command line of the Main window, or by using this command in a script (for more information, see Appendices : ILWIS script language (syntax), section Creating/Deleting pyramid layers).
