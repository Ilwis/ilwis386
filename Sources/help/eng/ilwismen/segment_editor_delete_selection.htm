Segment editor

 Edit menu

Delete
 

Delete selected segments from the current segment map.

To delete:

Select one or more segments, and:

  • choose Delete from the Edit menu, or
  • choose Delete from the context-sensitive menu, or
  • press Del on the keyboard.

For more information on selecting segments, refer to Select mode.

Tip:

To recover segments which were accidentally deleted, you can use:

  • Undelete segments: to undelete specific segments;
  • Undo all changes: to restore all changes made in the segment editor until the last time you used Save.
