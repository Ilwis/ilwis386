Graph window

 Graph Management

Y-axis (rose diagram)
 

Modify the Y-axis of the rose diagram as displayed in the graph window.

This dialog box appears:

  • when you double-click the Y-axis item in the Graph Management pane,
  • when you double-click the Y-axis as displayed next to the rose diagram in the graph window itself.

Dialog box options:

Show grid:

Select this check box when you wish to have grid lines in your graph.

The grid lines will automatically be drawn for all ticks along the Y-axis, i.e. as specified for Interval.

Axis type:

When using a value column for the Y-axis:

  • Select Linear when you wish to use a normal Y-axis.
  • Select Logarithmic when you wish to use a logarithmic Y-axis.

For examples of different types of axes, see X-axis, section Examples.

Min  Max:

When using a Linear or a Logarithmic Y-axis:

Specify the minimum and maximum value that should be used on the Y-values.

Interval:

When using a Linear Y-axis:

Specify the interval at which values should be shown.

Text rotation:

Type a value for the rotation of the values. Rotation is expressed in degrees (anti-clock wise).
