Insert/Edit Legend (ID point map + attributes)

Symbols as graphs: color

 

Insert or edit a size legend for an ID point map with an attribute table where the points are represented as pie charts, bar charts or composed bars. The legend will be added to your layout.

The legend will consist of a number of colored boxes, and accompanying texts.

  • The colors used in the pie charts, bar charts, composed bars, or volume cubes, are determined by the colors that you assigned to these selected columns in the Symbols as Graphs dialog box.
  • The texts are the names or the descriptions of the columns you selected to obtain graphs in the Symbols as Graphs dialog box.

This dialog box appears:

  • when you displayed an ID point map with an attribute table, and
  • in the Display Options - Point Map dialog box, you chose the Symbols as Graphs option, and clicked the Details button, and
  • in the Symbols as Graphs dialog box, you selected to display the map as pie charts, bar charts or composed bars, and,
  • you saved the map as a map view,
  • in the layout editor, you selected the map view and choose to add a color legend.

Dialog box options:

Number of columns:

Type a value for the number of columns in which the legend should be displayed.

Text:

Select what kind of texts should be used in the legend. The texts explain the colors in the legend.

Name:

To obtain the names of the selected columns as legend texts.

Description:

To obtain the descriptions of the selected columns as legend texts. When a column does not have a description, there will be no legend text.

Font name:

Select the font in which the legend texts should appear; you can select an installed font from the list.

Font size:

Type a value for the font size (pt) in which your legend texts should appear.

Bold:

Select this check box to have legend texts appear in bold.

Italic:

Select this check box to have legend texts appear in italics.

Tips:

  1. To adapt the graphs themselves or to change the colors both in the point map and in the legend, or to change the order of the legend:
    • double-click the map view containing the point map in the layout editor,
    • use the appearing Layer Management pane to adapt the display options of the point map; in the Symbols as Graphs dialog box, you can then specify other columns and other colors to be used in the graphs or change the order of the selected columns.

    Do not forget to save the changed map view.

  2. In the Layout editor, a legend can be sized by dragging for instance one of the corner size handles of a selected legend to a new position. The mouse pointer changes into a double-sided arrow when at the right position to start dragging. The box sizes and the font in which legend texts are displayed will be automatically adapted to the new size.
  3. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
