Table window

Columns menu

Add Column
 

Add a new column to the table.

This dialog box appears:

  • when you choose Add Column from the Columns menu, or
  • when you choose Column Management from the Columns menu, and click the Add Column button in the Column Management dialog box, or
  • when you scroll to the last column in the table, and double-click to the right of the last column in the gray column header area.

Dialog box options:

Column name:

Type a new name for the column.

Domain:

Select an existing domain for the new column, or create a new domain for the column.

  • If you wish to use values in the new column, you can usually select the system Value domain, and specify value range and precision.
  • You only need to select or create a user-defined value domain when the values have a specific meaning for which you have or want to create a meaningful representation.
  • If you wish to store coordinates in a new coordinate column, you can select an existing coordinate system to be used as domain of the column.

Value range:

If you selected a value domain, specify the minimum and maximum value of this domain; this is called the value range.

Precision:

If you selected a value domain, type a value for the precision of this domain.

Description:

Optionally, type a description for the column. The description of a column is visible on the status bar of the table window when moving the mouse pointer over the column name or the fields of the column.

The new column appears to the right of the last column. Initially, the new column is filled with question marks (undefineds). You can start editing the fields of the column.

Tip:

  • To change the order of columns, you can use drag and drop. In the table window, click a column name and hold the mouse button down, the mouse pointer changes into a column icon , drag the column to a new position in the table, then release the left mouse button.
  • To change the order of columns, you can also use Column Management, and drag columns in the dialog box to another position.
  • The Add Column command is not available when your table is read-only. You can change the read-only flag in the Table Properties sheet.
