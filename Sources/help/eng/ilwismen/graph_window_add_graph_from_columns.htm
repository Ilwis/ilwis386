 Edit menu

Add Graph from Columns
 

Add another graph (from columns) to the current graph window.

This dialog box appears:

  • when you choose the Add Graph from Columns command from the Edit menu in a graph window,
  • when you click the Add Graph button in the toolbar of a graph window,
  • when you use the right mouse button in the white space of the Graph Management pane, and choose the Add Graph from Columns command from the context-sensitive menu.

Dialog box options:

Table:

Select the table which contains the column(s) that you wish to show as a graph.

X-axis:

Select a column for the X-axis of the graph. The list box shows all columns of the table.

This X-axis list box is not available when you used the domain of the table for the X-axis of the graph in the Create Graph / New Graph dialog box.

Y-axis:

Select a column for the Y-axis of the graph. The list box shows all columns of the table.

A new line graph or bar graph will be immediately displayed. For some examples of graphs, see the Example section below.

Tips:

  • To modify the appearance, type of graph, and/or the name of the Graph from Columns layer, double-click the graph layer in the Graph Management pane: the Graph Options - Graph from Columns dialog box will appear.
  • To adapt the title, the X-axis, or the Y-axis (left or right) of the graph, double-click the appriopriate item in the Graph Management pane or in the graph window itself: the Title, X-axis, Y-axis (left), or the Y-axis (right) dialog box will appear.

    • The grid lines of the graph can be modified in the X-axis and Y-axis dialog boxes.

  • To create a new graph in a new graph window, choose Create Graph from the File menu in the graph window: the Create Graph dialog box will appear.
  • To save the contents of a graph window as a graph, choose Save or Save As from the File menu in the graph window.
