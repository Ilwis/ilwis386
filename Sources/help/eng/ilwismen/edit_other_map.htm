Edit Other Map
 

Edit another map than the maps that are currently displayed in this map window.

This option is designed to edit a map while using the contents of the current map window as a background. In this way, you can update an existing map on the basis of the maps which are displayed as background. This is also called screen digitizing.

Select from the cascading menu:

Pixel editor:

The Edit Raster Map dialog box follows.

Point editor:

The Edit Point Map dialog box follows.

Segment editor:

The Edit Segment Map dialog box follows.

Polygon editor:

The Edit Polygon Map dialog box follows.

Tip:

To create and edit a new map while using the contents of the current map window as a background, open the File menu in the map window, and choose the Create command.

 

The layers that are displayed remain displayed in the map window.

  • When you select a polygon, segment or point map to be edited, the polygons, segments or points of this second map will be displayed as a new layer over the existing layers displayed in the map window; you can then edit the newly added vector layer.
  • When only vector data is displayed in your map window, and you select to edit a raster map, the georeference of the raster map will become the georeference of the map window.
  • When you already have a raster map displayed in your map window, and you select another raster map to be edited, usually it is necessary to zoom in until you can view and edit the pixels of the second map.

The maps in the map window and the map that you select to edit must use the same georeference or coordinate system, or a compatible coordinate system.

Mind:

Dependent maps and maps marked as read-only cannot be edited; also maps using a Picture domain or a Color domain cannot be edited. When you want to edit a dependent map, you first have to break the dependency links of the map in the map's Properties sheet.
