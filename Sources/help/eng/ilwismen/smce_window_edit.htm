 Edit menu

Edit

 

Edit the currently selected item in the criteria tree. The actual edit action depends on the mode (problem definition mode or multicriteria analysis mode) and the type of item (group, factor or constraint) selected.

When the SMCE application is in Problem Definition mode you can use the edit option:

  • to edit item properties,
  • to select input maps, and
  • to edit the name(s) of the output map(s).

When the SMCE application is in Multicriteria Analysis mode you can use the edit option:

  • to standardize factors and contraints (in this case the Edit button does exactly the same as the Standardize button),
  • to weigh the criteria root or groups of factors (in this case the Edit button does exactly the same as the Weigh button), and
  • to edit the name(s) of the output map(s).

The edit action is executed:

  • when you choose Edit from the Edit menu in a SMCE window;
  • when you click the Edit button in the toolbar of the SMCE window;
  • when you press the Enter Key on the keyboard;
  • when you click an item in the criteria tree with the right mouse button and choose Edit from the context-sensitive menu;
  • when you double-click the root, a constraint or a factor in the criteria tree;
  • when you single-click a selected group of factors, an input map or an output map in the criteria tree.
