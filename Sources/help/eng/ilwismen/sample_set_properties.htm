Sample Set Properties

The Sample Set Properties sheet lists all kind of important information on a selected sample set. In fact, all relevant information that is stored in the object definition file of the sample set is displayed. For more information, refer to Basic concepts : properties of objects.

The Sample Set Properties sheet appears:

  • when you click a sample set in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a sample set in a Catalog, and choose Properties from the Edit menu in the Main window.

Sample Set tab:

Object type and name:

SampleSet "Name". When you click this flat button, the sample set will be (re)opened.

Domain:

For your information, the domain used by the sample set and the sample map is listed; this domain contains all the class names that you can assign to your training pixels.

Sample map:

Shows which sample map is used by the sample set. This map contains the locations of the training pixels and the class names assigned to them.

Optionally, you can select another sample map from the drop-down list, if you want to view the results of another sample set (e.g. using different sampling techniques).

Map list:

Shows which map list is used by the sample set, that is the set of images you want to classify in a later stage.

Optionally, you can select another map list from the drop down list, or create one by pressing the create button. This option may be usefull if you want to check spectral class separation in another set of input bands (e.g. principle components of your original bands).

General tab:

Object type and name:

SampleSet "Name". When you click this flat button, the sample set will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this sample set was last changed.

Object size:

Total size of the sample set on disk in bytes (.SMS).

Read only:

Select this check box if this sample set should be read only. You are then not allowed to edit or delete the sample set. Clear this check box if you want to be able to edit the sample set.

Description:

Optionally, change the description of this sample set. The description of a sample set appears in the title bar when the sample set is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the sample set in the Catalog.

Used By tab:

Object type and name:

SampleSet "Name". When you click this flat button, the sample set will be (re)opened.

Used by:

In a tree view, it is listed by which other objects this sample set is used.

Sample sets are never dependent objects.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
