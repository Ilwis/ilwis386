Pixel information window

Customize

 Options menu
 

Customize the working of the pixel information window:

  • receive coordinates at the position of the mouse pointer in a map window and/or at the position of the digitizer cursor; subsequently, the class names, identifiers and/or values of the maps (and tables) that were added to the pixel info window at that coordinate.
  • continuously receive coordinates and retrieve information or only receive coordinates and retrieve information at a click of the mouse pointer or the digitizer cursor, and
  • of all maps (and columns of attribute tables) that were added to the pixel info window, select the ones of which you want to see the class names, identifiers, or values in the pixel info window.

This dialog box appears:

  • when you choose Customize from the Options menu in the pixel info window, or
  • when you clicked the upper left gray field in the pixel info window.

Dialog box options:

Mouse:

Select this check box when the pixel info window should receive coordinates from the mouse pointer in a map window. Subsequently, map/column information is retrieved at that coordinate. Clear this check box if the mouse pointer should not be active.

Continuous:

Select this check box if the mouse pointer should continuously receive coordinates and retrieve information when it is positioned in a map window. Clear this check box when the mouse pointer should only receive coordinates and retrieve information at a click in the map window.

Digitizer:

Select this check box when the pixel info window should receive coordinates from the digitizer cursor. Subsequently, map/column information is retrieved at that coordinate. Clear this check box if the digitizer cursor should not be active.

Continuous:

Select this check box if the digitizer cursor should continuously receive coordinates and retrieve information when it is positioned on a referenced map on the digitizer. Clear this check box when the digitizer cursor should only receive coordinates and retrieve information when you press a button on the digitizer cursor.

Show values of:

In the list box, select the maps and attribute columns of which class names, identifiers, or values that should be shown in the pixel info window. In the list box, selected maps and columns appear highlighted. You can deselect the maps and columns that you do not want to see in the pixel info window.
