Find undefineds

 

To find polygons which do not have a value, class name or ID yet.

The Find Undefineds command is active when a check mark appears in front of this menu command.

When using Find Undefineds:

  • undefined polygons will be displayed in the color as specified in the Polygon editor - Customize dialog box;
  • with the mouse, select one or more undefined polygons by clicking;
  • then, choose Edit from the context-sensitive menu or from the Edit menu to assign a proper value, class name or ID to the selection.

When finished, you can switch off the Find Undefineds command again.

 

Tip: To select multiple undefined polygons and give them the same value or class name: press and hold down the Ctrl key while clicking undefined polygons. Then choose the Edit command as usual.
