Remove Layer

 

Remove a layer from the current graph.

 

  • First, select the layer that you wish to remove in the Graph Management pane.
  • Then, either:
    • choose the Remove Layer command from the Edit menu, or
    • click the Remove Layer button in the toolbar of the graph window.

You can also remove a layer from the graph window by selecting a layer in the Graph Management pane with the right mouse button, and then choose the Remove Layer command from the context-sensitive menu.

 

Tip: To temporarily not display a layer in the graph window, you can also switch the layer off in the Graph Management pane.
