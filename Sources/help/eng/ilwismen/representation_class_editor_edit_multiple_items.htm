Representation Class editor

Edit multiple items

 Edit menu
 

When editing the representation of a class domain in the representation class editor, assign a single color or assign a range of colors to multiple selected classes.

Assigning a range of colors can be used:

  • to obtain smooth color changes between multiple classes, for instance from yellow to red, through orange, or
  • to obtain similar colors for related classes in a domain, e.g. various forest types in similar shades of green, etc.

This dialog box appears when:

  • you selected multiple classes in the editor window (see the Multiple Selection information below), and
    • you choose Edit Multiple Items from the Edit menu, or
    • you press the Enter key on the keyboard.

For more information, refer to Representation Class editor : functionality.

Dialog box options:

Single color:

Select this option to assign the same color to all selected classes.

Color range:

Select this option to assign a range of colors to the selected classes. Specify the color for the first class in your selection in the From Color list box, and specify the color for the last class in your selection in the To Color list box. A straight line will be drawn between these two colors in the Red, Green, Blue color cube; and all other selected classes will obtain a color on this line according to the class order in the domain. To change the order of classes in the domain, see the Tips below.

From color:

Select a color for the first domain item in the selection. To define your own color, click the create button or double-click the Custom item at the bottom of the list; the Custom Colors dialog box will appear.

To color:

Select a color for the last domain item in the selection. To define your own color, click the create button or double-click the Custom item at the bottom of the list; the Custom Colors dialog box will appear.

Variation:

Select this check box when a random color variation should be used around the straight line between the specified From and To Color in the RGB color cube. Subsequently, specify a value between 1 and 255 for the maximum variation allowed.

Tips:

  1. Once you have assigned a range of colors to multiple domain items, you can refine the colors for individual items:
    • by selecting a single class in the editor window and use the Red, Green, Blue slide bars, or
    • by double-clicking a class in the editor window; the Edit Item (raster), Edit Item (polygon), Edit Item (segment) or Edit Item (point) dialog box will appear.
  2. To change the order of classes in the domain, choose the Domain command from the Edit menu. The Domain Class editor will be opened.
    In the Domain Class editor:
    • select Manual Ordering from the Edit menu; and
    • drag the listed domain items (i.e. your classes) in the domain to the desired positions.

    The position of the classes in the representation class editor will change accordingly.

  3. To apply the changes in the representation in a displayed map (with a domain to which this representation is linked), press the Redraw button in the map window. In this way, you can work with the representation editor and a map window simultaneously.

Multiple Selection

To select two or more items in sequence:

  • click the first domain item you want to select,
  • keep the left mouse button down, and
  • move the mouse pointer to the last domain item you want to select,
  • then release the left mouse button;

or

  • click the first domain item you want to select,
  • press and hold down the Shift key while you click the last domain item in the group.

To select two or more items out of sequence:

  • press and hold down the Ctrl key while you click each domain item.

To cancel a selected item:

  • press and hold down the Ctrl key while you click a selected domain item.
