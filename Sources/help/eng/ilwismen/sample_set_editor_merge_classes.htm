Merge Classes
 

Merge one class of training pixels in the sample set into another class.

When you find that the spectral values of two classes of training pixels are very similar to each other, you may decide to merge these classes into one class. This decision can be based on similar statistics of these classes in the sample statistics and/or overlapping classes in feature spaces.

Dialog box options:

Merge Class:

Select a class of training pixels that you want to merge into another class (i.e. specify the old class name).

Into Class:

Select the class into which the other class should be merged (i.e. specify the new class name).
