Polygon editor

 Options menu

Normal or Select mode   
 

 

Choose Normal/Select mode to select one or more polygons with the mouse.

After the selection, you can edit the code of the polygon(s) or copy the polygon(s) to clipboard.

When Normal/Select mode is active:

  • the mouse pointer appears as a hand,
  • an option bullet appears in front of this menu command, and
  • the Normal/Select mode button in the toolbar appears down.

 

Tip: You can also press the Esc key on the keyboard to go to Select mode.

To select a single polygon:

  • click the polygon you wish to select.

 

Tip: When you double-click a polygon, you can directly edit the value, class name or ID of the polygon.

To add polygons to a selection:

  • press and hold down the Ctrl key on the keyboard,
  • click the polygons you wish to select.

When finished, release the Ctrl key.

To clear part of a selection:

  • press and hold down the Ctrl+Shift keys on the keyboard,
  • click the selected polygons that should not be part of the selection.

To toggle between selecting and deselecting:

  • press and hold down the Shift key on the keyboard,
  • when you click a non-selected polygon, it will be selected,
  • when you click a selected polygon, it will be cleared from the selection.

To clear the whole selection:

  • press the Esc key on the keyboard, or
  • click in the white space next to any polygons.

 

Tip: To select all polygons, choose Select All from the Edit menu.
