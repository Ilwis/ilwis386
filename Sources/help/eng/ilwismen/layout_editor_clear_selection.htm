Layout editor

 Edit menu

Clear

 

Clear or delete a selected map view with additional annotation or a selected annotation element from the layout.

Note:

When you use Clear while you selected a map view for which you already added annotation (legends, map borders, North arrow, scale bar, scale text), then:

  • the map view and the annotation that has been added for this map view will be deleted from the current layout.

To select a map view or an annotation element:

  • click a map view or an annotation element in the layout editor or in the Layout Management pane.

To clear:

  • select a map view or an annotation element, and:
    • select Clear from the Edit menu, or
    • click the Clear button in the standard toolbar of the layout editor,
    • press the Del key on the keyboard, or
    • click the selected object with the right mouse button and choose Clear from the context-sensitive menu.

Tip:

If you do not wish to save any changes, leave the layout editor without saving the layout.
