Graph window

 Edit menu

Add Rose Diagram
 

Add another rose diagram to the current graph window.

This dialog box appears:

  • when you choose the Add Graph command from the Edit menu in a graph window which already displays a rose diagram;
  • when you click the Add Graph button in the toolbar of the graph window;
  • when you use the right mouse button in the white space of the Graph Management pane, and choose the Add Graph command from the context-sensitive menu.

Dialog box options:

Table:

Select a table which contains the columns that you wish to show as a rose diagram.

Angle:

Select a value column which represents angles (X-axis).

Note: The column used for the X-axis of a rose diagram is supposed to contain values between 0-180° or between 0-360°.

Value:

Select a value column to be used as Y-axis.

Tips:

  • To modify the appearance or the name of a rose diagram, double-click the graph layer in the Graph Management pane: the Graph Options - Rose Diagram dialog box will appear.
  • To adapt the title, X-axis, or Y-axis of the rose diagram, double-click the appriopriate item in the Graph Management pane or in the graph window itself: the Title, X-axis (rose diagram), or the Y-axis (rose diagram) dialog box will appear.

    • The grid lines of the rose diagram can be modified in the X-axis and Y-axis dialog boxes.

  • To create a new rose diagram in a new graph window, choose Create Graph from the File menu in the graph window: the Create a rose diagram dialog box will appear.
  • To save the contents of a graph window as a graph, choose Save or Save As from the File menu in the graph window.
