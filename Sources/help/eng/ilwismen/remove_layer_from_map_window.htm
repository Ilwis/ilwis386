Remove Layer

 

Remove a layer from the map window.

 

  • First, select the layer that you wish to remove in the Layer Management pane.
  • Then, either:
    • choose the Remove Layer command from the Layers menu, or
    • click the Remove Layer button in the toolbar of the map window.

You can also remove a layer from the map window by selecting a layer in the Layer Management pane with the right mouse button, and then choose the Remove Layer command from the context-sensitive menu.

 

Tip: To temporarily not display a layer in the map window, you can also switch the layer off in the Layer Management pane.
