Insert/Edit Scale Text

 

Insert or edit scale text for a selected map view in your layout, for instance 1:100000.

Scale text is always the exactly same as the scale that you set for the selected map view; you can set the scale for a map view in the Layout Management pane. After you have set a scale for a map view, it is advised to fix or keep the scale.

Dialog box options:

Color:

Select a color for the scale text; select a color from the list. To create your own color, select 'Custom' at the bottom of the drop-down list, or click the create button ; the Windows Color dialog box will appear.

Font name:

Select the font in which the scale text should appear; you can select an installed font from the list.

Font size:

Type a value for the font size (pt) in which the scale text should appear.

Bold:

Select this check box to have the scale text appear in bold.

Italic:

Select this check box to have the scale text appear in italics.

Transparent:

Select this check box to simply obtain the scale text; if you position the scale text on layer, i.e. a map view, or an annotation element, the other layers will still be visible 'below' the scale text (minimum loss of other displayed layers).

Clear this check box to create some (white) space on top of which the scale text will be positioned.

Tips:

  1. In the Layout editor, scale text can be sized by dragging for instance one of the corner size handles of a selected text to a new position. The mouse pointer changes into a double-sided arrow when at the right position to start dragging. The font size in which the text is displayed will be automatically adapted to the new size.
  2. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
