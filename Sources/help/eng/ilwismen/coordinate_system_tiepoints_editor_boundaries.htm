Coordinate system Tiepoints editor

Boundaries

 Edit menu
 

Change the minimum and maximum XY-values of the boundaries of the current coordinate system tiepoints (artificial coordinates).

Dialog box options:

Description:

Type a description for the coordinate system. The description is visible on the status bar of the Main window when moving the mouse pointer over the coordinate system in the Catalog.

Min X, Y:

Type the minimum X-coordinate and the minimum Y-coordinate for this coordinate system; the boundary values should more or less coincide with your artificial coordinates.

Max X, Y:

Type the maximum X-coordinate and the maximum Y-coordinate for this coordinate system; the boundary values should more or less coincide with your artificial coordinates.

Defaults:

When you click the Defaults button, the coordinate boundaries of the related coordinate system are transformed according to the current transformation method and tiepoints. This will usually show the coordinate boundaries of the map with artificial coordinates. In most cases, you will only use the defaults button to check whether the transformation is more or less correct.

In case you used the Defaults button and the boundary values changed to rather unexpected values: click the Cancel button to prevent changes.
