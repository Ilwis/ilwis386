Insert/Edit North Arrow

 

Insert or edit a North arrow for a selected map view in your layout.

The scale bar that will be created is based on the georeference used by the raster map in the selected map view; or if the map view does not contain a raster map, by the georeference of the map view itself.

Dialog box options:

Arrow type:

Select the type of North arrow you wish to obtain (see examples below).

Simple:

Choose this option button to obtain a North arrow in the shape of a straightforward arrow.

Cross:

Choose this option button to obtain a North arrow in the shape of a cross/plus sign.

Wing:

Choose this option button to obtain a North arrow in the shape of a wing.

Compass:

Choose this option button to obtain a North arrow in the shape of a compass-card; also choose whether the compass-card should point to 4, 8 or 16 directions.

Half:

For a Simple North Arrow: select this check box if you wish to obtain only half an arrow head.

For a Wing North Arrow: select this check box if you wish to obtain half of the wing in white and the other half of the wing in the selected color.

Circle:

Select this check box if you wish to obtain an extra circle in or around your North arrow.

Width (mm):

In case of a 'simple' or a 'cross' North arrow, specify the line width (mm) of the main part of the North arrow. Value 0 means that the North arrow will be drawn and printed as fine as possible (hairline).

Color:

Select a color for the North arrow and optional accompanying text. To create your own color, select 'Custom' at the bottom of the drop-down list, or click the create button; the Windows Color dialog box will appear.

Text:

Select this check box if you wish to obtain additional text for your North arrow such as N, S, E, W.

Clear this check box if you wish no text at all for the North arrow.

All sides:

Select this check box if you wish to obtain text on all sides of the North arrow, i.e. N, S or N, S, E, W.

Clear this check box if you only wish to obtain the letter N for North.

Font name:

Select the font in which additional text for the North arrow should appear; you can select an installed font from the list.

Font size:

Type a value for the font size (pt) in which additional text for the North arrow should appear.

Bold:

Select this check box to have additional text for the North arrow appear in bold.

Italic:

Select this check box to have additional text for the North arrow appear in italics.

Examples:

 

 
1 2 3 4  5

 

Arrow 1:

Simple North arrow with text.

Arrow 2:

Cross North arrow with text.

Arrow 3:

Wing North arrow, half, with text.

Arrow 4:

Wing North arrow with circle and text.

Arrow 5:

Compass North arrow, 8 headed, with text on all sides.

Tips:

  • In the Layout editor, a North arrow can be sized by dragging one of the corner size handles of a North arrow to a new position. The mouse pointer changes into a double-sided arrow when at the right position to start dragging. The North arrow itself will be automatically adapted to the new size; you have to specify a new font size for accompanying text yourself.
  • To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
