SMCE window

 File menu

Open a criteria tree

 

Open an existing criteria tree in the current SMCE window.

If the criteria tree that is currently displayed in the SMCE window has changed, first a question will be asked whether the current criteria tree should be saved or not.

This dialog box appears:

  • when you choose Open from the File menu in the SMCE window;
  • when you click the Open button in the toolbar of the SMCE window;
  • when you choose the Open an existing Criteria Tree option from the Spatial Multicriteria Evaluation wizard.

Dialog box options:

Select a criteria tree:

Select a criteria tree that you wish to open in the SMCE window.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The SMCE window will show the selected criteria tree. You can modify the criteria tree according to your wishes.

Tips:

  1. To open a criteria tree in another new SMCE window, you can:
    • double-click a saved criteria tree in a Catalog,
    • click the right mouse button on a criteria tree in a Catalog and choose Open from the context-sensitive menu,
    • start the SMCE application from the ILWIS Main window, and choose Open an existing criteria tree,
    • choose Open from the File menu in the Main window, or
    • click the Open button in the standard toolbar of the Main window.
  2. You can also open a criteria tree by typing the Open or Show command on the command line of the Main window. For more information, see Appendices : ILWIS commands.
