New Catalog
 

Open a new (directory) Catalog so that you can work with multiple Catalogs. A small dialog box appears in which you can choose a directory.

In ILWIS 3, you can use multiple Catalogs. For each Catalog, you can choose to view the contents in List View or in Details View; you can indicate which object types you wish to see (Show); and you can sort the objects on Name, Type, Modification Date, Description, etc. (Sort By). Multiple Catalogs allow you to work easily with multiple directories, to drag and drop objects between Catalogs to copy these objects, etc.

Tips:

To open a new Catalog, you can also:

  • use the Open New Catalog button in the standard toolbar of the Main window, or
  • click a directory in the current Catalog with the right mouse button, and select Open from the context-sensitive menu, or
  • click a folder/directory in the Navigator with the right mouse button, and select New Catalog from the context-sensitive menu.

When you open a map list or an object collection, their contents are always shown in a new Catalog.

Tips:

To replace the contents of the current Catalog by those of another directory:

  • click the Navigator tab in the Main window, and select another directory,
  • double-click a directory in the current Catalog,
  • click a directory in the current Catalog with the right mouse button, and select Open Here from the context-sensitive menu.
