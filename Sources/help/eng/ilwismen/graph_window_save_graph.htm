Save a graph

 

Save the current graph.

You can use the Save command on the File menu of the graph window, but you can also click the Save button in the toolbar of the graph window.

 

  • When you have saved the graph earlier, the graph will be directly saved with the same name as before.
  • When you save a new graph for the first time, the Save As dialog box will appear. In that dialog box, you can enter a new name for the new graph.

 

Note: The Save command and the Save button are only available when the currently displayed graph has been changed since the last save.
