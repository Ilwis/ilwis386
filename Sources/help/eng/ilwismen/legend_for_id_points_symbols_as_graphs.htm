Insert/Edit Legend (ID point maps)

Symbols as pie charts, bar charts or composed bars

 

When an ID point map that has an attribute table is displayed as stretched pie charts, bar charts or composed bars, you can add two types of legends: a size legend and a color legend.

Choose which type of legend you wish to add; you can add the other legend type later on.

Dialog box options:

Size legend:

To add a legend which explains the size of pie charts, bar charts or composed bars. The Display Options - Legend (symbols as graphs, size) dialog box will appear.

Color legend:

To add a legend which explains the colors in the pie charts, bar charts, or composed bars. The Display Options - Legend (symbols as graphs, color) dialog box will appear.
