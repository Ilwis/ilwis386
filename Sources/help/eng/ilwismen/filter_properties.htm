Filter Properties

The Filter Properties sheet lists all kind of important information on a selected filter. In fact, all relevant information that is stored in the object definition file of the filter is displayed. For more information, refer to Basic concepts : properties of objects.

The Filter Properties sheet appears:

  • when you click a filter in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a filter in a Catalog, and choose Properties from the Edit menu in the Main window;
  • when you open a filter, and choose Properties from the File menu.

General tab:

Object type and name:

Filter "object name". When you click this flat button, the filter will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this object was last changed.

Object size:

Total size of the filter on disk in bytes (.FIL).

Read only:

Select this check box to make the object read only. Clear this check box to make the object changeable.

Description:

Description of the object; freely editable. The description of a map view appears in the title bar of a map window when the map view is displayed. Further, for all objects, the description is visible on the status bar of the Main window when moving the mouse pointer over the object in the Catalog.

Used By tab:

Object type and name:

Filter "object name". When you click this flat button, the filter will be (re)opened.

Used by:

In a tree view, it is listed by which other objects this filter is used.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
