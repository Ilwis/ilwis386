Symbol

Point maps with a value domain

Specify symbol type, color, line width and line color for all points. The size of symbols is either the same for all points (no stretch) or proportional to the point values (stretch).

This dialog box appears when you open a value point map, or display a point map by a value attribute column. In the Display Options - Point Map dialog box, you clicked the Symbol button.

Dialog box options:

Symbol Type:

Select the ILWIS Simple symbols or any installed symbol font; open the list box by clicking it. Windows symbol fonts are for example Symbol, Wingdings, etc.

In case of ILWIS Simple symbols:

Symbol ...:

Select a symbol type from the list box: circle, square, diamond, delta up, delta down, plus, minus, or cross.

Fill Color:

For circle, square, diamond, delta up, delta down: select a color to fill the symbols with; open the list box by clicking it. To create your own color, select Custom at the bottom of the drop-down list. To obtain symbols which are not filled, select Transparent from the drop-down list.

Line width:

Specify a value for the line width of the symbols. Line width is expressed in screen pixels.

Color:

Select a color to draw the (outer lines of) symbols; open the list box by clicking it. To create your own color, select Custom at the bottom of the drop-down list, or click the create button.

In case of a Windows symbol font:

Symbol box:

In case of a Windows symbol font, select a symbol from the symbol box by clicking it.

Color:

Select a color to draw the symbols; open the list box by clicking it. To create your own color, select Custom at the bottom of the drop-down list, or click the create button.

For all symbol types:

Stretch:

Select this check box if you want to size the point symbols proportionally to the point values. Specify the stretch range; by default, the 0% interval values are given. When you want to add a legend later on, it is advised to choose round values.

Clear this check box, if you want to display all points with the same size.

Size:

In case of no stretching: Specify a value for the symbol size of all points.

In case of stretching: Specify a symbol size for the smallest value in the stretch range and a symbol size for the largest value in the stretch range.

Linear, Logarithmic:

In case of symbol size stretching, choose between Linear or Logarithmic stretching.

Radius, Area:

In case of symbol size stretching, choose to make the point values proportional to the Radius or to the Area of the symbol.

Tip:

When point values are stretched proportionally to symbol sizes, you can add a legend to this point map. Choose Add Annotation, Legend from the Layers menu in the map window. The Display Options - Legend (point values as symbols) will appear.
