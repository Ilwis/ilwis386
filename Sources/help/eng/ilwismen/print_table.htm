Print Options
 

Print (a selection of) the current table on a printer.

This dialog box appears:

  • when you choose Print from the File menu in a table;
  • when you click the Print button in the toolbar of a table window.

Dialog box options:

Header on all pages:

Select this check box to obtain a header on each page. The header will read: Table "Table name".

Clear this check box when you do not want to obtain a header on the print out(s).

Leftmost column on all pages:

Select this check box to print the class names, IDs, or record numbers (i.e. the domain items) of your table; this will appear to the left of any printed columns and fields.

Clear this check box when you only wish to print columns and/or fields.

Font Name:

Select one of the fonts from the list of installed fonts.

Font Size:

Type a font size for printing.

Bold:

Select this check box when you wish to print columns/fields in Bold.

Italics:

Select this check box when you wish to print columns/fields in Italics.

All:

When you made a selection in the table, select this option to print the entire table.

Current selection:

When you made a selection in the table, select this option to print the selected part of the table only.

When you click OK, a standard Windows Print dialog box will follow in which you can select a printer and specify some settings for the selected printer.

When you click OK in the Windows Print dialog box, the table or the table selection will be printed.

 

Tip: To install a printer and choose a default printer, see your Windows documentation.
