Polygon editor

 Edit menu

Copy to clipboard
 

Copy selected polygons with their coordinates and their class names, identifiers or values to the clipboard. In the clipboard, the information is available both in text format and in picture format.

Subsequently, the copied polygons can be pasted as segments into a segment editor, as points into a point editor and/or into any other application which uses the picture format or the text format (e.g. a word processor or a data base).

To copy:

Select one or more polygons, and:

  • choose Copy from the Edit menu, or
  • press Ctrl+C on the keyboard.

For more information on selecting polygons, see Polygon editor : Normal/Select mode.

Tips:

  • To copy the active window including the window borders, title bar, menu bar and status bar as a bitmap to clipboard, press Alt+PrintScrn on the keyboard.
  • Also from within a map window, you can copy (the displayed part of) a map as a picture to the clipboard; choose Copy from the Edit menu in a map window. Coordinate information will not be available however.
  • You can use Windows' Clipboard/Clipbook Viewer to inspect the current contents of the clipboard. For more information, see the Windows Help.
