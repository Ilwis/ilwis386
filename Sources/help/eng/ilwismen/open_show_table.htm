Show Table

Select a table and show/open it in a new table window.

This dialog box appears:

  • when you choose Show Table from the Operations menu in the Main window,
  • when you choose Open Table from the File menu in a table window,
  • when you click the Open Table button in the toolbar of the Main window.

Dialog box options:

Select a table:

Select a table which you want to open in a new table window.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

When the selected table is not open yet, the table is opened in a new table window.

When the selected table is already open, the table is restored and the table window becomes the active window.

Tip:

To open a table, you can also:

  • double-click a table in the Catalog, or
  • click the right mouse button on a table in the Catalog and select Open from the context-sensitive menu, or
  • drag a table from the Catalog to the Show item in the Operation-list.

Further you can type a command on the command line of the Main window. For more information, see Appendices : commands.
