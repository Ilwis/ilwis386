Segment editor

 File menu

Save   
 

 

Save all changes in the segment map until this moment, so that they cannot be undone anymore by using the Undo All Changes command.

Using this Save command is optional, as any change is directly and immediately stored. Thus, you can also simply close the editor and not use the Save command.

When using the Save command:

  • the segment map will be saved as it is at the moment;
  • any changes you made to this segment map prior to using this Save command cannot be undone anymore through the Undo All Changes command;
  • only changes you make to the segment map after using the Save command can be undone by using the Undo All Changes command.

 

Tip: To undelete specific segments, use the Undelete segments command.
