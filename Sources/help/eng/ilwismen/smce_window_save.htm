Save a criteria tree

 

Save the current criteria tree.

You can use the Save command on the File menu of the SMCE window, but you can also click the Save button in the toolbar of the SMCE window.

 

  • When you have saved the criteria tree earlier, the criteria tree will be directly saved with the same name as before.
  • When you save a new criteria tree for the first time, the Save As dialog box will appear. In that dialog box, you can enter a new name for the new criteria tree.
