Polygon editor

 File menu

Apply labels
 

Assign names to polygons according to a label point map.

For each label point in the point map, the surrounding polygon is determined; then the class name, ID, or value of the label point is assigned to that polygon. When multiple labels are found within one polygon, the last point (as seen when you display the point map as a table) is assigned to the polygon.

Dialog box option:

Label points:

Select the name of a point map which contains labels that you want to assign to the polygons.

Tip:

To assign label points to polygons, you can also use the Assign labels to polygons (LabelPol) operation.
