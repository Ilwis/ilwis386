Setup Digitizer wizard

Automatic Detection

 < Back | Next >
 

This page helps you to check:

  • whether you selected the correct port;
  • whether you specified for the Port Settings exactly the same values as you used for the configuration of the digitizer tablet itself (settings for Baud Rate, Parity, Data Bits and Stop Bits);
  • whether your digitizer is configured to send coordinates in Run Mode;
  • whether your digitizer is configured so send coordinates in ASCII format;
  • whether the digitizer configuration for Data Rate is accepted.

If the communication between digitizer and computer port is correct, you will assign the digitizer cursor buttons with the Automatic Setup.

Received digitizer coordinates (check whether you have communication):

Do not press any buttons on the digitizer cursor. Move your digitizer cursor over the tablet. Inspect the string that is listed on the first line after 'Received'.

You have communication between the digitizer and the computer:

  • when the first line of this page reads:

    • Received: coordinates as readable numbers

  • when coordinates change from about 0,0 (when moving the digitizer cursor into the lower left corner of the tablet) to greater positive figures (when moving the digitizer cursor into upper right corner of the tablet).

The message Trying Automatic Detection will appear.

The digitizer setup is proceeding satisfactorily: you can continue to have ILWIS recognize the digitizer cursor buttons. Follow the instructions for Automatic Detection (below).

You definitely don't have communication between your digitizer and your computer:

  • when the first line of this page reads:

    • Received: Nothing

  • when the first line of this page reads:

    • Received: coordinates as unreadable characters

  • when coordinates do not change when you move the digitizer cursor over the tablet;
  • when the message Press Button 0 does not appear.

Go to the Trouble shooting section (below).

Trying Automatic Detection:

When you have communication and when the message Press Button 0 appears:

  • move the digitizer cursor into the upper right corner of the tablet (but still inside the active area);
  • press button 0 on the digitizer cursor;
  • when asked for, also press buttons 1, 2, and 3 on the digitizer cursor.

You will automatically go to the last page of the Setup Digitizer wizard: the Setup Digitizer - Size, Resolution, Final Check page.

If you experience any strange behaviour when using Automatic Detection, you may go Back to the Port Settings page and then with Next to this Automatic Detection page and try again. Otherwise go to the Trouble shooting section (below).

Trouble shooting:

  • The first line on this page reads:

    • Received: Nothing

    Problems: You may have selected the wrong Port number; the Port Settings (such as Baud Rate, Data Bits, etc.) may not correspond to the digitizer configuration settings; the digitizer may not send coordinates in ASCII format; the digitizer may not be configured to work in Run Mode.

    Received: coordinates appear as unreadable characters

    Problem: Digitizer may not send coordinates in ASCII format.

  • Coordinates do not change when moving the digitizer cursor over the tablet.

    Problem: Digitizer may not be configured to work in Run Mode.

  • The message Press Button 0 does not appear.

    • Problem: Digitizer may not be configured to work in Run Mode and/or reduce the Data Rate.

  • Coordinates only appear after pressing a button on the digitizer cursor.

    • Problem: Digitizer is not configured to work in Run Mode.

    Port Settings may be already correct.

  • The page does not respond to pressing buttons 0, 1, 2, 3 on the digitizer cursor when Trying Automatic Detection.

    Try going Back to the Port Settings page and then with Next to this Automatic Detection page and try again.

    If this does not solve your problem: check all digitizer settings.

Remedy:

  1. Check the configuration settings of the digitizer tablet itself: Run mode, ASCII format, Baud Rate, Data Bits, Parity, Stop Bits and write the figures down;
  2. Press the < Back button and return to the Setup Digitizer - Port Settings page;
  3. On the Setup Digitizer - Port Settings page, make sure that the settings for the port (Baud Rate, Data Bits, Parity, Stop Bits) is exactly the same as the settings you used for the digitizer tablet, then press the Next button;
  4. On this Setup Digitizer - Automatic Detection page again, check whether you communication:
    • if you have communication now, continue with the Automatic Detection.
    • if you don't have communication, check whether all cables are properly connected, whether you use the correct port number, cancel the setup, switch the digitizer off and on again, reboot your computer and try again.
  5. Keep trying until you have communication.

Tip:

It is strongly advised to first ensure the communication between the digitizer and the port and then to choose the ILWIS Automatic Detection. Trying the manual setup button on this page will not make things easier.

Manual setup (advanced):

Press the Manual button if you are sure that you have communication, but when Automatic Detection is not able to recognize a certain 'difficult' digitizer Format. You can unravel the received ASCII string yourself: the Setup Digitizer - Manual Setup page will appear.
