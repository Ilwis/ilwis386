Double-Click Action
 

Select the action to be performed when double-clicking a mapping unit in a domain Class map or an element in domain ID map which is displayed in the map window. The double-click action works on point, segment, polygon and raster maps.

First, you must make sure that the Info check box has been selected for the data layer you wish to use. You can select the Info check box in the layer's Display Options dialog box.

On a double-click, it is attempted to find the (defined) class name or ID of the mapping unit that was double-clicked. When multiple data layers with a selected Info check box are displayed, all data layers will be searched from top to bottom. You can see the class name or ID that will be found by pressing the left mouse button in the map window (Info).

 

 Attributes:

Select this option to view and/or edit the attributes of a mapping unit or element when you double-click it. The map should have a Class or an ID domain, and an attribute table should be linked to the map or to the domain of the map.

If a class name or an identifier is found, then the attributes of this class or identifier are shown in a separate window. You can also edit the attributes there. If no class name or ID is found, or if no attribute table is found nothing happens.

 Representation:

Select this option to edit the representation (colors, line width, patterns, etc.) of a mapping unit when you double-click it. The map should have a Class domain.

If a class name is found, then an Edit Representation item dialog box appears in which you can select another color, etc. for the class. Changes will be directly applied in the map window. If no class name is found, nothing happens.

 Execute Action:

Select this option to execute an action when you double-click a certain mapping unit or element in a displayed map. The map should have a Class or ID domain, and an attribute table should be linked to the map or to the domain of the map. Furthermore, the attribute table should contain a column named Action (using the String domain).

In this Action column, and for the classes or the ID which should execute a certain action when double-clicked in a map window, you should have specified:

  • the map name or the table name (including path and extension .MPR, .MPA, .MPS, .MPP, .TBT) that should be opened, or
  • the file name of the GIF image, picture, sound or video (including path and extension) that you wish to see/hear.

For any registered file type in Windows, the program will be used that is specified for the default 'Open' action of that registered file type (for more information, see the Windows Help). Native ILWIS extensions are automatically registered to be opened with ILWIS. If, on a double-click, a class name or identifier is found, then the specified map, table, GIF image, bitmap, photograph of the site (.TIF etc.) will be shown, or the specified sound or video will be played, etc.
