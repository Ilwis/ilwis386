Domain ID editor

Change Prefix

 Edit menu
 

Change the prefix of all currently prefixed items in this ID domain.

Dialog box options:

Prefix:

Change the existing prefix of IDs in this domain into another prefix.
The colon character : may not be used in a prefix.
You are not allowed to specify an empty prefix.

The new prefix will be applied to all IDs in the domain that had the old prefix.

A prefix appears 'integrated' with the 'Name' of IDs; IDs will be proceeded by the new prefix as in My Prefix 1, My Prefix 2, etc.

This is visible when pressing the left mouse button (Info) in a map window; in record names of attribute tables that use this domain; and in legends. When enclosed in double quotes, you can use the combination of a prefix and an ID in calculations.

Mind:

  • When you edited individual IDs in this domain, you may have deleted the prefix of these items. When using the Change Prefix command, items without prefix do not obtain the new prefix.
  • As you cannot specify an empty prefix, it is not possible to delete the prefix of all prefixed IDs in this domain.
