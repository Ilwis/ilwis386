Open Stereo Pair

Open a stereo pair.

1. To open an existing stereo pair in the Epipolar Stereo Pair - Creation window:

  • choose Open Stereo Pair from the File menu in the Epipolar Stereo Pair - Creation window, or
  • click the Open button in the toolbar of the Epipolar Stereo Pair - Creation window.

In the Epipolar Stereo Pair - Creation window, you will be able to view or edit position of the fiducial marks, principal points, transferred principal points, and optional scaling points.

2. To open an existing stereo pair in the Stereo Pair - Stereoscope window:

  • open the Operations menu in the Main window, and choose Visualization, Stereoscope,
  • double-click the Stereoscope item in the Operation-list,
  • open the Visualization item in the Operation-tree and double-click the Stereoscope item,
  • choose Open Stereo Pair from the File menu in the Stereo Pair - Stereoscope window, or
  • click the Open button in the toolbar of the Stereo Pair - Stereoscope window.

In the Stereo Pair - Stereoscope window, you will be able to view the stereo pair with a stereoscope.

Dialog box options:

Stereo pair:

Select the stereo pair that you wish to show.

Tips:

To open a stereo pair in the Epipolar Stereo Pair - Creation window, you can also:

  • click the right mouse button on a stereo pair in a Catalog and select Edit from the context-sensitive menu.

 

To open a stereo pair in the Stereo Pair - Stereoscope window, you can also:

  • double-click a stereo pair in the Catalog, or
  • click the right mouse button on a stereo pair in a Catalog and select Open from the context-sensitive menu.
  • drag a stereo pair from the Catalog to the Stereoscope item in the Operation-list.

 

To open a stereo pair, you can furthermore type a command on the command line of the Main window. For more information, see Appendices : commands.
