Create a georeference 3D

Create a georeference 3D in order to obtain a three-dimensional view of your study area; a Digital Terrain Model (DTM) also called a Digital Elevation Mode (DEM) is required. For more information on obtaining a DTM/DEM, see How to calculate a Digital Elevation Model.

This dialog box appears:

  • when you click the create georeference button in the Display 3D dialog box or in the Apply 3D dialog box.

Dialog box options:

Georeference name:

Type a new name for the georeference 3D.

Description:

Type a description for the georeference. The description is visible on the status bar of the Main window when moving the mouse pointer over the georeference in the Catalog.

Rows Columns:

Type values for the size of the georeference 3D. During Display 3D, the map window will use the same width/height ratio. The Apply 3D operation will produce a map with these dimensions.

DTM:

Select a raster map with height values (Digital Terrain Model/Digital Elevation Model) from which the three-dimensional picture should be constructed. For more information on creating a DTM/DEM, see How to calculate a Digital Elevation Model.

By creating a georeference 3D, the size of the window in which the 3D view will be displayed and the name of the DEM/DTM to use are specified. Also, some default 3D view parameters are assigned but these are not visible to the user at this point. These parameters such as view angle, height scaling and rotations can be edited when you display the 3D view and open the Georeference 3D editor.

When you click the OK button, the Display 3D dialog box or the Apply 3D dialog box is re-activated.
