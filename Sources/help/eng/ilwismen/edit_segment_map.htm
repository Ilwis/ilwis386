Edit Segment Map
 

Select a segment map that you want to edit while using the current contents of the map window as a background. This is also called screen digitizing.

This dialog box appears when from the Edit menu in a map window, you choose Edit Other Map, Segment editor.

Dialog box options:

Segment map:

Select the segment map you wish to edit.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The Segment editor is started; you can start editing the selected segment map.

Tips:

  1. Dependent maps and maps marked as read-only cannot be edited. First break the dependency link or remove the read-only flag in the Segment Map - Properties sheet.
  2. To directly edit a segment map:
    • in the Catalog, select an editable segment map, then open the Edit menu in the Main window and choose Edit Object, or
    • in the Catalog, click an editable segment map with the right mouse button and choose Edit from the context-sensitive menu, or
    • in a map window where the segment map is displayed, open the Edit menu and select Edit Layer.
