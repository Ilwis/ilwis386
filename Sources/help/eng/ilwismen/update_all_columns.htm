Columns menu

Update All Columns   
 

 

Recalculate all dependent columns which are not up-to-date anymore because of changes in columns on which they depend. Thus, on the basis of the definition of each dependent column, all dependent columns are made up-to-date.

For an introduction on dependent data, see Basic concepts : dependent data objects.
