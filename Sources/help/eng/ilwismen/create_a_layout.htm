Create a layout

A layout is required to print maps with annotation. A layout is a virtual sheet of paper on which you can make a composition of one or more map views, specify the scale on which these map views should be printed, and on which you can insert annotation.

In the layout editor, you can add to a layout:

  • one or more map views, each of which may include one raster map and/or one or more point maps, segment maps and/or polygon maps, annotation text layers, grid lines and/or graticules;
  • texts such a titles;
  • boxes;
  • pictures and/or bitmaps;
  • a page border.

For each map view, you can add to the layout:

  • a legend for a layer in a map view;
  • a map border, including coordinates and ticks for grid lines and/or a graticule;
  • a North arrow;
  • a scale bar and scale text.

To create a layout:

  • In the Main window, choose the Create Layout command from the File menu;
  • In a Catalog, click a map view with the right mouse button and choose the Create Layout command from the context-sensitive menu;
  • In the Operation-tree, open the Create item, and select the New Layout command;
  • In the Operation-list, double-click the New Layout command;
  • In a map window, choose the Create Layout command from the File menu.

The Layout editor is started.

While working in the layout editor, or at least when you are finished editing, you need to save the layout.
