SMCE window

 Generate menu

Histogram Sliced   
 

 

Show the histogram(s) of the currently selected sliced output map(s).

 

  • When you select the main goal and press the Histogram Sliced button, the histograms of the sliced SMCE output maps will be calculated and shown.
  • When you select a sub goal with named output maps and press the Histogram Sliced button, the histograms of the sliced output maps of the sub goal will be calculated and shown.
  • When you select a single output map and press the Histogram Sliced button, the histogram of the selected sliced map will be calculated and shown.

After selecting the main goal, a sub goal or a single map in the criteria tree viewer, you can use:

  • the Histogram Sliced button in the toolbar,
  • the Histogram Sliced command from the Generate menu, or
  • the Histogram Sliced command from the context-sensitive menu.

Notes:

  • Histogram Sliced is not available when the SMCE output maps have not been sliced.
  • Histogram Sliced will force the calculation of sliced SMCE output maps when 'Define' was used during slicing.
  • Histogram Sliced will calculate the histogram of a sliced map if the histogram is not available yet.
