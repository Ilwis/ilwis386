Graph window

 File menu

Save Graph As ...

 

Save the current graph with a new name.

This dialog box appears:

  • when you choose Save As from the File menu in a graph window;
  • when you have not saved the graph before, and you choose Save from the File menu or click the Save button in the toolbar.

Dialog box options:

Save in:

If necessary, select another directory or folder where the graph should be stored.

File name:

Type a new name for your graph. It is not necessary to specify the extension.

Save as type:

You cannot select anything else than type Graph.
