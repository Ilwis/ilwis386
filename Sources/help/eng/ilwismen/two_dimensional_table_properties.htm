Two-dimensional Table Properties

The Two-dimensional Table Properties sheet lists all kind of important information on a selected two-dimensional table. In fact, all relevant information that is stored in the object definition file of the two-dimensional table is displayed. For more information, refer to Basic concepts : properties of objects.

The Two-dimensional Table Properties sheet appears:

  • when you click a two-dimensional table in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a two-dimensional table in a Catalog, and choose Properties from the Edit menu in the Main window;
  • when you open the two-dimensional table, and choose Properties from the File menu of the two-dimensional table viewer/editor;
  • when you open a two-dimensional table, and click the Properties button in the toolbar of the two-dimensional table viewer/editor.

2-dim Table tab:

Object type and name:

2-Dimensional Table "Name". When you click this flat button, the two-dimensional table will be (re)opened.

Primary:

Shows the domain type, name and description of the primary domain, this is the domain that is used for the rows in the two-dimensional table.

Secondary:

Shows the domain type, name and description of the secondary domain, this is the domain that is used for the columns in the two-dimensional table.

Domain:

Shows the type and name of the domain that is used for the fields, i.e. the contents, of the two-dimensional table.

Value Range:

In case the two-dimensional table uses a value domain for its fields, the minimum and maximum value and precision of values that can be used in the fields is listed.

General tab:

Object type and name:

2-Dimensional Table "Name". When you click this flat button, the two-dimensional table will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this two-dimensional table was last changed.

Object size:

Total size of the two-dimensional table on disk in bytes (.TA2 and T2#).

Read only:

Select this check box if this two-dimensional table should be read only. You are then not allowed to edit or delete the two-dimensional table. Clear this check box if you want to be able to edit the two-dimensional table.

Description:

Optionally, change the description of this two-dimensional table. The description of a two-dimensional table appears in the title bar when the two-dimensional table is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the two-dimensional table in the Catalog.

Used by tab:

Object type and name:

2-Dimensional Table "Name". When you click this flat button, the two-dimensional table will be opened.

Used by:

In a tree view, it is listed by which other objects this two-dimensional table is used.

Two-dimensional tables are never dependent objects.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
