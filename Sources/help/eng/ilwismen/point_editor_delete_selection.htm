Point editor

 Edit menu

Delete
 

Delete selected points from the current point map.

To delete:

Select one or more points, and:

  • choose Delete from the Edit menu, or
  • choose Delete from the context-sensitive menu, or
  • press Del on the keyboard.

For more information on selecting points, refer to Select mode.

A message box will appear asking you to confirm the deletion of the selected points.

Answer with:

  • yes: to delete the selection;
  • no: to return to the editor without changes.

Tip:

To recover points which were accidentally deleted, you can use:

  • Undo all changes: to restore all changes made in the point editor until the last time you used Save.
