Georeference Tiepoints editor

Customize

 Edit menu
 

Customize the way in which tiepoints, also called ground control points, are displayed on the background image in the Tiepoint editor.

Dialog box options:

Active color:

Select a color for well positioned tiepoints and for the active tiepoint; open the list box by clicking it. To create your own custom color, click the create button or select Custom at the bottom of the drop-down list.
A tiepoint is considered well positioned if:
Ö(DRow2 + DCol2) <= (1.2 * Sigma).

Medium error:

Select a color for tiepoints with medium error.
A tiepoint is considered to have a medium error if:
(1.2 * Sigma) < Ö(DRow2 + DCol2) <= (2 * Sigma).

Large error:

Select a color for tiepoints with a large error.
A tiepoint is considered to have a large error if:
Ö(DRow2 + DCol2) > (2 * Sigma).

Passive color:

Select a color for the non-active tiepoints; open the list box by clicking it. To create your own custom color, click the create button or select Custom at the bottom of the drop-down list.

Fiducial marks:

When editing a georef orthophoto and the Locate Fiducial Marks dialog box is open: select a color for the fiducial marks and their numbers; open the list box by clicking it. To create your own custom color, click the create button or select Custom at the bottom of the drop-down list.

Symbol size:

Type a value (integer > 0) for the size of the symbols that represent your tiepoints. Tiepoints are always represented by a plus sign.
