Import ASTER level 1A and/or level 1B images by using ILWIS Map Import.

The dialog box described below appears when you choose Import Map

  • from the File menu in the Main window;
  • in the Operation-tree or;
  • in the Operation-list

and select in the Import dialog box Import Format Aster (1A/1B).

When using Import ASTER, you can choose:

  • to perform a genuine import and translate the external file to an ILWIS data format, or
  • to keep the external file in its original format (Use As).

For more information, see the Convert to ILWIS data format option below.

After you used Import Map with Import Format Aster (1A/1B), you will obtain an ILWIS object collection . The object collection contains the ILWIS objects that are the result of the import. You can open an object collection by double-clicking it in a Catalog; subsequently you can also open the imported objects by double-clicking.

Import dialog box options:

Select a file:

Select the file that you wish to import.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

Import format:

 ASTER (1A/1B)

When you select another Import format, please refer to the other Import topics in the Help; you can click one of the links in the header of this topic.

Output file name:

Type a name for the output ILWIS object collection. Do not specify an extension; ILWIS will add its own extensions. The imported data will be part of the output object collection.

Show:

Select this check box when you like to directly open the resulting object collection.

Use Radiances (Wm-2sr-1µm-1):

 

For ASTER level 1A images:

  • Select this check box if you want to convert the raw Digital Numbers of the ASTER HDF file to radiometric corrected radiance values.
  • Clear this check box if you want to convert the raw Digital Numbers of the ASTER HDF file to sensor calibrated DN values.

For ASTER level 1B images:

  • Select this check box if you want to convert the radiometrically corrected DN values of the ASTER HDF file to radiance values.
  • Clear this check box if you want to use the radiometrically corrected DN values of the ASTER HDF file.

Notes:

  • Sensor calibrated DN values are not the actual digital unprocessed numbers of the file but radiometric corrected radiance values scaled back into an appropriate range.
  • For more technical information about radiometric and geometric corrections of ASTER images refer to ASTER Import : Radiometric and geometric corrections (technical information).

Convert to ILWIS data format:

 

Select this check box to perform a genuine import and translate the external file to an ILWIS format:

  • the external data will be translated into ILWIS file formats: ILWIS object definition files (e.g. .MPR) as well as ILWIS data files will be created;
  • you can view the imported data, calculate with it, use it in ILWIS operations, etc.;
  • the imported data can be edited in ILWIS.

 

Clear this check box when you wish to keep the external file in its original format (Use As):

  • the external data will not be translated into ILWIS file formats: only ILWIS object definition files (e.g. .MPR) will be created; as the same data file will not appear twice on your hard disk in different formats, you will save some hard disk space in this way;
  • you can view the imported data, calculate with it, use it in ILWIS operations, etc.;
  • the original file can not be edited in ILWIS;

Necessary service objects like georeferences, etc. will always be imported into the ILWIS format.

Tips:

  • When you wish to import multiple files, it is advised to type an Import expression on the command line or to write a script with Import expressions. The Import expressions are described in Appendices : ILWIS script language (syntax), section ILWIS ASTER Import.
  • To show ASTER files with an extension in a Catalog: open the Customize Catalog sheet, on the Selection tab, and specify the external extension(s) you wish to see in the External file extensions text box. ASTER files with no extension can not be shown in the Catalog.
