Map window

Layers menu

Add Layer
 

 

Add a data layer (raster map, polygon map, segment map, point map or a map list to be displayed as a color composite) or an annotation text layer to the current map window.

Dialog box options:

Select layer:

Select the object that you wish to add as a layer to the current map window:

a raster map,

a polygon map,

a segment map,

a point map,

a map list, or

an annotation text object.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

Depending on the type of object you selected, a Display Options dialog box will be opened in which you can specify the display options for the selected layer.

Tips:

  • When a map window already displays a raster map (or a map list) and you wish to add another raster map (or map list) to the map window, the selected raster map (or map list) can only be added to the map window when it uses the same georeference as the raster map(s) that are already displayed.
  • You can also drag one or more maps, map lists or annotation text layers from a Catalog or from Windows Explorer to the desired map window with the mouse.
