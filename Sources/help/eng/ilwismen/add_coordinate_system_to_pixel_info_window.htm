Pixel information window

Add coordinate system

 File menu
 

Add an extra coordinate system to the pixel information window. If ILWIS is able to convert the coordinates as received from the coordinate system of the map window or the digitizer into the 'new' coordinate system, then the pixel information window also displays the XY-coordinates of the position of the mouse pointer in the 'new' coordinate system.

The Add Coordinate System command can be used to compare XY- coordinates of projections that consist of multiple zones such as Lambert Transverse Mercator (Gauss-Krüger) and UTM projections. When a map window displays a map which has a coordinate system with a projection of a particular zone, and you add a second coordinate system to the pixel information window with a projection of the next zone, then the pixel info window also displays the XY-coordinates as received from the map window expressed in the second coordinate system.

An example of adding multiple coordinate systems to a pixel information window is described in Pixel information window : functionality (advanced).

Dialog box:

Select coordinate system:

Select a coordinate system to be used by the pixel information window.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

Tips:

  • Instead of choosing the menu command Add Coordinate System, you can also drag a coordinate system from the Catalog to the pixel info window. This has the same effect.
  • To permanently change the coordinate system of a vector map, use one of the Transform operations.
  • To show raster maps with a different coordinate system or georeference, the Resample operation should be used.
