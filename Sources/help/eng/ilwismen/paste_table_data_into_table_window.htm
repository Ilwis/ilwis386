Table window

 Edit menu

Paste from clipboard
 

Paste the contents of the clipboard into the selected fields of the current table.

When no fields are selected in a table with system domain None, the clipboard contents will be appended to the bottom of the table.

When you copied for example (a part of) an ILWIS table, a spreadsheet or a database to the clipboard, you can paste this data into an ILWIS table.

To paste into selected fields:

Select the fields, the sequence of columns, or the sequence of records into which you wish to paste the clipboard contents, then:

 

  • choose Paste from the Edit menu,
  • click the Paste button in the toolbar,
  • press the Ctrl+V keys on the keyboard, or
  • click the selection with the right mouse button, and choose Paste from the context-sensitive menu.

Notes:

Editable columns must be available in the table into which you wish to paste.

These columns should be in the same order as the clipboard contents that you wish to paste.

  • If the table has no columns yet, you should thus manually add columns before pasting.
  • The column names are not important: the values in clipboard will be pasted regardless of the column names.
  • The domain/value range of the columns are important in the sense that pasted values which do not fit the domain or value range will appear as undefined values, i.e. question marks.
  • It is NOT possible to paste into dependent columns, into read-only columns, or into table-owned columns. These columns will be skipped during a paste.
  • When pasting into a table, all changes are directly stored on disk.

    • You CANNOT UNDO a paste action.

Tips:

  • Use Windows' Clipboard Viewer (clipbrd.exe) to inspect the current contents of the clipboard before pasting.
  • Before pasting into a number of columns, you may wish to first re-order your columns. To change the position of a column in the table, drag a column header to a new position. The mouse pointer will change into a column icon while dragging.
  • To clear a selection, press Esc on the keyboard, or click in the white space of a table.

To paste into selected fields:

  1. Select the fields into which you wish to paste:
    • To select a single field: click the single field into which you wish to paste.
    • To select multiple fields in sequence:
      • drag a rectangle over the block of fields into which you wish to paste,

        • or

      • click the first field into which you wish to paste,
      • then press the Shift key and hold it down, and
      • select the last field of a sequence of fields into which you wish to paste.
  2. Use one of the Paste methods described above.

The clipboard contents will be pasted into the selected fields; the selected fields should be editable. When pasting (part of) a copied ILWIS table, any column names and/or record names or record numbers available in the clipboard will be skipped; thus only the 'values' of copied fields will be pasted.

To paste into one or more selected columns:

  1. Select the column(s) into which you wish to paste:
    • To select a single column: click the single column into which you wish to paste.
    • To select multiple columns in sequence:
      • click the first column of a sequence of columns into which you wish to paste,
      • then, press the Shift key on the keyboard and hold it down, and
      • click the last column of a sequence of columns into which you wish to paste.
  2. Use one of the Paste methods described above.

The clipboard contents will be pasted into the fields of the selected column(s); the fields of the column(s) should be editable. When pasting (part of) a copied ILWIS table, any column names and/or record names or record numbers available in the clipboard will be skipped; thus only the 'values' of copied fields will be pasted.

To paste into one or more selected records (rows):

  1. Select the records into which you wish to paste:
    • To select a single record: click the record button of the record into which you wish to paste.
    • To select multiple records in sequence:
      • keep the left mouse button down while moving over the record buttons of the records into which you wish to paste,

        • or

      • click the first record button of a sequence of records into which you wish to paste,
      • press the Shift key on the keyboard and hold it down,
      • click the last record button of a sequence or records into which you wish to paste.
  2. Use one of the Paste methods described above.

The clipboard contents will be pasted into the fields of the selected record(s); the fields of the record(s) should be editable. When pasting (part of) a copied ILWIS table, any column names and/or record names or record numbers available in the clipboard will be skipped; thus only the 'values' of copied fields will be pasted.

Note:

No warnings are issued whether the size or shape of the selected fields fit the clipboard contents or not.

  • If the size of the table in the clipboard is larger than the size of the selection in the current table into which you wish to paste, then, only the selected part of the table is filled with the clipboard data.
  • If the size of the table in the clipboard is smaller than the size of the selection in the current table into which you wish to paste, then the clipboard contents are pasted into the upper and left part of the selection, and the remaining part of the selection keeps the data it had before pasting.

To append records to a table with domain None:

  1. Open a table using system domain None to which you wish to append the clipboard contents.
  2. Make sure there is not a single field, column, or record selected in the table:
    • press the Esc key on the keyboard, or
    • click a single field, and press the Esc key on the keyboard.
  3. Use one of the Paste methods described above.

The method of adding records to a table with domain None by a paste action, can also be used to paste existing tiepoint coordinates and row/column numbers into a georeference tiepoints. For more information, see How to paste coordinates into a georeference tiepoints.
