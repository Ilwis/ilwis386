Save a layout

 

Save the layout in the layout editor window. When you use this command when a map view is selected in the layout, then map view will be saved.

You can use the Save command on the File menu of the layout editor, but you can also click the Save button in the standard toolbar of the layout editor.

 

  • When you have saved the layout earlier, the layout will be directly saved with the same name as before.
  • When you save a new layout for the first time, a Save As dialog box will appear. In that dialog box, you can enter a new name for the new layout.

Tips:

  • To replace a map view in the current layout by another map view:
    • click the map view you wish to replace with the right mouse button (in the layout editor or in layout management), and
    • choose Replace Map View from the context-sensitive menu.

    You can then save the layout with a different name, using the Save As command.

  • To save a changed map view in a layout, i.e. when you added layers to or removed layers from the map view, or when you changed display options of layers in the map view:
    • select the changed map view in the layout, and
    • click the Save button in the standard toolbar, or
    • click the selected map view with the right mouse button and choose Save View or Save View As from the context-sensitive menu.
