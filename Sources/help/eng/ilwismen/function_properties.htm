Function Properties

The Function Properties sheet lists all kind of important information on a selected function. In fact, all relevant information that is stored in the object definition file of the function is displayed. For more information, refer to Basic concepts : properties of objects.

The Function Properties sheet appears:

  • when you click a function in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu,
  • when you select a function in a Catalog, and choose Properties from the Edit menu in the Main window;
  • when you open a function, you choose Propertiesfrom the File menu in the Function editor.

General tab:

Object type and name:

Function "function name". When you click this flat button, the function will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this function was last changed.

Object size:

Total size of the function on disk in bytes (.FUN and .FND).

Read only:

Select this check box if this function should be read only. You are then not allowed to edit or delete the function. Clear this check box if you want to be able to edit the function.

Description:

Optionally, change the description of this function. The description of a function appears in the description bar when the function is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the function in the Catalog.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.

Functions are never dependent objects.
