Symbols as arrows

Point map with ID domain and attribute table

For an ID point map with an attribute table which contains a column with directional attribute information, specify how the directional attributes should be represented as arrows. The length, thickness and colors of arrows can also be specified by choosing attribute columns.

This dialog box appears:

  • when you open a point map with an ID domain, and an attribute table is linked to the map or to the domain,
    and in the Display Options - Point Map dialog box, you selected the Arrow option and clicked the Details button.

Dialog box options:

Color:

Select this check box when the color of the arrows should be determined by a column in the attribute table of your ID point map. You can select a class column, or a column with a Picture domain or the Color domain. When you want to add a legend for the arrows later on, you should select a class column.

Clear this check box when you want to use default colors; the Windows text color will be used.

Length:

Select this check box when the length of the arrows should be determined by a column from the attribute table of your ID point map.

You can select any value column. You can for instance select a column containing wind speeds. Clear this check box when you want to use default arrow length 10.

Line width:

Select this check box when the line width of the arrows should be determined by a column from the attribute table of your ID point map.

You can select any value column. You can for instance select a column containing wind speeds.
Clear this check box when you want to use default length 0.

Direction:

Select the column in the attribute table which contains the directions of the arrows. You can select any value column; direction values of 0 or 360 will result in an arrows pointing to the North, a direction value of 90 will result in arrows pointing to the East. Values larger than 360 are also allowed (modulo 360).

Tip:

When points are displayed as arrows, and when you selected to use a class column for the color of the arrows, you can add a legend to this point map. Choose Add Annotation, Legend from the Layers menu in the map window. The Display Options - Legend (symbols as arrows) dialog box will appear.
