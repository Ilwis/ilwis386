Show Map List as slide show

Dialog box

Select a map list and show its maps as a slide show: all raster maps in the map list will be displayed one after the other in the map window at a user-specified rate. This visualization technique is designed to present multi-temporal changes in maps. All maps in the map list must use the same domain and the same georeference.

The first step of making a slide show is thus to create a map list of all the raster maps that you want to show. For more information, refer to Create a map list. All raster maps in a map list need to have the same domain and the same georeference. Further, in a slide show, all maps in the map list will be displayed using one set of display options. When the maps in the map list use a class domain, a slide show always works fine. When the maps in the map list use a value domain, it is important that the stretch range of these maps is approximately the same.

Raster maps that you can include in a map list for a slide show may be:

  • satellite images of the same area of for instance different months,
  • derived products from satellite images, such as NDVI maps, of for instance different months,
  • classified satellite images of for instance different months or years,
  • thematic raster maps (e.g. land use maps) of for instance different years,
  • output raster maps of the Apply 3D operation that depict the same area in 3D but for instance from different angles.

It is generally not very useful to display a slide show of a map list which contains the bands (different wave-length intervals) of one satellite image (taken at a certain time), as the values in each band may strongly vary. By specifying a stretch range from 0 to 255 for all these bands, one band may appear quite dark, another one quite light, etc. It is often better to display the individual bands of an image in separate map windows; each band stretched according to the values that occur in that band.

This dialog box appears:

  • when you choose Show Map List as Slide Show from the Operations, Visualization menu in the Main window, or
  • when you double-click the Slide Show item in the Operation-list.

Dialog box options:

Select a map list:

Select an existing map list which you want to display as a slide show.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

A Display Options - Raster Map dialog box is opened. This dialog box shows information on one raster map in the map list. The display options specified in this dialog box will be used to display all raster maps in the map list. In case your map list contains value maps, pay special attention to the stretch range as all maps will be displayed using the same stretch range.

After the display options are specified, the Display Options - Map List as Slide Show dialog box appears.
