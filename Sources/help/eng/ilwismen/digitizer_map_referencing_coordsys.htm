Map Reference

Coordinate System

First dialog box to establish a relation between digitizer coordinates (as received from your digitizer) and map coordinates.

The purpose of using Map Reference is to be able to use a digitizer:

  • to add points or segments to a new digital point or segment map, and/or
  • to edit an existing digital point, segment or polygon map.

This dialog box appears:

  • when you choose Map Reference from the File menu of the Main window, or
  • when you choose Digitizer, Map Reference from the File menu of the point editor, from the File menu of the segment editor, or from the File menu of the polygon editor.

Before you can use Map Reference, you should have:

  • set up your digitizer correctly:
    • the Setup Digitizer command on the File menu of the Main window should be preceded by a check mark ;
  • you fixed an analog map onto your digitizer;
  • you know the real world map coordinates of at least three points in your analog map; these points will be used as reference or control points. You can use for instance the coordinates of the corners of your analog map or the coordinates of crossings of grid lines on your analog map.

Dialog box options:

Coordinate System:

Select the coordinate system which fits the map(s) that you will digitize, or create a new coordinate system by clicking the little create button .

It is advised to use a coordinate system of type Projection: for the time being you can just specify the coordinates of the outer boundaries of your study area; specific information on a projection, ellipsoid or datum may be added to the coordinate system in a later stage or not at all.

You can use a coordinate system of type Boundaries Only when you are sure that you will not work with projections at all in your project.

Mind: It is not allowed to select or create a coordinate system of type LatLon, nor to select system coordsys LatLon.csy, nor system coordsys LatLonWGS84.csy.

Metric Coordinates:

Select this option when, in the next dialog box, you want to enter the map coordinates of your control points as XY-coordinates in meters.

Geographic Coords:

Select this option when you can only read Latitudes and Longitudes from your analog map. In the next dialog box, enter the map coordinates of your control points as LatLong-coordinates in minutes, degrees and seconds.

Mind: to use geographic coordinates, the selected coordinate system MUST have projection information.

When you click OK, the Map Reference - Digitizer Control Points dialog box will appear.
