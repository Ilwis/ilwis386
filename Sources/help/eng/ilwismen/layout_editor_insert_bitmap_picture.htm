Layout editor

 Insert menu

Insert Bitmap or Picture

 

Insert a bitmap or a picture from disk to your layout.

Dialog box options:

Select bitmap/picture:

Select a bitmap (.BMP) or a picture (.WMF, .EMF) from disk that you wish to add to the current layout.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

Tips:

  1. You can also paste bitmaps and pictures from the clipboard into the current layout. For more information, see the Windows Help, or Layout editor : Copy and Layout editor : Paste.
  2. You can add as many bitmaps and pictures to a layout as you like, for example logos.
  3. In the Layout editor, bitmaps and pictures can be sized by dragging one of the size handles of a selected object to a new position. The mouse pointer changes into a double-sided arrow when at the right position to start dragging.
    • Bitmaps can be sized isotropically by dragging a corner handle: the original height-width ratio of the bitmap is maintained.
    • Pictures can be sized isotropically by dragging a corner handle (maintain original heigh-width ratio) but they can also be sized anisotropically by dragging one of the middle handles: the height-width ratio of the picture will change.
  4. By double-clicking a bitmap or picture that has been inserted or pasted into a layout, you can switch off the Isotropic check box. After that, you are able to size the bitmap or the picture anisotropically by dragging a middle handle. Mind: once you changed the size of a bitmap or picture anisotropically, changes cannot be easily restored.
  5. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
