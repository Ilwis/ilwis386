Select object to be deleted

Select a map or table, or any other object and delete it.

This dialog box only appears when:

  • when you type Del on the command line of the Main window.

Dialog box options:

Select an object:

Select the object which you want to delete.

Select object type:

Select the type of object you want to delete: a map, a table, a map list, a map view, a domain, a representation, a georeference, a coordinate system, a histogram, etc.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

First, ILWIS checks whether the object you selected is still in use by other objects. For example, a domain or a georeference can be used by many maps.

  • If the selected object is not used by other objects, you will get a question whether you are sure to delete the selected object.
  • If the selected object is used by other objects, you will get a message saying that in principle you cannot delete the selected object; however, if you want, you can still delete the object.

To view which objects use the current object, check the properties of the object.

Tips:

  • You can also delete one or more objects, by selecting objects in the Catalog, then choose Delete from the Edit menu or from the context-sensitive menu.
  • You can also delete multiple objects by using the command line in the Main window. For more information, see Appendices : ILWIS script language (syntax).
