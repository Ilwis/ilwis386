WinTab driver:

Some of the larger manufacturers of digitizer tablets offer the standard WinTab digitizer driver for Windows on their web sites. The WinTab driver will take care of a correct communication between a digitizer and any software application (such as ILWIS).

The major advantages of using WinTab are:

  • an extremely easy digitizer setup, i.e. no knowledge required to establish communication between a digitizer tablet and your computer, and
  • your digitizer can be used by multiple Windows applications, i.e. without the need of changing any settings on the digitizer tablet anymore as might normally be required for a particular application.

Steps to take:

  • Download and install the WinTab driver.
  • In ILWIS, simply specify on this first page of the Setup Digitizer wizard that you are using the WinTab driver; all other pages of the Setup Digitizer wizard will be skipped.

Your digitizer will now be ready for use.

The Setup Digitizer command on the File menu of the Main window will be preceded by a check mark .

Preparation when not using a WinTab driver:

Before you can start to tell ILWIS how the communication between your digitizer and your computer will take place, you should already have configured your digitizer: Run mode, ASCII format, Data Rate, Resolution, Baud Rate, Parity, Data Bits, Stop Bits etc. For more information, please refer to your digitizer manual.

  • The digitizer must be configured in Run Mode, No Increment, No Prompt.
  • For the digitizer's Data Rate, we recommend 20 points per second.
  • For the Resolution, we recommend a figure of 10 lines per mm = 254 lines per inch or higher.
  • The digitizer must be configured to send information in ASCII. We recommend the format CalComp 2000-A also known as Summagraphics BitPad 2/1103 ASCII.
  • For the Baud Rate, we recommend 9600 or 19200.
  • For Parity, you can choose Even, Odd or None.
  • For Data Bits, you can choose 7 or 8.
  • For Stop Bits, you can choose 1 or 2.

Combinations of settings for Data Bits, Parity and Stop Bits that are often used are for example:

  • Data bits 7, Parity Even, Stop Bits 2;
  • Data bits 8, Parity None, Stop Bits 1.

It is however not so important what you choose, as long as you use the same settings both for the configuration of the digitizer tablet itself and for the port settings!

Tips:

  • When configuring the digitizer tablet, please make notes of the settings you use.
  • Up to this point, the digitizer settings are only valid on the digitizer end of the cable between your digitizer and your computer.
  • The crux of setting up a digitizer is that all settings used for the digitizer configuration have to exactly match the settings for selected port.
  • The port configuration is performed on the next page of the wizard.

Page options:

None:

Select this option if you don't have a digitizer. The point, segment and polygon editors will not show a digitizer menu.

WinTab:

When you installed the WinTab driver, select this option so that the WinTab driver will take care of the correct communication between your digitizer and your computer. You can directly click the Finish button.

COM1, COM2, COM3, COM4:

When you are not using an installed WinTab driver, select the serial port to which you connected the digitizer. You will have to specify settings on 3 more pages of the Setup Digitizer wizard. Click the Next button.

 

After clicking the Next button, the Setup Digitizer - Port Settings page will appear. On that page, you must make sure that you specify exactly the same settings (exactly the same figures) for Baud Rate, Parity, Data Bits, and Stop Bits for the port settings as were used for the digitizer tablet configuration.

After specifying the serial port settings, the Setup Digitizer - Automatic Detection page will appear. If you have communication between digitizer and computer, you should be able to follow the Automatic Detection instructions on that page. It means that the settings for Baud Rate, Parity, Data Bits, Stop Bits as well as the required Run mode, ASCII Data Format, and Data Rate are OK, and that buttons on the digitizer cursor are recognized.

Finally, on the last page of the wizard, the Setup Digitizer - Size, Resolution and Final Check page will appear. There you can specify the size of the tablet, the Resolution you use, and you can check the correct behaviour of the buttons.
