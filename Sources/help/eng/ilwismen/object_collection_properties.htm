Object Collection Properties

The Object Collection Properties sheet lists all kind of important information on a selected object collection. In fact, all relevant information that is stored in the object definition file of the object collection is displayed. For more information, refer to Basic concepts : Properties of objects.

When the object collection is a database collection, which is a special form of an object collection, this Property sheet lists the properties of the database collection.

This Properties sheet appears:

  • when you click an object collection or a database collection in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select an object collection or a database collection in a Catalog, and choose Properties from the Edit menu in the Main window.

General tab:

Object type and name:

Object Collection "Name", or Database Collection "Name". When you click this flat button, the object collection or database collection will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this object was last changed.

Object size:

Total size of this object collection or database collection (.IOC) on disk in bytes. This does not include any objects which are part of the object collection or database collection.

Read only:

Select this check box if this object collection should be read-only. You are then not allowed to edit or delete the object collection. Clear this check box if you want to be able to edit the object collection.

For database collections: you are never allowed to add any tables to, or to delete any tables from, the database collection. If a database collection is read-only, you cannot delete the database collection itself.

Description:

Optionally, change the description of this object collection or this database collection. The description of an object collection appears in the title bar when the object collection is displayed. Furthermore, the description of an object collection or a database collection is visible on the status bar of the Main window when moving the mouse pointer over the object collection or database collection in a Catalog.

Contains tab:

Object type and name:

Object Collection "Name", or Database Collection "Name". When you click this flat button, the object collection or database collection will be (re)opened.

Objects in container:

The ILWIS objects that are stored by this object collection are listed. Other objects that are also part of this object collection may be used in their native format (these are not listed).

For database collections: the tables that are part of the database collection are listed.

Used By tab:

Object type and name:

Object Collection "Name", or Database Collection "Name". When you click this flat button, the object collection or database collection will be (re)opened.

Used By:

In a tree view, it is listed by which other objects this object collection or this database collection is used.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
