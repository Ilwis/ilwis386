Insert/Edit Map Border

Insert or edit a map border, and/or ticks and/or coordinates for grid lines or a graticule for a selected map view in your layout.

You can insert or edit:

  • a neatline which exactly fits the selected map view;
  • grid ticks and/or coordinates along the map view in meters;
  • graticule ticks and/or coordinates along the map view in degrees, minutes, seconds;
  • coordinates for the corners of a map view;
  • an outline at a certain distance around a map view.

Use the tabs to go to a specific page.

On this Neat Line page, you can add or edit a border which exactly fits the selected map view. To add a border at a certain distance around your map view, use the Outline tab. To obtain a border along the margins of the page of a layout, choose Page Border from the Edit menu.

Neat line page options:

Color:

Select a color for the line that will be drawn along the selected map view. To define your own color, select the 'Custom' item at the bottom of the list or click the create button; the Windows Color dialog box will appear.

Neat line graduation:

Select this check box when you wish to obtain a blocked border of alternating colors around the selected map view (user-defined width in mm).

Clear this dialog box when you wish to obtain only a neat line of 0.1 mm around the selected map view.

Meters:

Choose this option button when the intervals in the blocked border should be in meters.

Degrees:

Choose this option button when the intervals in the blocked border should be in degrees, minutes, seconds.

Interval:

Type a value for the size of each interval or block in your blocked border, i.e. the distance that each block should represent.

Width (mm):

Type a value for the width of the blocked border (both on screen and on printer).

Tips:

  1. If you size a selected map border in the Layout editor, by dragging one of the size handles of a selected map border, then the size or the scale of your map view will change. This depends on whether you used the Keep Scale button.
  2. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
