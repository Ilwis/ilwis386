Map window

 File menu

Save View
 

Save the contents of the current map window as a map view.

All layers in the map window and their display settings as well as the georeference of the map window are stored. When you open a map view later on, the map window will appear exactly as it was saved. You do not need to add individual maps to the map window anymore, nor specify their display options.

For presentation purposes and printing, you need to create a layout to which you can add one or more map views and make a final composition. To the layout, you can further add annotation such as legends, map borders including ticks and coordinates for grid lines or a graticule, north arrows, scale bars and scale texts, texts (such as a title), boxes, bitmaps and pictures, and a page border.

Note:

When you are in the Layout editor and save a map view, please go to topic Layout editor : Save Map View.

When you save the contents of a map window for the first time, the Save View As dialog box appears. In that dialog box, you can enter a name and a title for the new map view.

Also, when you choose the Create Layout command from the File menu in a map window, and you have not saved the multiple layers displayed in the map window as a map view before, you will be asked to save the layers as a view first in the Save View As dialog box.

When you use the Save View command and when the map window already displays an existing map view, the contents of the current window are directly saved. The same map view name and title are used as previously specified in the Save View As dialog box, i.e. the previously stored map view is overwritten.
