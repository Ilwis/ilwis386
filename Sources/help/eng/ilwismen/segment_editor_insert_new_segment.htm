Segment editor

Insert new segment

 

Insert a new segment and select a class name, ID, or value for the new segment.

This dialog box appears:

  • after you finished inserting a new segment with the digitizer, or
  • after you finished inserting a new segment with the mouse in Insert mode.

 

For more information on:

  • using the digitizer, refer to Segment editor : functionality, section Working with the digitizer.
  • using the mouse, refer to the same topic, section Working with the mouse.

Dialog box options:

Class name/ID/Value:

To assign a class name, ID, or value to the new segment, and depending on the domain of the segment map: select a class name from the drop-down list, type an ID, or type a value.

Tips:

  • Segment codes are mainly important when digitizing contour lines, etc., i.e. segments that contain information on line features.
  • When digitizing segments that need to be polygonized later on, codes of segments are not very important. For more information, see How to create a polygon map.
  • However, codes are important when you wish to use the same segments in other segment maps, for instance when a segment is a boundary in multiple maps. You should then copy segments into another segment map using a mask.

Class names, IDs, values, etc.

  • For a map using a class domain: for each newly inserted segment, by default the class name will be assigned as specified for the Insert code.

    •  

  • For a map using an ID domain: for each newly inserted segment, an ID will be asked in a dialog box.

    • When an ID is assigned that already exists in the map, a question is asked whether you wish to use the same ID again or not.

    A new ID will be automatically added to the ID domain.

     

  • For a map using a value domain: for each newly inserted segment, a value will be asked in a dialog box.

    • The value should fall inside the minimum and maximum value of the value domain used by the map.

     

  • For a map using a Bool domain: for each newly inserted segment, a value True, False, or ? (= undefined) will be asked in a dialog box.

    •  

  • For a map using a UniqueID domain: each newly inserted segment will automatically obtain an ID as seg 1, seg 2, seg 3, etc. Optionally, you can type a Name in the Attribute window. You can also enter other attribute to the attribute table in this window.

    • The Names will be available as a String column in the attribute table of the map. In the attribute table, the Name column can be easily converted to a regular Class or ID column through the Column Properties dialog box of the column by clicking the Create new domain from Strings button.

 

When you choose the '?' question mark, the new segment will not get a class name or value; it will have the undefined value.

Tip:

To Edit the names, values etc. of existing segments, see Segment editor : Edit.
