Command line

History entries:

Specify the number of commands, formulae, expressions that should be kept in the history of the command line in the Main window and table windows.

 

General options

Give a warning when an object is opened which is not up-to-date:

 

Select this check box when you wish to receive a warning when you open an object that is not up-to-date anymore.

Clear this check box when an object should be opened without checking.

Show a Raster Map Definition form when calculating:

 

Select this check box when you wish to get the Raster Map Definition dialog box after you typed a MapCalc formula on the command line of the Main window; in the Raster Map Definition dialog box, you can specify for instance the desired precision of the output map.

Clear this check box when you do not wish to obtain the Raster Map Definition dialog box, i.e. use defaults during calculations.

 

Default double-click actions

Filter:

Select the action to be performed when double-clicking a filter in a Catalog:

  • Open: to open the filter in the filter viewer/editor.
  • Filter: to start the Filter operation by opening the Filter : dialog box.

Map list:

Select the action to be performed when double-clicking a map list in a Catalog:

  • Open: to open the map list in the map list editor.
  • ColorComp: to open the Display Options - Map List as Color Composite dialog box to display three images in the map list as an interactive color composite.
  • SlideShow: to open a Display Options - Raster Map dialog box, and then the Display Options - Map List as Slide Show dialog box to show all maps in the map list as a slide show.

Stereo pair:

Select the action to be performed when double-clicking a stereo pair in a Catalog:

  • Anaglyph: to open the Display Options - Stereo Pair as Anaglyph dialog box, to show the stereo pair as an anaglyph in a map window.
  • Stereoscope: to directly open the stereo pair in the stereoscope window.

Editors
Point editor

Specify some settings for the point editor, see also Point editor : Customize.

 

Show text:

Select this check box when you wish to show the class names, IDs, or values of points next to the points.

Text color:

Select or create a color for the text that will appear next to the points.

Background:

Select or create a color for the background of the text that will appear next to the points.

Symbol type:

Select a symbol type with which the points should be displayed.

Segment editor

Specify some settings for the segment editor, see also Segment editor : Customize. Some settings will be used by the point editor and the polygon editor.

 

Color digitizer cursor:

 

Select or create a color for the digitizer cursor in the segment editor.

Note: The selected color will also be used as the Digitizer Cursor color in the point editor and the polygon editor.

Auto snap:

Select this check box to automatically snap the begin node and end node of a newly inserted segment (using the mouse) to existing nodes or to existing segments, provided that the distance to the existing node or segment is less than the snap tolerance.

Clear this check box if newly inserted segments should not be automatically snapped to existing nodes or existing segments. If you clear this check box, use the Shift key to snap to an existing node, and use the Ctrl key to split an existing segment and snap to the newly created node. For more information, see Appendices : mouse functions.

Show nodes:

Select this check box to show the nodes of segments when in Select mode and Insert mode. When you are in Move Points mode, segment nodes and intermediate points within segments are always shown.

Clear this check box when segment nodes should not be visible in Select mode and Insert mode.

Snap tolerance (pixels):

 

Specify a value for the distance interval in which snapping between newly inserted segments and existing ones is allowed, when working with the mouse.

Normal color:

Select or create a color for segments, nodes and intermediate points within segments, for any segment map that is opened in the segment editor.

Retouch color:

Select or create a color for segments, nodes and intermediate points within segments, when moving a node or an intermediate point to another position in Move Points mode.

Deleted color:

Select or create a color to show deleted segments when using the Undelete Segments option.

Find undef color:

 

Select or create a color to show undefined segments (i.e. segments without value, class name or ID) when using the Find Undefineds option.

Note: The selected color will also be used as the Find Undefineds color in the point editor and the polygon editor.

Directories

Tools directory:

Specify the path(s) for the folder(s) where commonly used external tools are located. If desired, multiple Tools folders can be specified: separate folders by semicolons ; . When one or more Tools folders are specified, a Tools entry will appear on the Operations menu, in the Operation-tree, and in the Operation-list. This will list all tools that are present in the specified folder(s). See also: How to customize the Operation-tree, the Operation-list and the Operations menu.

Script directory:

Specify the path(s) for the folder(s) where commonly used ILWIS scripts are located. If desired, multiple Script folders can be specified: separate folders by semicolons ; . When one or more Script folders are specified, a Script entry will appear on the Operations menu, in the Operation-tree, and in the Operation-list. This will list all scripts that are present in the specified folder(s). See also: How to customize the Operation-tree, the Operation-list and the Operations menu.

Directory for LOG file:

Specify the path for the ILWIS.LOG file. The LOG file stores all commands that have been used on the command line of the Main window.

Default startup dir:

First close all Catalogs in the Main window. Then, specify the path for the directory in which ILWIS should start next time. Close ILWIS.

Mind: If you close ILWIS while any Catalogs are still open, these Catalogs will be reopened when you restart ILWIS.

For more information, see also How to change the working or startup directory.
