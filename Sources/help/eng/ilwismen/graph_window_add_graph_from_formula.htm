Add Graph from Formula
 

Create a graph by typing a formula or an expression.

 

Note: A graph from formula is only properly displayed if the graph window already shows a graph from columns where both columns are Value columns.

This dialog box appears:

  • when you choose the Add Graph from Formula command from the Edit menu in a graph window;
  • when you click the arrow next to the Add Graph button in the toolbar of the graph window, and choose the from Formula command;
  • when you use the right mouse button in the white space of the Graph Management pane, and choose the Add Graph from Formula command from the context-sensitive menu.

Dialog box options:

Expression:

Type an expression which defines a function y=f(x).

  • The outcome of the expression will be used as the Y values.
  • In the expression, you can only use X as a parameter, representing the values of the current X-axis.
  • Furthermore, you can use all existing TabCalc operators and functions which work on values.

Note: It is not allowed to use column names in the expression.

A new line graph will be immediately displayed.

Examples:

y =

SQ(x)

draws a parabola.

y =

SIN(x)

draws a sine function.

y =

EXP(x)

draws an exponential function.

Tips:

  • To modify the appearance or the name of the Graph from Formula, double-click the graph layer in the Graph Management pane: the Graph Options - Graph from Formula dialog box will appear.
  • To adapt the title, the X-axis, or the Y-axis (left or right) of the graph, double-click the appriopriate item in the Graph Management pane or in the graph window itself: the Title, X-axis, Y-axis (left), or the Y-axis (right) dialog box will appear.

    • The grid lines of the graph can be modified in the X-axis and Y-axis dialog boxes.

  • To create a new graph in a new graph window, choose Create Graph from the File menu in the graph window: the Create Graph dialog box will appear.
  • To save the contents of a graph window as a graph, choose Save or Save As from the File menu in the graph window.
