Polygon editor

 File menu

Undo all changes   
 

 

Restore all changes in this polygon map since the last Save.

The Undo All Changes command can be used to restore all recent changes.

When using this Undo All Changes command:

  • all changes you made to the polygon map since you last used Save will be restored;
  • if you have not used Save so far, all changes since you started editing the map will be undone.
