Symbol

Point maps with a class or ID domain

Specify symbol type, size, color, line width and line color in which all points should be displayed. You can use simple symbols or use any installed True Type symbol font.

This dialog box appears when you open a class or ID point map, or when you display a point map by a class or ID attribute column, and you choose to show the points by the same kind of symbols. In the Display Options dialog box of a point map, you selected the Single Symbols option and/or clicked the Symbol button.

Dialog box options:

Symbol type:

Select Simple symbols or any installed True Type symbol font; open the list box by clicking it. Windows symbol fonts are for example Symbol, Wingdings, etc; you can also select an installed Corel Draw symbol font, see also the Tip below.

In case of symbol type Simple:

Symbol ...:

Select a symbol type from the list box: circle, square, diamond, delta up, delta down, plus, minus, or cross.

Size:

Specify a value for the symbol size. Symbol size is expressed in points (1/72 inch), i.e. as choosing a font size in word processing packages.

Fill Color:

For symbols circle, square, diamond, delta up, delta down: select a color to fill the symbols with; open the list box by clicking it. To create your own color, select Custom at the bottom of the drop-down list. To obtain symbols which are not filled, select Transparent from the drop-down list.

Line width:

Specify a value for the line width with which the selected symbols should be drawn. Line width values are expressed in screen pixels (for on the screen) and in dots (for on the printer).

Color:

Select a color to draw the (outer lines of) symbols; open the list box by clicking it. To create your own color, select Custom at the bottom of the drop-down list, or click the create button.

In case of a Windows symbol font:

Symbol box:

Select a symbol from the symbol box by clicking it.

Size:

Specify a value for the symbol size. Symbol size is expressed in points, i.e. as choosing a font size in word processing packages.

Color:

Select a color with which symbols should be drawn; open the list box by clicking it. To create your own color, select Custom at the bottom of the drop-down list, or click the create button.

Rotation:

Type a value for the rotation of the symbols. Rotation is expressed in degrees (anti-clock wise).

Mind:

These settings are not automatically stored. To store the display of the point map, you can save the contents of the map window as a map view.

Tip:

To use for instance a Corel Draw symbol font, first make sure that the font is registered by Windows:

  • In the Windows Settings, choose Control Panel, and choose Fonts. A list of installed fonts will appear.
  • When the desired symbol fonts are not yet in this list, choose File, Install New Font. The Add Font dialog box will appear.
  • Specify the Corel Draw/Symbols directory, select the fonts you want to use and click OK.

Now that the symbol fonts are registered by Windows, you can use them in ILWIS as well.
