Save View
 

Save the contents of the currently selected map view in this layout. The map view as inserted into this layout and as saved before in a map window will be overwritten.

All layers in the map view and their display settings as well as the scale and size of the map view are stored.

You can use this command:

  • when you added or removed layers to a map view in the layout editor, or
  • when you changed display options of layers in a map view in a layout.

Tip:

To save a changed map view with a different name:

  • select the map view you wish to save,
  • click it with the right mouse button, and
  • choose Save View As from the context-sensitive menu.
