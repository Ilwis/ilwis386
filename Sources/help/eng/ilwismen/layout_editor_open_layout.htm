Layout editor

 File menu

Open a layout

 

Open another existing layout in the current layout editor window.

If the current layout has been changed, first a question will be asked whether the current layout should be saved or not.

This dialog box appears:

  • when you choose Open from the File menu in the Layout editor;
  • when you click the Open button in the standard toolbar of the layout editor.

Dialog box options:

Select a layout:

Select a layout that you wish to open in the layout editor.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The layout editor will show the selected layout. You can edit the layout according to your wishes and print it.

Tips:

  • To open a layout in a new window, you can double-click a layout in a Catalog, or click the right mouse button on a layout in a Catalog and choose Open from the context-sensitive menu.
  • To replace one map view in the current layout by another: click the map view you wish to replace with the right mouse button (in the layout editor or in layout management) and choose Replace Map View from the context-sensitive menu.
