Note: This page is only shown when importing a dBase DBF file.

As DBF files can be imported both by ILWIS import and by database access (ODBC), you have to choose the method you wish to use.

 

This page is skipped when importing:

  • an ASCII table in ILWIS 1.4 format,
  • an ASCII table in comma delimited format,
  • an ASCII table in space delimited format, or
  • an ASCII table in fixed format.

In these cases, you will automatically be directed to the ILWIS import method. You can click ILWIS import, Next at the top of this Help page.

This page is also skipped when importing:

  • a database format such as Access .MDB, for which an ODBC driver has been installed on your system.

In this case, you will automatically be directed to the Database method. You can click Database, Next at the top of this Help page.

Page options:

Use ILWIS import:

To import the dBase DBF file with ILWIS DBF import.

Use Open Database Connectivity (ODBC):

 

To import the dBase DBF file using an ODBC driver or OLE DB.

If you choose ILWIS import:

After clicking the Next button, the Import Table - Select Table Format page will appear. On that page, you have to specify the format of the table you wish to import.

When you specified that the external file is a file in ASCII format (ILWIS 1.4, Comma delimited, Space delimited, or Fixed format), the Import Table - View File page will appear.

This page will be skipped if you specified that the external file is a file in dBase format.

The Import Table - Edit Column Details can be used to specify the name and type of domains of each column and optionally, the domain of the table.

Finally, on the last page of the wizard, the Import Table - Specify ILWIS Table Name page will appear. There you can specify the name of the ILWIS table that will be the result of import.

If you choose ODBC:

After clicking the Next button, the Import Database - Select Table and/or Table Domain page will appear. On that page, you can choose a domain for the output table.

Subsequently, you will go to the Import Database - SQL query page. On that page, you can specify the SQL query to retrieve specific record from the table.

Finally, on the last page of the wizard, the Import Table - Specify ILWIS Table Name page will appear. There, you can specify the output name for the table, and you can choose to obtain a Use As table or to convert the table to the ILWIS table format.
