Create an epipolar stereo pair

Create an epipolar stereo pair from two aerial photographs with overlap. You can view the resulting stereo pair with a stereoscope in a stereoscope window, or as an anaglyph in a map window.

This dialog box appears when creating a new stereo pair:

  • from the Operations menu of the Main window, choose Image Processing, Epipolar Stereo Pair, or
  • in the Operation-list, double-click the Epipolar Stereo Pair item, or
  • in the Operation-tree, expand the Image Processing item and double-click the Epipolar Stereo Pair item, or
  • from the File menu in an existing Epipolar Stereo Pair - Creation window, choose the New Stereo Pair command, or
  • click the New Stereo Pair button in the toolbar of the Epipolar Stereo Pair - Creation window.

Dialog box options:

Left image:

Select the raster map (e.g. a scanned aerial photograph) that is positioned on the left-hand side.

Right image:

Select the raster map (e.g. a scanned aerial photograph) that is positioned on the right-hand side.

Output stereo pair:

Type a new name for the stereo pair. The (resampled) output raster maps of the stereo pair will also obtain this name followed by _Left or _Right.

Description:

Type a description for the output stereo pair. The description of a stereo pair is visible on the status bar of the Main window when moving the mouse pointer over the map in the Catalog.

Tips:

  • You should have two scanned images (for instance two aerial photographs with fiducial marks) that have overlap.
  • When you wish to view the stereo pair later on with a stereoscope, you can use input raster maps with the Color domain, the Image domain, or another value domain.
  • When you wish to view the stereo pair as an anaglyph later on, you can only use raster maps with the Image domain, or with a value domain.

The two input raster maps will be displayed in the Epipolar Stereo Pair - Creation window. You can mark the positions of the fiducial marks, the principal points, the transferred principal points, and optional scaling points.
