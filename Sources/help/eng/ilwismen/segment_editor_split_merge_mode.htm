Segment editor

 Edit menu

Split/Merge mode   
 

 

Choose Split/Merge mode to use the mouse to:

  • split a segment, i.e. insert a node within a segment, or
  • merge segments, i.e. delete a node between two segments with the same class name, ID or value.

When Split/Merge mode is active:

  • the mouse pointer appears as a pair of scissors,
  • an option bullet appears in front of this menu command, and
  • the Split/Merge button in the toolbar appears down.

To split a segment:

  • position the mouse pointer at the location in the segment where you want to split;
  • click the left mouse button;
  • a message box asks for confirmation to split the segment.

To merge two segments:

To remove a superfluous node which connects only two segments which have the same code:

  • position the mouse pointer at the node;
  • click the left mouse button.
  • a message box asks for confirmation to merge the segments.
