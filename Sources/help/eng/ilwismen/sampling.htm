Sample

Dialog box

Select an existing sample set or create a new sample set. Then, sample training pixels prior to an image classification. Sampling is the process of assigning class names to groups of pixels with similar spectral values and that are supposed to represent a known feature on the ground.

Dialog box options:

Sample set:

Select an existing sample set to continue sampling (open the list box by clicking it), or create a new sample set by clicking the little create button .

When you create a new sample set, the Create Sample Set dialog box appears. In the Create Sample Set dialog box, you can specify which background map you want to use, the map list that should be used and the domain of the sample set. After that, you return to this Sampling dialog box.

When everything is OK, the Sample set editor is started.
