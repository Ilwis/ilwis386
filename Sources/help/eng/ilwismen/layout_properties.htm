Layout Properties

The Layout Properties sheet lists all kind of important information on a selected layout. In fact, all relevant information that is stored in the object definition file of the layout is displayed. For more information, refer to Basic concepts : Properties of objects.

The Layout Properties sheet appears:

  • when you click a layout in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a layout in a Catalog, and choose Properties from the Edit menu in the Main window.

General tab:

Object type and name:

Layout "Name". When you click this flat button, the layout will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this layout was last changed.

Object size:

Total size of this layout (.ILO) on disk in bytes. This does not include any map views, or maps that may have been added to the layout. However, inserted pictures or bitmaps are included.

Read only:

Select this check box if this layout should be read only. You are then not allowed to edit or delete the layout. Clear this check box if you want to be able to edit the layout.

Description:

Optionally, change the description of this layout. The description of a layout appears in the title bar when the layout is displayed. Furthermore, the description is visible on the status bar of the Main window when moving the mouse pointer over the layout in a Catalog.

Contains tab:

Object type and name:

Layout "Name". When you click this flat button, the layout will be (re)opened.

Objects in container:

The map views that have been included in this layout are listed.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
