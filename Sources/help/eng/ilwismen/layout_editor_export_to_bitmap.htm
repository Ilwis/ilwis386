Layout editor

 File menu

Export to Bitmap

 

Save the complete layout that is currently displayed in the layout editor as a bitmap.

Dialog box options:

File name:

Type a name for the output bitmap.

Resolution (dpi):

Specify the resolution (in dots per inch) for the bitmap.

Size:

For your information, the size of the output bitmap is listed in pixels.

File size:

For your information, the file size of the output bitmap is listed in kB or MB.

Mind: As the bitmap will first be created in memory, you should not specify a resolution for the bitmap that will result in a file size which exceeds the amount of free RAM in your computer.
