Specify the columns that should be shown when displaying a Catalog, an opened object collection, or an opened map list in Details View.

The Customize Catalog sheet appears:

  • when in the Main window, you choose the Customize Catalog command from the View menu,
  • when you click the Customize Catalog button in the standard toolbar of the Main window,
  • when you click the right mouse button in the white space of a Catalog and choose the Customize Catalog command from the context-sensitive menu.

 

Subsequently, click the Details View tab.

Page options:

Visible columns:

Select the columns that you wish to see when in Details View. For more information on how to select a number of columns, see below.

Grid visible:

Select this check box if grid lines should be drawn around the 'cells' in the Details View table. Clear this check box if you do not want to see grid lines.

Use current settings for new Catalogs:

 

Click this button to apply the settings of the currently active Catalog to all new Catalogs that will be opened. Settings include visible objects, sorting method, and queries.

Use default settings for new Catalogs:

 

Click this button to ensure that the settings of the active Catalog are NOT applied to any new Catalog that will be opened. For any new Catalog that will be opened, the default Customize Catalog settings will be used.

Available columns:

Name lists the object icons and the object names.
D lists whether objects are Dependent (D) or not (blank).
C lists whether dependent objects are Calculated (C) or not (N); blank for non-dependent objects.
U lists whether dependent objects are Up-to-Date (U) or not (N); blank for non-dependent objects.
R lists whether objects are Read-only (R) or not (blank).
Modified lists the date and time that objects were modified last.
Domain lists the name of the domain that is used by objects; blank if object does not use a domain.
Domain type lists the type of domain used by objects.
CoordSystem lists the name of the coordinate system that is used by objects; blank if object does not use a coordinate system.
Csy type lists the type of coordinate system that is used by objects; blank if object does not use a coordinate system.
Georef lists the name of the georeference that is used by the object; blank if object does not use a georeference.
Georef type lists the type of georeference that is used by an object; blank if object does not use a georeference.
Size lists the 'Contents Size' of objects, e.g. nr of lines and cols for raster maps, nr of polygons for polygon maps, nr of records and columns in a table, nr of items in a class or ID domain, etc.
Attribute lists the name of the attribute table that is linked to a map or to a domain; blank if object does not have an attribute table.
Description lists the Description of objects; when during an operation no description is filled out for the dependent output object, the dependent output object will use its definition or expression as description; blank if a non-dependent object has no description.
Expression lists for dependent objects the expression or definition by which the object was created; blank for non-dependent objects.
Type lists the type of the objects as text.

Tips:

  • To view only certain objects, use the Show command on the View menu, the Object Selection toolbar, or the Customize Catalog, Selection tab.
  • To sort objects, use the Sort By command on the View menu, or click the column headers when in Details View.
  • To search for certain objects, use the Catalog Queries command on the View menu, or use the Customize Catalog, Query tab.

Selecting multiple items in a list

To select two or more items in sequence:

  • click the first object type you want to select,
  • keep the left mouse button down, and
  • move the mouse pointer to the last object type you want to select,
  • then release the left mouse button;

or

  • click the first object type you want to select,
  • press and hold down the Shift key while you click the last object type of the group.

To select two or more items out of sequence:

  • press and hold down the Ctrl key while you click each object type.

To cancel a selected object type:

  • press and hold down the Ctrl key while you click a selected object type.
