Stereo Pair as Anaglyph

 

Specify display options for a stereo pair that you wish to show as an anaglyph in a map window.

Before the stereo pair is displayed as an anaglyph in a map window, you have to answer some questions on how the stereo pair should be displayed. Also, once maps are displayed in a map window, you may wish to change the display options of a layer:

  • from the Layers menu in a map window, choose the Display Options command, then select a layer from the cascading menu, or
  • click the right mouse button in the map window, choose the Display Options command from the context-sensitive menu, and select a layer from the cascading menu, or
  • double-click a layer in the Layer Management pane of a map window.

Dialog box options:

Red - Green:

Select the Red-Green option, if you wish to view the anaglyph with Red-Green glasses.

Red - Blue:

Select the Red-Blue option, if you wish to view the anaglyph with Red-Blue glasses.

Left Range:

Optionally, stretch the values of the left photograph/image to enhance the contrast in the image.

By default, the 1% interval is used for stretching.

Right Range:

Optionally, stretch the values of the right photograph/image to enhance the contrast in the image.

By default, the 1% interval is used for stretching.

Create Pyramid Layers:

 

In case pyramid layers are not yet available for the (output) raster maps of the stereo pair:

  • Select this check box to create pyramid layers for the (output) raster maps of the stereo pair. Pyramid layers speed up the display of large raster maps in a map window.
  • Clear this check box if you do not wish to create pyramid layers for the raster maps.

For more information on pyramid layers, see ILWIS objects : Raster maps - Pyramid layers.

The first text line in this dialog box states the object type (stereo pair) and the object name. Furthermore, the dialog box shows the description of the stereo pair.

Tips:

  • As the calculated output images of the stereo pair may have a very large file size, it is advised to create Pyramid Layers for both output raster maps. This will speed up display in the map window.
  • In order to show the stereo pair as an anaglyph, the output images of the stereo pair must use the Image domain or another value domain. To convert a map with the Color domain to the Image domain, you can use the Color Separation operation; choose option Intensity. See also How to change the domain of a map.
