Layout editor

 File menu

Save a layout as ...

 

Save the current layout with a new name.

This dialog box appears:

  • when you choose Save As from the File menu in the Layout editor;
  • when you have not saved the layout before, and you choose Save from the File menu or click the Save button in the standard toolbar.

If any map views in the layout have been changed and when you wish to save the map views as well: save the map views yourself before using this option. For more information, see the tips below.

Dialog box options:

Save in:

If necessary, select another directory or folder where the layout should be stored.

File name:

Type a new name for your layout. It is not necessary to specify the extension.

Save as type:

You cannot select anything else than type Layout.

Tips:

  • To replace one map view in the current layout by another:
    • click the map view you wish to replace with the right mouse button (in the layout editor or in layout management), and
    • choose Replace Map View from the context-sensitive menu.

    You can then save the layout with a different name, using this Save As command.

  • To save a changed map view in a layout, i.e. when you added layers to or removed layers from the map view, or when you changed display options of layers in the map view:
    • select the changed map view in the layout, and
    • click the Save button in the standard toolbar, or
    • click the selected map view with the right mouse button and choose Save View or Save View As from the context-sensitive menu.
