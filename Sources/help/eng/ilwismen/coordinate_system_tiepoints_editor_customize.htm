Coordinate system Tiepoints editor

Customize

 Edit menu
 

Customize the way in which tiepoints are displayed on the background map in the Tiepoint editor.

Dialog box options:

Active color:

Select a color for the active tiepoints and their numbers; open the list box by clicking it. To create your own custom color, click the create button or select 'Custom' at the bottom of the drop-down list.

Passive color:

Select a color for the passive tiepoints and their numbers; open the list box by clicking it. To create your own custom color, click the create button or select 'Custom' at the bottom of the drop-down list.

Symbol size:

Type a value (integer > 0) for the size of the symbols that represent your tiepoints. Tiepoints are always represented by a plus sign.
