Pixel editor

 Edit menu

Copy to clipboard
 

Copy selected pixels with their coordinates and their class names, identifiers or values to the clipboard. In the clipboard, the information is available both in text format and in bitmap format.

If the raster map in the pixel editor does not have coordinates, i.e. when the raster map uses georeference None, then the Row and Column numbers and class names, IDs or values of the selection are copied to the clipboard.

Subsequently, the copied pixels can be pasted as points into a point editor, as one segment into a segment editor, and/or into any other application which uses the bitmap format or the text format (e.g. a word processor or a data base).

To copy:

Select one or more pixels, and:

  • choose Copy from the Edit menu, or
  • press Ctrl+C on the keyboard.

For more information on selecting pixels, see Pixel editor : Normal/Select mode.

Tips:

  • To copy the active window including the window borders, title bar, menu bar and status bar as a bitmap to clipboard, press Alt+PrintScrn on the keyboard.
  • From within a map window, you can copy (the displayed part of) a map as a picture to the clipboard; choose Copy from the Edit menu in a map window. Coordinate information will not be available however.
  • You can use Windows' Clipboard/Clipbook Viewer to inspect the current contents of the clipboard. For more information, see the Windows Help.
