Graph Properties

The Graph Properties sheet lists all kind of important information on a selected graph. In fact, all relevant information that is stored in the object definition file of the graph is displayed. For more information, refer Basic concepts : Properties of objects.

The Graph Properties sheet appears:

  • when you click a graph in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a graph in a Catalog, and choose Properties from the Edit menu in the Main window.

General tab:

Object type and name:

Graph "Name". When you click this flat button, the graph will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this graph was last changed.

Object size:

Total size of this graph (.MPV) on disk in bytes. This does not include any maps or annotation text that may have been added to the graph.

Read only:

Select this check box if this graph should be read only. You are then not allowed to edit or delete the graph. Clear this check box if you want to be able to edit the graph.

Description:

Optionally, change the description of this graph. The description of a graph appears in the title bar when the graph is displayed. Furthermore, the description is visible on the status bar of the Main window when moving the mouse pointer over the graph in a Catalog.

Contains tab:

Object type and name:

Graph "Name". When you click this flat button, the graph will be (re)opened.

Objects in container:

The table(s) that contain the columns from which the graph is constructed are listed.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
