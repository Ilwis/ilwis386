Display Options - Graticule
 

Specify display options for a graticule to be displayed in a map window.

A graticule can only be created if the map(s) displayed in the map window use a coordinate system that contains projection information. It is not yet possible to have Latitude and Longitude coordinates appear as text next to the graticule.

Dialog box options:

Graticule distance:

Type values in degrees, minutes and seconds for the spacing of the graticule.

Line type:

Select a line type in which the graticule should be drawn: Single, Dot, Dash, Dash dot, or Dash dot dot. The graticule will always be drawn as fine as possible, i.e. using one pixel on the screen and using one dot on the printer.

Color:

Select a color for the graticule. To define your own color, select the 'Custom' item at the bottom of the list or click the create button; the Custom Colors dialog box will appear.

Scale limits:

Select this check box when this graticule should be displayed or hidden depending on the scale specified in the scale control of the map window; specify the scale range within which the graticule should be displayed. The graticule will be automatically hidden if the scale of the map window is outside this range.

With this option you can achieve for instance, that a certain graticule is only shown when using a large scale in a map window, and/or that another graticule (other spacing) is only shown when using a small scales in a map window, etc.

Clear this check box when you wish to display the graticule regardless of the scale of the map window.

Tips:

  • To obtain a graticule of:
    1 degree: use distance values 1°, 0', 0.00";
    1 minute: use distance values 0°, 1', 0.00";
    30 seconds: use distance values 0°, 0', 30.00".
  • To store the contents of a map window, you need to save the map window as a map view. Open the File menu of the map window and choose the Save View or the Save View As command.
