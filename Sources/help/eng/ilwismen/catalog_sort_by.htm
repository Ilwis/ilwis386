Catalog - Sort By ...
 

View the objects in a Catalog (directory) or in an opened object collection ordered by Name, Type, Modification date, or any of the other possibilities. The Sort By command works both in List View and in Details View.

In short, you can choose to sort on:

  • Name, Type, Modification Date of the objects,
  • whether objects are Dependent, Calculated, Up-to-Date or Read-only,
  • the name of the Domain used by the objects, and the Domain Type,
  • the name of the Coordinate System used by the objects, and the Coordinate System Type,
  • the name of the Georeference used by the objects, and the Georeference Type,
  • the 'Contents Size' of the object, as the number of lines and columns for raster maps, number of items in a class or ID domain, number of polygons in a polygon map, etc.
  • the name of a possible Attribute Table linked to a map or to a class or ID domain,
  • the Description of the objects,
  • the Expression by which a dependent object is created.

The available sorting methods are described in detail in topic Catalog - Details View.

The selected sorting method will be preceded by an option bullet .

Tips:

  • To view only certain types of objects in a Catalog, see Catalog - Show ..., Main window : Object selection toolbar, or Customize Catalog, Object Selection tab.
  • To search for certain objects, see Catalog - Predefined queries, or Customize Catalog, Query tab.

Mind:

  • Settings on visible object types, sorting/ordering method and queries on objects are either applied to all new Catalogs that will be opened, or to the currently active Catalog only. This depends on whether you click the Use current settings for new Catalogs button or the Use default settings for new Catalogs button on the Customize Catalog, Details View page.
  • Only those objects will be shown which match the selected object types and a used query.
