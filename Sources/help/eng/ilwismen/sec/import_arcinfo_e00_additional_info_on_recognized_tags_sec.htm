Import Arc/Info .E00 - Additional info on recognized tags

The following tags are recognized and used by ILWIS:

  

ARC:

plain segments or topologies (for polygons).

GRD:

ASCII grid (raster map), both integer and floating point types.

LAB:

points or label points (for polygons).

PAL:

polygon information; this is detected only, and will cause the import to start a polygonize outine when the import itself is done.

IFO:

subsections. When the IFO section contains one table only, this table will be imported as a table with domain None.

BND:

the coverage boundary.

The following tags are recognized, but skipped:

  

CNT:

centroids.

LOG:

logfile containing the history of the coverage data.

MSK:

unknown tag.

PAR:

unknown tag.

PFF:

unknown tag.

PRJ:

projection/coordinate system information.

SIN:

spatial index.

TOL:

tolerances.

TX6, TX7, TXT:

text.
