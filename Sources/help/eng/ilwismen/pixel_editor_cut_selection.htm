Pixel editor

 Edit menu

Cut
 

Copy selected pixels with their coordinates and class names, identifiers or values to the clipboard, and then clear the selected defined pixels so that the selected pixels become undefined.

Subsequently, the copied pixels can be pasted as points into a point editor, as one segment into a segment editor, and/or into any other application which uses the bitmap format or the text format (e.g. a word processor or a data base).

To cut:

Select one or more pixels, and:

  • choose Cut from the Edit menu, or
  • press Ctrl+X on the keyboard.

For more information on selecting pixels, see Normal/Select Mode.

A message box will appear asking you to confirm the clearing of the selected pixels. Answer with:

  • yes: to clear the selection, or
  • no: to not clear the selected pixels; the selected pixels will be copied.

Tip:

  • There is no possibility to recover any cleared pixels.
  • You can use Windows' Clipboard/Clipbook Viewer to inspect the copied pixels on the clipboard in text format and in bitmap format. For more information on the Clipboard Viewer, see the Windows Help.
  • An example of pixels copied to the clipboard is presented in Pixel editor : Copy.
