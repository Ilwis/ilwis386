Find undefineds
 

To find segments which do not have a value, class name or ID yet.

The Find Undefineds command is active when a check mark appears in front of this menu command.

When using Find Undefineds:

  • undefined segments will be displayed in the color as specified in the Segment editor - Customize dialog box;
  • very small undefined segments will also obtain a colored square box around them so that you can localize them easily;
  • with the mouse, select one or more undefined segments by clicking;
  • then, choose Edit from the context-sensitive menu or from the Edit menu to assign a proper value, class name or ID to the selection.

When finished, you can switch off the Find Undefineds command again.

 

Tip: To select multiple undefined segments and give them the same value or class name: press and hold down the Ctrl key while clicking undefined segments. Then choose the Edit command as usual.
