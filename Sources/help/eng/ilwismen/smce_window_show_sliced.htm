Show Sliced   
 

 

Show the classified maps that are the result of slicing the SMCE output map(s).

 

  • When you select the main goal and press the Show Sliced button, the sliced SMCE output maps will be shown.
  • When you select a sub goal with named output maps and press the Show Sliced button, the sliced maps of the sub goal will be shown.
  • When you select a single output map and press the Show Sliced button, the sliced map of the selected map will be shown.

After selecting the main goal, a sub goal or a single map in the criteria tree viewer, you can use:

  • the Show Sliced button in the toolbar,
  • the Show Sliced command from the Generate menu, or
  • the Show Sliced command from the context-sensitive menu.

Notes:

  • Show Sliced is not available when the SMCE output maps have not been sliced.
  • Show Sliced will force the calculation of sliced SMCE output maps when 'Define' was used during slicing.
