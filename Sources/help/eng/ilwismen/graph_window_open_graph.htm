Graph window

 File menu

Open a graph

 

Open another existing graph in the current graph window.

If the graph that is currently displayed in the graph window has changed, first a question will be asked whether the current graph should be saved or not.

This dialog box appears:

  • when you choose Open from the File menu in a Graph window;
  • when you click the Open button in the toolbar of the graph window.

Dialog box options:

Select a graph:

Select a graph that you wish to open in the graph window.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The graph window will show the selected graph. You can modify the graph according to your wishes and optionally print it.

Tip:

  • To open a graph in a new graph window, you can double-click a graph in a Catalog, or click the right mouse button on a graph in a Catalog and choose Open from the context-sensitive menu.
