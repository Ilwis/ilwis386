Custom colors

Select or define a new custom color.

This dialog box appears when creating a new custom color:

  • when you click the create button next to a Color drop-down list box,
  • when you select the Custom item in a list box of ILWIS colors,
  • when you click the Custom Color command button when editing a class representation in the Representation Class editor.

Custom colors can for example be defined:

  • in Display Options dialog boxes of data and annotation layers,
  • when using double-click action in a map window (Edit Representation),
  • when creating or editing a representation Class in the Representation Class editor,
  • when creating or editing a representation Value or Gradual in the Representation Value/Gradual editor.
  • when editing graphs.

In the left part of this dialog box, you can select an existing color from the Basic Colors or Custom Colors palette.

In the right part of this dialog box, you can create your own color:

  • drag the cursor in the color refiner box and the arrow beside the luminosity bar to define your color, or
  • type numbers between 0 and 255 in the Red, Green, and Blue boxes, or the Hue, Sat (saturation), and Lum (luminosity) boxes.

The color you create is shown in the left side of the Color/Solid box.

If you want to use this custom color more often, you can add it to the existing custom colors:

  1. In the Custom Colors palette, select a box for the color. Select an empty box or one that contains a color you want to change to the new color.
  2. Choose the Add Custom Colors button.
  3. Create any other colors you want, and add them to the palette.

The colors available in the Custom Colors palette remain available during one ILWIS session.

When satisfied, choose the OK button.
