Polygon editor

 Edit menu

Edit ...
 

Edit the class name(s) or value(s) of the current selection of polygons, or edit the ID of a single selected polygon (when using the mouse). The polygon(s) that you wish to edit should first be selected with the mouse in Normal/Select mode.

Edit the class name, ID, or value of a single polygon (when using the digitizer).

This dialog box appears:

  • when you select one or more polygons with the mouse in Normal/Select mode, and:
    • choose Edit from the Edit menu, or
    • choose Edit from the context-sensitive menu, or
    • press the Enter key on the keyboard.
  • when you press the Edit Value digitizer button on a polygon.

 

For more information on:

  • using the mouse, refer to Polygon editor : functionality, section Working with the mouse or Polygon editor : Normal/Select mode.
  • using the digitizer, refer to the Polygon editor : functionality, section Working with the digitizer.

 

Mind: In a polygon map that uses an ID domain or a system UniqueID domain, you can only edit polygons one by one.

 

Tip: To edit the class name, ID or value of a single polygon, you can also double-click the polygon with the mouse. You will then not get this Edit dialog box, but you can select a class name from a list, or type an ID or value in a small text box.

Dialog box:

Area (ha) & Length, or Nr of polygons:

 

For your information:

  • when a a single polygon has been selected, the area and length of the selected polygon are listed;
  • when multiple polygons have been selected, the number of selected polygons is listed.

Class name/ID/Value:

To assign a class name, ID, or value to the selected polygon(s), and depending on the domain of the polygon map: select a class name from the drop-down list, type an ID, or type a value.

Class names, IDs, values, etc.

  • For a map using a class domain: select an existing class name from the drop-down list.

    • To assign a new class name: select <new> from the list, then type the new class name, and, optionally, type a class code and choose a color for the new class.

    A new class name will be automatically added to the class domain.

     

  • For a map using an ID domain: type an ID for the single selected polygon.

    • When an ID is assigned that already exists in the map, a question is asked whether you wish to use the same ID again or not.

    A new ID will be automatically added to the ID domain.

     

  • For a map using a value domain: type a value.

    • The value should fall inside the minimum and maximum value of the value domain used by the map.

     

  • For a map using a Bool domain: select True, False, or ? (= undefined) from the drop-down list.

 

When you choose the '?' question mark, the selected polygon(s) will have not get a class name or value; they will have the undefined value.

Tips:

  • The width of class names, IDs, or values as shown in the Edit dialog box, depends on the width of the domain. You can change the width of a domain by editing the Domain Properties sheet.
  • Dependent maps cannot be edited; first break the dependency links of the map in the Polygon Map Properties sheet.
