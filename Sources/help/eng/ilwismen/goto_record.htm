Table window

 Records menu

Goto Record   
 

 

Goto a specific record in the table.

Dialog box options:

Record:

Select a record in the table: depending on the domain of your table, select a class name or ID from the list, or type a record number.

 

  • The table will scroll to the record and the record will be slightly highlighted.
  • The record will also be displayed in Record View.
  • The order of the records in the table remains the same.

Tip:

You can also click a gray record button in a table to go to a specific record.
