Layout editor

 Edit menu

Copy to clipboard

 

Copies a selected map view or a selected annotation element from the layout to clipboard as text, picture or bitmap.

Copied map views and annotation can be pasted into other Windows applications which use text, pictures or bitmaps.

 

  • Only one map view or one annotation element can be copied at a time.
  • When copying a map view to clipboard, the entire map view will be copied, regardless whether you have zoomed in on a part of the map view or not.

To select a map view or an annotation element:

  • click a map view or an annotation element in the layout editor or in the Layout Management pane.

To copy:

  • select a map view or an annotation element, and:
    • select Copy from the Edit menu, or
    • click the Copy button in the standard toolbar of the layout editor,
    • press the Ctrl+C keys on the keyboard.

Tips:

  • Afer pasting, you may need to resize the pasted pictures, bitmaps or text. To resize an object, you can select an object and drag its size handles; you can also adapt the width and height values in the Layout Management pane.
  • It is not a very good idea to copy and paste annotation elements from one layout into another layout: as most objects are copied as pictures, the meaning of coordinates, sizes of a scale bar, etc. will get lost.

    • When you wish to create a layout which should look almost exactly as another layout, it is therefore advised to:

    • save the layout with another name using Save As;
    • in the new layout, replace map views if required: select a map view, then click it with the right mouse button and choose Replace Map View from the context-sensitive menu;
    • in the new layout, edit annotation elements if required: double-click the annotation element you wish to edit.

    For more information, see Layout editor : functionality.

  • To copy the whole layout editor window including the window borders, title bar, menu bar, toolbar, status bar, etc.: press Alt+PrintScrn on the keyboard.

    • The layout editor window will be copied as a bitmap to the clipboard.

  • You can use Windows' Clipboard Viewer to inspect the current contents of the clipboard.
  • To store a complete layout as a bitmap on disk, use the Export to Bitmap command on the File menu.
