Map window

 Layers menu

Background color
 

Select a color for the undefined parts of map(s) displayed in a map window. By default, undefined parts of maps are displayed as white.

Everything in the map(s) which is undefined is considered as background, i.e.:

  • everything which does not have a class name or an ID as defined in the class domains or ID domains as used by the map(s), and
  • everything that has an undefined value according to the used value domains.

Dialog box options:

Background:

Select a color from the drop-down list box. To create your own color, select Custom at the bottom of the list or click the little create button.

 

Tip: The color of the map window background itself, i.e. the space outside the map(s) displayed in a map window (by default dark gray), can be changed by adapting the Windows Display/Appearance Settings. On the Appearance tab of the Windows Display Properties sheet, choose another color for Application Background.
