Polygon editor

 File menu

Extract segments
 

Extract all segments from the polygon map and create a segment map of it.

This operation can be used if you wish to edit the polygon boundaries as segments in the segment editor, and then repolygonize the segments.

This operation can also be used when you want to obtain different polygon maps out of one polygon map: extract segments, assign different class names, IDs or values to the segments, then polygonize the segment map again using a mask.

Dialog box options:

Output segment map:

Type a name for the output segment map that will contain the extracted polygon boundaries.

Domain:

Select an existing domain for the output segment map. All extracted segments obtain the first class name, ID or value that is found in the selected domain.

Edit:

Select this check box to continue in the segment editor when the segment extraction has finished. Clear this check box if you do not want to see the segment map immediately: you simply define how the map should be created; you remain in the polygon editor.

Tip:

To extract boundaries of specific polygons, i.e. by using a mask, use the Polygons to Segments operation.
