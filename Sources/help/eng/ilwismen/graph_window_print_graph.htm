Graph window

 File menu

Print options
 

Print the contents of the current graph window.

This dialog box appears:

  • when you choose Print from the File menu in a graph window;
  • when you click the Print button in the toolbar of a graph window.

The Print command and the Print button are only available when the pane displaying the graph(s) is active.

Dialog box options:

  Fit on page:

Select this option when the graph should be printed on a full page.

  Width in cm:

Select this option when you wish to obtain a printed graph with a certain size. Subsequently, specify the desired width of the printed graph in centimeters.

When you click OK, a standard Windows Print dialog box will follow in which you can select a printer and specify some settings for the selected printer.

When you click OK in the Windows Print dialog box, the graph will be printed.

 

Tip: To install a printer and choose a default printer, see your Windows documentation.
