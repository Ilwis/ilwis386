Properties
 

Open the Property sheet of one or more selected object(s) in the currently active Catalog to view and/or edit the properties of the object(s).

A Property sheet is listing all kind of important information on an ILWIS object. In fact, all relevant information as stored in an object's object definition file is displayed. Furthermore, Property sheets are designed to handle the properties of dependent objects.

Depending on the selected object(s), a Property sheet is opened:

 

Raster map properties

Polygon map properties

Segment map properties

Point map properties

Table properties

Column properties

Map list properties

Object collection properties

Map View properties

Layout properties

Annotation Text properties

Domain properties

Representation properties

Georeference properties

Coordinate system properties

Histogram properties

Sample Set properties

Two-dimensional Table properties

Matrix properties

Filter properties

Function properties

Script properties

Tip:

  • To view and edit properties of an object, you can also click with the right mouse button on an object in the Catalog, a context-sensitive menu appears, select Properties. The Properties sheet of the selected object is displayed.
