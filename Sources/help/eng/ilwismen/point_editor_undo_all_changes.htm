Point editor

 File menu

Undo all changes   
 

 

Restore all changes in this point map since the last Save.

The Undo All Changes command can be used to restore all recent changes. The point editor has no possibility to undelete specific points that were accidentally deleted before.

When using this Undo All Changes command:

  • all changes you made to the point map since you last used Save will be restored;
  • if you have not used Save so far, all changes since you started editing the map will be undone.
