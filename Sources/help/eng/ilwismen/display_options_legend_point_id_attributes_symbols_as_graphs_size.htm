Insert/Edit Legend (ID point map + attributes)

Symbols as graphs: size

 

Insert or edit a size legend for an ID point map with an attribute table where the points are represented as pie charts, bar charts or composed bars. During display in a map window, you should have used the Stretch check box in the Symbols as Graphs dialog box so that point values are displayed as graphs on different sizes depending on a specified Size column. The legend will be added to your layout.

The legend will have the following appearance:

  • when the point map is displayed as pie charts:

    • the size legend will consist of a number of sized circles, one inside the other; and text values explaining the sizes of the circles;

  • when the point map is displayed as bar charts or as composed bars:

    • the size legend will consist of a sized bar, and text values explaining the height of the bar(s).

The total size of the pie charts, bar charts or composed bars is determined by the specified Size column and the specified minimum and maximum Stretch values in the Symbols as Graphs dialog box.

This dialog box appears:

  • when you displayed an ID point map with an attribute table, and
  • in the Display Options - Point Map dialog box, you chose the Symbols as Graphs option, and clicked the Details button, and
  • in the Symbols as Graphs dialog box, you selected to display the map as pie charts, bar charts or composed bars; furthermore, you must have specified minimum and maximum stretch values in that dialog box, and,
  • you saved the map as a map view,
  • in the layout editor, you selected the map view and choose to add a size legend.

Dialog box options:

Values:

Select the number of values that should be displayed as text next to the legend.

When you select value 3:

  • for pie charts: 3 circles will be drawn inside each other with 3 accompanying texts;
  • for bar charts and composed bars: a bar consisting of 2 parts on top of each other will be drawn with 3 accompanying texts.

The values specified as the minimum and maximum stretch value in the Symbols as Graphs dialog box will always be used as texts in the legend.

Font name:

Select the font in which the legend texts should appear; you can select an installed font from the list.

Font size:

Type a value for the font size (pt) in which your legend texts should appear.

Bold:

Select this check box to have legend texts appear in bold.

Italic:

Select this check box to have legend texts appear in italics.

Tips:

  1. When creating a legend for a point layer displayed as pie charts, bar charts or composed bars: you may have to reduce the number of number of values and/or to reduce the legend text font size to obtain a good legend.
  2. To adapt the symbol sizes both in the point map and in the legend, or to have round values appear in your graph legend:
    • double-click the map view containing the point map in the layout editor,
    • use the appearing Layer Management pane to adapt the display options of the point map; in the Symbols as Graphs dialog box, you can then specify another column for the Size, and other values for Stretch and Minimum and Maximum Stretch Size.

    Do not forget to save the changed map view.

  3. In the Layout editor, a legend can be sized by dragging for instance one of the corner size handles of a selected legend to a new position. The mouse pointer changes into a double-sided arrow when at the right position to start dragging. Only the font used by the legend will be automatically adapted to the new size; the graph sizes will remain.
  4. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
