Edit Layer
 

Edit a map that is currently displayed in the map window.

Editing a map means that you want to:

  • change values or meaning of pixels in a raster map, or
  • change values or meanings of polygons in a polygon map, or
  • change values or meanings of segments in a segment map, change the shape of segments, or insert new segments or delete existing ones, or
  • change values or meanings of points in a point map, or insert new points or delete existing ones, or move points to a new position.

Depending on the selected layer, the Pixel editor, the Polygon editor, the Segment editor or the Point editor is opened.

Mind:

Dependent maps and maps marked as read-only cannot be edited; also maps using a Picture domain or a Color domain cannot be edited. When you wish to edit a dependent map, you first have to break the dependency links of the map in the map's Properties sheet.
