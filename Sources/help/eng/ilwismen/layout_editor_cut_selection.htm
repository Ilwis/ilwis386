Layout editor

 Edit menu

Cut

 

Copies a selected map view with additional annotation or a selected annotation element from the layout to clipboard as text, picture or bitmap, and deletes the selection from the layout.

Copied map views and annotation can be pasted into other Windows applications which use text, pictures or bitmaps.

Note:

When you use Cut while you selected a map view for which you already added annotation (legends, map borders, North arrow, scale bar, scale text), then:

  • only the map view will be copied as a picture to clipboard,
  • the map view and the annotation that has been added for this map view will be deleted from the current layout.

To select a map view or an annotation element:

  • click a map view or an annotation element in the layout editor or in the Layout Management pane.

To cut:

  • select a map view or an annotation element, and:
    • select Cut from the Edit menu, or
    • click the Cut button in the standard toolbar of the layout editor,
    • press the Ctrl+X keys on the keyboard.

Tips:

  • Afer pasting, you may need to resize the pasted pictures, bitmaps or text. To resize an object, you can select an object and drag its size handles; you can also adapt the width and height values in the Layout Management pane.
  • It is not a very good idea to copy and paste annotation elements from one layout into another layout: as most objects are copied as pictures, the meaning of coordinates, sizes of a scale bar, etc. will get lost.

    • When you wish to create a layout which should look almost exactly as another layout, it is therefore advised to save the current layout with another name and replace the map view(s) and/or edit the annotation. For more information, refer to the tips in the Copy topic.

  • To copy the whole layout editor window including the window borders, title bar, menu bar, toolbar, status bar, etc.: press Alt+PrintScrn on the keyboard.

    • The layout editor window will be copied as a bitmap to the clipboard.

  • You can use Windows' Clipboard Viewer to inspect the current contents of the clipboard.
