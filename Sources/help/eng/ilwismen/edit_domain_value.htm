Edit a Value domain

Edit the minimum and maximum values (the value range) that can be used by this domain and the precision of a user-defined Value domain.

This dialog box appears:

  • when you double-click a user-defined Value domain in a Catalog,
  • when you press the right mouse button on a user-defined Value domain in a Catalog, and choose Open from the context-sensitive menu,
  • when you you select a user-defined Value domain in a Catalog, and choose Edit Object from the Edit menu in the Main window,
  • when you choose Domain from the Edit menu in a map window and subsequently you select a layer that uses a user-defined Value domain.

Dialog box options:

Description:

You can change the description of the domain. The description is visible on the status bar of the Main window when moving the mouse pointer over the domain in the Catalog.

Min   Max:

You can change the minimum and maximum value of this domain. This is also called the value range.

The specified minimum and maximum values of a value domain will be used as the default minimum and maximum values for any new maps or columns that use this domain.

When you create a representation for this value domain, the specified minimum and maximum values will determine the absolute lower and upper boundaries of that representation.

Precision:

You can change the value for the precision of this domain.

By specifying a precision of 0.01, values will use 2 decimals; by specifying a precision of 25, values will increase or decrease in steps of 25, etc.

The combination of specified values for minimum, maximum and precision implies the store type of raster maps that use this domain.

Fixed range:

Select this check box if any output map or column (created by a calculation or an operation) and which will use this domain, must use values as specified for the value range (Min Max) for this domain. During calculation or operations, the value range boxes for an output object which uses this domain will be grayed out. Any calculated values outside this range will become undefined.

Clear this check box when you wish to keep the possibility to set the value range yourself (during calculations or operations) for any output map or column that will use this domain. During calculations or operations, the value range for the output object which uses this domain is still changeable.

Fixed precision:

Select this check box if any output map or column (created by a calculation or an operation) and which will use this domain, must use the precision as specified for this domain. During calculation or operations, the precision box for an output object which uses this domain will be grayed out. Any calculated values that do not fit the precision will be rounded to the nearest value that fits the precision of the domain.

Clear this check box when you wish to keep the possibility to set the precision yourself (during calculations or operations) for any output map or column that will use this domain. During calculations or operations, the precision for the output object which uses this domain is still changeable.

Unit:

Select this check box when the values in this domain use a certain unit, for instance mm, ha, or m3/ha/year, etc. Subsequently, type a text for the Unit.

The specified Unit text will appear:

  • when using left mouse Info in a map window,
  • in legends, and
  • in the pixel information window.

Clear this check box when you do not wish to view units for the values of this domain.

Width:

You can change the value of the width to increase or decrease the number of characters shown.

Tip:

To view the representation that is currently linked to your value domain, or to create a new representation value for your value domain, you can:

  • open the Properties of your value domain and in the Domain Properties dialog box; you can click the create button next to the Representation list box; or
  • open the map with your value domain, and in the map's Display Options dialog box; you can click the create button next to the Representation list box.

The Create Representation dialog box follows.
