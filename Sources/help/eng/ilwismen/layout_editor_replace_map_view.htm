Replace Map View

 

In the current layout, replace a selected map view by another map view.

This option is designed to quickly produce multiple similar layouts, for instance each with different map views, and similar annotation.

This dialog box appears:

  • when in the layout editor, you select a map view in a layout,
  • use the right mouse button, and
  • select Replace Map View from the context-sensitive menu.

Dialog box options:

Map view:

Select another map view.

A small dialog box will follow in which you can specify an initial scale on which the map view should be displayed on the paper.

Tips:

  • You may need to adapt the scale on which the new map view should be displayed (and printed later on).
  • You can save the layout with a different name with the Save As command on the File menu, and optionally edit some annotation.
