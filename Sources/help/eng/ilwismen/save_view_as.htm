Map window

 File menu

Save View As ...
 

Save the contents of the current map window as a map view.

All layers in the map window and their display settings as well as the georeference of the map window are stored. When you open a map view later on, the map window will appear exactly as it was saved. You do not need to add individual maps to the map window anymore, nor specify their display options.

Note:

When you are in the Layout editor and save a map view, please go to topic Layout editor : Save Map View As.

This dialog box appears:

  • in a map window, by choosing the Save View As command on the File menu,
  • in a map window, when you save the contents of the map window for the first time, by choosing the Save View command from the File menu, or
  • in a map window, when you have not yet saved the multiple layers displayed in the map window as a map view before, by choosing the Create Layout command from the File menu.

Dialog box options:

Map View Name:

Type a name for the map view.

Title:

Type a title (=description) for the map view. The description of a map view appears in the title bar of a map window when the map view is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the map view in the Catalog.

For presentation purposes and printing, you need to create a layout to which you can add one or more map views and make a final composition. To the layout, you can further add annotation such as legends, map borders including ticks and coordinates for grid lines or a graticule, north arrows, scale bars and scale texts, texts (such as a title), boxes, bitmaps and pictures, and a page border.
