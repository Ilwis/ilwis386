Georeference Tiepoints editor

Delete Tiepoint

 Edit menu
 

Deletes a tiepoint or a ground control point.

Dialog box options:

Tiepoint number:

Type the number of the tiepoint you want to delete.

The tiepoint is removed from the map window as well as from the tiepoint table. The deleted tiepoint is no longer part of the georeference tiepoints which you are editing.

Tips:

  • To delete a tiepoint, you can also select a tiepoint in the tiepoint table (click the tiepoint number in the table) and press Del on the keyboard.
  • To delete a tiepoint, you can also select a tiepoint in the tiepoint table (click the tiepoint number in the table), then click the right mouse button and choose Delete from the context-sensitive menu.
  • To make a tiepoint inactive for this georeference: in column Active, set the value on False by typing an F.
