Weigh - Rank Ordering   
 

 

Manually assign a rank order to multiple factors and optional groups that occur under the selected main goal, or to multiple factors and optional groups that occur under the selected sub goal.

In the rank order method, you have to place all factors and optional sub goals in a rank-order (most important item at the top). From this rank-order, numerical weights are calculated using either the expected value method or the rank sum method. For more information on the Rank Ordering method, Expected value method, or Rank sum method, refer to Weigh methods (additional information).

Dialog box options:

Expected Value:

Select this option if you wish to calculate weight values from the specified rank-order, using the expected value method. For more information, refer to Weigh methods (additional information), section Expected value.

Rank Sum:

Select this option if you wish to calculate weight values from the specified rank-order, using the rank sum method. For more information, refer to Weigh methods (additional information), section Rank sum.

Rank Sum Power:

The Rank Sum Power text box appears when you select the Rank Sum option. By default the power is 1.000000.

Left-hand list box :

The left-hand list box displays the factors and/or groups of factors that you wish to rank-order.

  • Select an item in the left-hand list box, and click the or the button to move the item to the right-hand list box.
  • You can also drag an item from the left-hand list box to the right-hand list box.

Right-hand list box:

The right-hand list box displays the rank-order of the factors and/or groups of factors. The factor or group at the top of the list is considered the most important, the factor or group at the bottom of the list is considered the least important.

Press this button to move a selected item from the left-hand list box to the right-hand list box. The selected item will be positioned at the same rank level as the item currently selected on the right, which is usually the item positioned last. You can also drag an item from the left-hand list box to the right-hand list box: drop the item onto the item positioned last.

Press this button to move a selected item from the left list box to the right list box. The selected item will be positioned one rank level lower than the item positioned last. You can also drag an item from the left-hand list box to the right-hand list box: drop the item below the item positioned last.

Press this button to move a selected item back from the right-hand list box to the left-hand list box. You can also drag an item from the right-hand list box to the left-hand list box: it does not matter where you drop the item.

Resulting Weights column:

 

This column displays the factors and groups and their resulting weight values. Weights are automatically calculated using the expected value method or the rank sum method. For more information, refer to Weigh methods (additional information). The sum of all resulting weights is always 1.

Choose other method

Click the Choose other method button to open the Weigh dialog box to choose another Weigh method.

Note:

After weights have been assigned, the criteria tree viewer will show the following:

  • The selected weight method is shown after the selected main goal, or the selected sub goal.
  • The assigned weight values are shown in front of the factors and optional groups of factors that occur under the selected main goal or under the selected sub goal.
