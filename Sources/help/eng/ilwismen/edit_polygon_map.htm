Edit Polygon Map
 

Select a polygon map that you want to edit while using the current contents of the map window as a background. This is also called screen digitizing.

This dialog box appears when from the Edit menu in a map window, you choose Edit Other Map, Polygon editor.

Dialog box options:

Polygon map:

Select the polygon map you wish to edit.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The Polygon editor is started; you can start editing the selected polygon map.

Tips:

  1. Dependent maps and maps marked as read-only cannot be edited. First break the dependency link or remove the read-only flag in the Polygon Map - Properties sheet.
  2. To directly edit a polygon map:
    • in the Catalog, select an editable polygon map, then open the Edit menu in the Main window and choose Edit Object, or
    • in the Catalog, click an editable polygon map with the right mouse button and choose Edit from the context-sensitive menu, or
    • in a map window where the polygon map is displayed, open the Edit menu and select Edit Layer.
