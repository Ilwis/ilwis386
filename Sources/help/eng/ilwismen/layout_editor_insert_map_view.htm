Layout editor

 Insert menu

Insert Map View

 

Insert a map view in your layout.

Dialog box options:

Select map view:

Select the map view that you wish to add as a layer to the current layout.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

A small dialog box will follow in which you can specify an initial scale on which the map view should be displayed on the paper.

Tips:

  • You can add multiple map views to a layout.
  • You can also drag one or more map views from a Catalog or from Windows Explorer to a layout.
  • To add layers to or to remove layers from a map view in a layout, or to change display options of layers in a map view in a layout, double-click a map view in the layout.
  • To save a changed map view in your layout:
    • select the map view in the layout, and
    • click the Save button in the standard toolbar, or
    • click the selected map view with the right mouse button and choose Save View or Save View As from the context-sensitive menu.
  • To replace a map view in the current layout with another map view:
    • select the map view you wish to replace,
    • choose Replace Map View from the context-sensitive menu.
  • To save a layout in which you replaced a map view:
    • click somewhere in the white space of the layout, and
    • click the Save button in the standard toolbar, or
    • choose Save or Save As from the File menu of the layout editor.
