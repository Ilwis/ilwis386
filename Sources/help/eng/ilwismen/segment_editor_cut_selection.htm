Segment editor

 Edit menu

Cut
 

Copy selected segments with their coordinates and class names, identifiers or values to the clipboard, and then delete the selected segments from the current segment map.

Subsequently, the cut segments can be pasted as segments into another segment editor, as points into a point editor and/or into any other application which uses the picture format or the text format (e.g. a word processor or a data base).

To cut:

Select one or more segments, and:

  • choose Cut from the Edit menu, or
  • press Ctrl+X on the keyboard.

For more information on selecting segments, see Select Mode.

Tips:

  • To recover segments which were accidentally deleted, you can use:
    • Undelete segments: to undelete specific segments;
    • Undo all changes: to restore all changes made in the segment editor until the last time you used Save.
  • You can use Windows' Clipboard/Clipbook Viewer to inspect the copied segments on the clipboard in text format and in picture format. For more information on the Clipboard Viewer, see the Windows Help.
  • An example of segments copied to the clipboard is presented in Segment editor : Copy.
