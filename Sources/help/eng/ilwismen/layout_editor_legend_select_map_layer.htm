Insert a Legend / Map Layer Selection

 

When a selected map view in a layout contains multiple map layers, select the data layer for which you want to create a legend.

This dialog box appears:

  • when in the layout editor, you select a map view that contains more than one data layer, and then:
    • choose the Legend command from the Insert menu of the layout editor, or
    • click the Legend button in the Layout toolbar of the layout editor.

Dialog box options:

In the list box, select the map layer for which you want to create a legend.

Depending on the domain of the map layer you select, a dialog box will appear in which you can further specify the appearance of the legend.

When you select:

  • a raster, polygon, segment or point map layer with a Class domain,

    • the Display Options - Legend (Class) dialog box appears;

  • a raster, polygon or segment map layer with a value domain or the Image domain,

    • the Display Options - Legend (Image/Value) dialog box appears;

  • a point map with a value domain,

    • the Display Options - Legend (point values as sized symbols) dialog box appears;

  • an ID point map with an attribute table, displayed as arrows,

    • the Display Options - Legend (point ID + attributes, symbols as arrows) dialog box appears;

  • an ID point map with an attribute table, displayed as pie charts, bar charts, or composed bars,

    • first, a Create Legend dialog box appears;

    then, either the:

    • the Display Options - Legend (point ID + attributes, symbols as graphs, size) dialog box, (for pie charts, bar charts and composed bars), or
    • the Display Options - Legend (point ID + attributes, symbols as graphs, color) dialog box appears (for pie charts, bar graphs, composed bars, and volume cubes). For volume cubes, this dialog box will directly appear.
