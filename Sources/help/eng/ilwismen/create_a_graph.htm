Create a graph

Create a new Graph from Columns in a graph window.

This dialog box appears:

  • when you choose the Create Graph command from the File menu in a table window,
  • when you click the Create Graph button in the toolbar of a table window,
  • when you choose the Create Graph command from the File menu in the Main window,
  • when you double-click the New Graph item in the Operation-list,
  • when you click a table with the right mouse button in a Catalog and choose Create Graph from the context-sensitive menu, or
  • when you choose the New Graph command from the File menu in a graph window, in case the graph window already displays a graph from columns, a graph from formula, a least squares fit, or a semi-variogram model.

Dialog box options:

Table:

Select a table which contains the column(s) that you wish to show as a graph.

X-axis:

Select this check box when you wish to use a column for the X-axis of the graph.

Clear this check box when the domain of the table should be used on the X-axis of the graph.

Y-axis:

Select a column for the Y-axis of the graph. The list box shows all columns of the table.

A new graph window will be opened, displaying the graph.

  • When a value column is selected for the X-axis, you will by default obtain a point graph.
  • When using the domain of a table, and when the table uses system domain None, you will by default obtain a point graph.

  • When a class or ID column is selected for the X-axis, you will by default obtain a bar graph.
  • Similarly, when using the class or ID domain of a table, you will by default obtain a bar graph.

Tips:

  • When creating a new graph in an existing graph window and when the existing graph has changed, a question will first be asked whether you wish to save the existing graph.
  • To edit the appearance or the name of the graph, double-click the graph layer in the Graph Management pane: the Graph Options - Graph from Columns dialog box will appear.
  • To adapt the title, the X-axis, or the Y-axis (left or right) of the graph, double-click the appriopriate item in the Graph Management pane or in the graph window itself: the Title, X-axis, Y-axis (left), or the Y-axis (right) dialog box will appear.

    • The grid lines of the graph can be modified in the X-axis and Y-axis dialog boxes.

  • To add another graph layer to an existing graph displayed in a graph window, choose Add Graph from the Edit menu in the graph window. You can also click the Add Graph button in the toolbar of the graph window.
