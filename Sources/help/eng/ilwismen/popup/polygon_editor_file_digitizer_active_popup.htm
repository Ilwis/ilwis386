Digitizer, Active

The Digitizer, Active command should be preceded by a check mark before you can edit polygons in the polygon editor with a digitizer.

When the Digitizer, Active command is active, a small digitizer window appears titled Edit Polygons. This window contains the digitizer commands.

The Digitizer, Active command may be switched off if you wish to edit the polygon map with the mouse only, and not with a digitizer.

Notes:

  • The Digitizer, Active command is grayed out when you did not reference an analog map on your digitizer yet. Then, first choose Digitizer, Map Reference from the File menu.
  • If also the Map Reference command is grayed out, you have to switch on and/or setup your digitizer (see Main window : Setup digitizer wizard).
