Problem Definition mode

Choose the Problem Definition mode to create, fill and edit a criteria tree:

  • specify a textual name for the main goal,
  • insert new criteria, i.e. insert factors, insert constraints and/or insert groups of factors,
  • edit the names of (groups of) criteria,
  • delete (groups of) criteria,
  • select appropriate criteria maps (input data),
  • specify output map names for the main goal and optional sub goals, and
  • decide about the number and the names of the alternative situations.

When the Problem Definition mode is active:

  • an option bullet appears in front of this menu command, and
  • the Problem Definition mode button in the toolbar appears down.

When the Problem Definition phase is completed, the criteria tree can be fixed and the application can be switched to the Multicriteria Analysis mode.
