Copy

Copy the objects that are currently selected in the active Catalog to the clipboard. After you copied objects to the clipboard, you can use the Paste command to paste these objects into another Catalog (directory), map list or object collection, or into a map window.

To copy objects to the clipboard, you can also use the Ctrl+C keys on the keyboard. To copy a selected object to another directory, you can also use the Copy Object To command on the Edit menu.

For more information on copying objects, see also How to copy objects.
