Manual ordering

Allows you to order the domain items manually; an option bullet appears in front of this menu command. Subsequently, select a domain item in the domain Class/ID editor and drag it to a new position in the list and continue to do so until satisfied.

The sorting order of the classes or IDs in a domain determines the order of these items in a legend in a map window, the order of classes in a representation class, and the order of records in an attribute table.
