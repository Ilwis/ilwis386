Page border

Display page border lines along the margins of this layout.

 

  • When the Page Border command is selected and marked with a check mark, the page borders will be drawn as a black line on the layout, and the page border will also be printed (without further options).
  • When the Page Border command is cleared, the page borders will be drawn in gray in your layout for orientation purposes; the page border will then not be printed.

Tips:

  • To set the paper margins, refer to Page Setup.
  • To obtain borders around a map view, select a map view in your layout and use the Map Border button or the Map Border command on the Insert menu; use the Neatline tab or the Outline tab.
  • To obtain a sizeable box around text or around a legend in a layout, select a map view in your layout, and use the Box button or the Box command on the Insert menu.
