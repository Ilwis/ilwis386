Select All

Selects all text items in the Annotation Text object. When you subsequently select the Edit command from the Edit menu or from the context-sensitive menu, you can make changes (font, font size, color, etc.) to all text items in the Annotation Text object.

To clear the whole selection, press Esc on the keyboard. To exclude some text items from the selection, press and hold the Shift key down and click the text items which you want to exclude.
