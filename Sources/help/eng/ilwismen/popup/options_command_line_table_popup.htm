Show/Hide Command line

Show or hide the command line of a table window. The command line is shown when a check mark appears in front of this menu command.

The command line of a table window is used to perform calculations using one or more columns in the table.

Operations on columns like Aggregation, Join, or Least Squares Fit, can be found on the Columns menu.

You can move the command line and dock it along an edge of the window; you can also make the command line floating. Settings are not stored.
