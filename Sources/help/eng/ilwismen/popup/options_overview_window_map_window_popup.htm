Show/Hide Overview Window

Show or hide the Overview window in a map window. The Overview window displays the entire area of the maps displayed in the window; it allows you to quickly zoom in on a certain area, and it helps to orient yourself on which part of the entire map is shown in the map window. The Overview window is shown when a check mark appears in front of this menu command.

You can move the Overview window and dock it along an edge of the window; you can also make the Overview window floating. Settings are not stored.
