Normal

Return to Normal pointing mode after you used Zoom In, Zoom Out or Pan. The mouse pointer will change back into a normal pointer.

When the layout editor window is in Normal pointing mode, the Normal command is preceded by an option bullet .

Choosing the Normal command from the menu is the same as clicking the Normal button in the standard toolbar. You can also return to Normal pointing mode by pressing the Esc key or the Ctrl+N keys on the keyboard.
