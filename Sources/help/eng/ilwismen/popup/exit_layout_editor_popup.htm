Exit Editor

Close the Layout editor. If the layout has been changed, a question will be asked whether the layout should be saved or not.

 

Mind: You have to save any changed map views yourself before closing the layout!

 

To save a changed map view:

  • select a map view in the layout editor, and
  • click the selected map view with the right mouse button and choose Save View or Save View As from the context-sensitive menu, or
  • click the Save button in the standard toolbar of the layout editor.
