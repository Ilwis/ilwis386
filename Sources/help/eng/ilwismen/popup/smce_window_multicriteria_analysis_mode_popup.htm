Multicriteria Analysis mode

Choose the Multicriteria Analysis mode to:

  • standardize input maps and/or attribute columns,
  • assign weights to multiple factors under the main goal, and to assign weights to multiple factors under optional sub goals, and
  • generate output maps.

When the Multicriteria Analysis mode is active:

  • an option bullet appears in front of this menu command, and
  • the Multicriteria Analysis mode button in the toolbar appears down.

To edit the structure of a criteria tree, use the Problem Definition mode.
