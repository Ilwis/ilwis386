Close, Minimize or Restore All Data windows

ILWIS data windows are all ILWIS windows outside the Main window, e.g. map windows, table windows, pixel info windows, domain editors, representation editors, script editors, etc.

 

  • Close All: Close all ILWIS data windows.
  • Minimize All: Reduce all open ILWIS data windows to icons on the Windows taskbar.
  • Restore All: Restore all ILWIS data windows to their former size after you have minimized them.
