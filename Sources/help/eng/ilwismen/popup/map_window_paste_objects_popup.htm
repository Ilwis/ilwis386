Paste objects into map window

When one or more objects were copied to clipboard in a Catalog (by selecting objects in a Catalog and using the Copy command), the copied objects will be pasted into the current map window.

By using Copy and Paste, you can add to a map window:

  • multiple point, segment and polygon maps;
  • (multiple) raster maps, or map lists (to be displayed as a color composite).

Before the map(s) or the map list are displayed in the map window, you may need to specify display options.

Tips:

  • In the Display Options dialog box of a layer, you can specify whether a layer has to be displayed with a certain degree of transparency. For more information, see How to add extra layers to a map window.
  • You can also drag and drop objects between a Catalog and a map window to add objects to the map window.
  • To paste from clipboard, you can also press Ctrl+V on the keyboard.
  • You can use Windows' Clipboard Viewer to inspect the current contents of the clipboard.
