Edit domain

When editing the representation of a class domain, and when you want to assign the same or similar colors to various classes with the Edit Multiple Items command in the representation class editor, you may wish to change the order of classes in the domain.

By using this Edit, Domain command in the representation class editor, the domain class editor is opened with the class domain for which you are preparing a representation.

In the domain class editor:

  • select Manual Ordering from the Edit menu, and
  • drag the listed domain items (i.e. your classes) in the domain to the desired positions.

The position of the classes in the representation editor will change accordingly.
