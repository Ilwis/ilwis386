Arrange Catalogs

Organize all Catalogs, map lists and object collections that are open in the Main window.

 

  • Cascade: Arrange Catalogs so that all title bars appear; contents of Catalogs are overlapping.
  • Tile Horizontally: Arrange Catalogs so that they appear one below the other; the whole Catalog area in the Main window will be used.
  • Tile Vertically: Arrange Catalogs so that they appear one next to the other; the whole Catalog area in the Main window will be used.
