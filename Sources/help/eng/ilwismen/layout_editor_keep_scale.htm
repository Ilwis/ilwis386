Keep Scale

 

You can use the Keep Scale option on the View menu or the Keep Scale button in the standard toolbar to influence the behaviour of a selected map view in the layout when dragging the borders of the map view.

When you add a map view to a layout, you have to specify an initial scale for each map view. By default, this is the scale on which you saved the map view. However, if this would not fit on the paper size you specified for the layout, the scale of the map view is adapted so that it will fit the paper size.

When for a selected map view:

  • Keep Scale is active (button down):

    • the scale of the selected map view is fixed,

    the scale of that map view will not change when dragging a border of a map view;

  • Keep Scale is not active (button up):

    • the scale of that map view can be changed by dragging a border of a map view.

You must decide on the scale of each map view, because a map view will be printed later on its specified scale, and in the layout there should be enough space around map views to position annotation.

1. Keep Scale is active:

When for a selected map view, Keep Scale is active:

  • the Keep Scale button appears down;
  • the Keep Scale command is preceded by a check mark.

When dragging the borders of the map view:

  • the scale on which the map view is displayed remains the same;
  • the size of the map view in the layout will usually change;
  • the part of the study area shown by the map view will change.

 

Tip: You can still change the scale of the map view manually by typing a scale for the map view in the Layout Management pane.

When you drag the size handles of the map view to the outside:

  • if the map view was displayed as an entire map, the map view will remain as it was; the scale of the map view, the size of the map view, and the displayed study area will all remain the same.
  • if only part of the map view was displayed, then the displayed part of the study area will become larger, the size of the map view in the layout will become larger, but the scale on which the map view is displayed remains the same.

When you drag the size handles of the map view to the inside:

  • the part of the study area displayed by the map view and the size of the map view in the layout will become smaller, but the scale on which the map view is displayed remains the same.

 

2. Keep Scale is not active:

When for a selected map view, Keep Scale is not active:

  • the Keep Scale button appears up;
  • the Keep Scale command is not preceded by a check mark.

When dragging the borders of the map view:

  • the scale on which the map view is displayed will change;
  • the size of the map view in the layout will change;
  • the part of the study area shown by the map view will remain the same.

When you drag the size handles of the map view to the outside:

  • the (part of the) study area displayed by the map view will remain the same, but the size of the map view in the layout will become larger; the scale on which the map view is displayed will be automatically adjusted.

When you drag the size handles of the map view to the inside:

  • the (part of the) study area displayed by the map view will remain the same, but the size of the map view in the layout will become smaller; the scale on which the map view is displayed will be automatically adjusted.
