Insert/Edit Legend (Images/Value maps)

 

Insert or edit a legend for a raster, polygon or segment value map. The legend will be added to your layout.

A legend for a value map consists of a vertical color bar with gradual colors and corresponding values. The legend is based on the representation that is currently used by the layer.

Dialog box options:

MinMax:

Select this check box to specify the minimum and maximum value to be shown by the legend yourself.

Clear this check box if the legend should show the range of values of the domain; also all limits/boundaries inserted to the representation will be shown.

Upward:

Select this check box when the legend should display the largest value at the top and the smallest value at the bottom.

Clear this check box when the legend should display the smallest value at the top and the largest value at the bottom.

Ratio:

Specify a value for the height/width ratio of the color bar (1.0 <= real value <= 20.0).

When you use a ratio of 5, you will normally obtain 6 text values along your color bar.

Font name:

Select the font in which the legend values should appear; you can select an installed font from the list.

Font size:

Type a value for the font size (pt) in which your legend values should appear.

Bold:

Select this check box to have legend values appear in bold.

Italic:

Select this check box to have legend values appear in italics.

Tips:

  1. To adjust the number of values displayed along the color bar, specify another height/width Ratio.
  2. To have round values appear along the color bar, specify MinMax values yourself and think about the Ratio.
  3. In the Layout editor, a legend can be sized by dragging for instance one of the corner size handles of a selected legend to a new position. The mouse pointer changes into a double-sided arrow when at the right position to start dragging. The color bar and the font used by the legend will be automatically adapted to the new size.
  4. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
