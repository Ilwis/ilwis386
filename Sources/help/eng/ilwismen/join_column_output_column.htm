Join Column wizard

Select Output Column Name

 < Back | Finish >
 

On this last page of the Join Column wizard, you need to specify a new column name for the output of the join.

Dialog box options:

Output Column:

Type a new column name for the data that will be joined into the current table. You cannot use a column name which already exists in the current table.

The output column in the current table which will contain the joined data, is always a dependent column.

When you click the Finish button:

  • The Column Properties dialog box will appear; you can generally accept the suggested output domain, value range and precision for the output column.

 

Tip: Experienced users may also join columns from other tables by typing an expression on the TabCalc command line. For more information, see Table calculation : joining columns from other tables. This topic also provides some examples, both for using the wizard and for using the command line.
