Setup Digitizer wizard

Port Settings

 < Back | Next >
 

Specify the port settings: Baud Rate, Parity, Data Bits, and Stop Bits.

You must make sure that the settings used for Baud Rate, Parity, Data Bits, and Stop Bits during the configuration of the digitizer tablet will also be used (exactly the same figures) for the port settings.

Page options:

Baud Rate:

Specify a value for the Baud Rate;
choose the same value as you used for the configuration of the digitizer tablet.

Data Bits:

Select a value for the Data Bits (7 or 8);
choose the same value as you used for the configuration of the digitizer tablet.

Parity:

Select a value for the Parity (Even, Odd or None);
choose the same value as you used for the configuration of the digitizer tablet.

Stop Bits:

Select a value for the Stop Bits (1 or 2);
choose the same value as you used for the configuration of the digitizer tablet.

 

When setting up a digitizer for the first time, you may need to change the default port configuration. For all settings on this page, you MUST fill out the exact same values as you used when configuring the digitizer tablet.

When clicking the Next button, the Setup Digitizer - Automatic Detection page will appear.
