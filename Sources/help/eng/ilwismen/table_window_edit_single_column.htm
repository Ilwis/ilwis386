Edit fields in a single column
 

Edit a single field in a table, or edit a block of selected fields in a single column in a table.

To edit multiple fields in multiple columns, see Edit fields in multiple columns.

Notes:

  • The contents of fields of dependent columns cannot be edited, as these 'values' are the result of a calculation or a column operation.
  • The contents of fields of read-only columns and table-owned columns cannot be edited.
  • All changes are directly stored on disk. You CANNOT UNDO an edit action.

To edit a single field:

  1. Click a field: you can directly type a new value, select a class name, etc.
  2. Then, either press Enter to confirm the new value and stop editing, or
    use the Tab or Arrow keys to confirm the new value and continue editing in a next field.

To edit a selection of multiple fields in a single column:

  1. Select the block of fields you wish to edit.
  2. Then, either:
    • choose Edit from the Edit menu, or
    • click the selection with the right mouse button, and choose Edit from the context-sensitive menu.
  3. This Edit dialog box will appear:

    4. Edit record ...

    Specifies the first record selected.

    until record ...

    Specifies the last record selected.

    Column name:

    Type a value, choose a class, or type an ID.

    This value will be applied to all fields that are selected.

Note:

In the record view window, you can only edit fields one by one.
