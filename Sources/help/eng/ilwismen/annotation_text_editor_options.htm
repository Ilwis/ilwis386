Annotation Text editor

Options

 File menu
 

Change the working scale, or change from working in XY-coordinates to working in RowCols or vice versa.

This dialog box appears:

  • when in the Annotation Text editor, you choose the Options command from the File menu or from the context-sensitive menu.

Dialog box options:

Working scale:

Type a value for your new working scale; this can be for instance the scale on which you will print your maps later on.

Use XY-coordinates:

You can use XY-coordinates for the positions of your texts in the Annotation Text object when you are working on a map with a georeference corners, or a georeference tiepoints (conformal or affine transformation).

Use RowCols:

You should use RowCol-numbers for the positions of your texts in the Annotation Text object when you are working on a map with a georeference tiepoints (second order transformation or higher order transformation), a georeference 3D, a georeference direct linear, or a georeference orthophoto.

When changing the working scale, a message will follow asking you to update all font sizes as stored in the Annotation Text object (i.e. the table):

  • to have your texts appear in the same proportions on your map, answer this question with Yes; the font sizes in the table will be updated to your new working scale,
  • to use the same font sizes as currently stored in the Annotation Text object, answer No; your texts will either appear larger or smaller in your map window, depending on whether you specified a larger value for the working scale or a smaller one.

Tips:

  • Annotation Text objects using XY-coordinates can be used on any georeference of about the same area, e.g. drag drop from Catalog to map window.
  • Text items in an Annotation Text objects using RowCols can be easily aligned regardless of the georeference. For precise editing of RowCol numbers, it is adivised to open the annotation text object as a table.
  • An Annotation Text object should work in Row Cols when you have a georef direct linear, georef orthophoto, or a georef3D and you want to position texts in the map window but outside the DTM area (there are no coordinates defined outside your DTM area).
