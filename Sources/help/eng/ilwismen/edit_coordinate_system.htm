View/Edit coordinate system
 

View and/or edit the coordinate system that is currently used by the map window.

Depending on the type of coordinate system that is currently used by the map window, a dialog box or an editor will be opened:

  • coordinate system boundary only: the Edit Coordinate System Boundary Only dialog box will appear.
  • coordinate system projection: the Edit Coordinate System Projection dialog box will appear.
  • coordinate system latlon: the Edit Coordinate System LatLon dialog box will appear.
  • coordinate system formula: the Edit Coordinate System Formula dialog box will appear.
  • coordinate system tiepoints: the Coordinate system Tiepoints editor will be opened.

This option is not available when the map window only displays raster maps. Further, this option is not available when the current coordinate system is marked read-only.

Tip:

To view or edit a coordinate system, you can also double-click it in the Catalog. To make coordinate systems visible in the Catalog, refer to How to customize the Catalog.
