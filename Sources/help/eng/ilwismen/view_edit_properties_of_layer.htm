View/Edit Properties of Layer
 

View and/or edit the properties of a map that is currently displayed in the map window.

The Property sheet of a map shows important information. In fact, the Property sheet of an object lists information that is stored in the object definition file of an object; only information relevant to a user is shown. Furthermore, Property sheets are designed to handle the properties of dependent data objects, e.g. Make up-to-date, Edit Definition, Release Disk Space, etc. For a general introduction on object properties, see Basic concepts : properties of objects.

Depending on the type of map you selected:

  • the Raster Map Properties sheet follows;
  • the Polygon Map Properties sheet follows;
  • the Segment Map Properties sheet follows;
  • the Point Map Properties sheet follows.
