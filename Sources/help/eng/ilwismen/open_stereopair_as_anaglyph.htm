Show Stereo Pair as Anaglyph

Show a stereo pair as an anaglyph in a map window.

This dialog box appears:

  • when you open the Operations menu in the Main window, and choose Visualization, Stereo Pair as Anaglyph,
  • when you double-click the Anaglyph item in the Operation-list,
  • when you open the Visualization item in the Operation-tree and double-click the Anaglyph item,
  • when you click a stereo pair in a Catalog with the right mouse button and choose Visualization, Stereo Pair as Anaglyph from the context-sensitive menu.

Dialog box options:

Stereo pair:

Select the stereo pair that you wish to show as an anaglyph.

Directories:

If necessary, change the current directory.

Drives:

If necessary, change the current drive.

The Display Options - Stereo Pair as Anaglyph dialog box will follow.

Tips:

  • A stereo pair can also be viewed with a Stereoscope in a Stereo Pair - Stereoscope window.
  • To open a stereo pair, you can furthermore type a command on the command line of the Main window. For more information, see Appendices : commands.
