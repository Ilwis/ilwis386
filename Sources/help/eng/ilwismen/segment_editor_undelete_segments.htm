Undelete segments
 

Undelete specific segments that have been accidentally deleted.

The Undelete Segments command is active when a check mark appears in front of this menu command.

 

Mind: It is not possible to undelete segments which have been packed.

When using Undelete Segments:

  • all previously deleted segments will be displayed in the color as specified in the Segment editor - Customize dialog box;
  • then, with the mouse, and one by one, click those segments that you really wish to restore;
  • restored segments will be displayed in the Normal color as specified in the Customize dialog box.

When finished, you can switch off the Undelete Segments command again.

 

Tip: To undo all recent changes, i.e. all changes since the last Save, use the Undo All Changes command.
