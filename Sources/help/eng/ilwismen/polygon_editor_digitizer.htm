Digitizer
 

When you wish to use the digitizer to edit the class name, ID or value of single polygons in your polygon map:

 

  1. Your digitizer should be set up properly:

    2. the Setup Digitizer command, on the File menu of the Main window should be preceded with a check mark .

     

  2. You should have performed a Map Reference, so that the coordinates sent by the digitizer can be translated into real world coordinates:

    3. the Map Reference command on the File menu of the Main window, or the Digitizer, Map Reference command on the File menu of the polygon editor should be preceded with a check mark .

     

  3. You should make the digitizer Active in the editor, so that a floating window appears with the digitizer commands (the digitizer menu):

    4. the Digitizer, Active command on the File menu of the polygon editor should be preceded with a check mark .

The digitizer menu with the digitizer commands looks like:

 

 

The coordinates at the top of the digitizer menu show the real world coordinates of your map (this is the result of Map Reference); these coordinates should change when you move the digitizer cursor over the digitizer tablet.

For detailed information on using the digitizer to edit the names, IDs or values of polygons, see Polygon editor : functionality, section Working with the digitizer.

Tip:

You can also use the mouse to edit the names or values of multiple selected polygons. For more information, see Normal/Select Mode and Edit.
