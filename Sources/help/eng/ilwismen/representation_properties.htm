Representation Properties

The Representation Properties sheet lists all kind of important information on a selected representation. In fact, all relevant information that is stored in the object definition file of the representation is displayed. For more information, refer to Basic concepts : properties of objects.

The Representation Properties sheet appears:

  • when you click a representation in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select a representation in a Catalog, and choose Properties from the Edit menu in the Main window;
  • when you open a representation, and choose Properties from the File menu in the representation class editor, or from the File menu in the representation value/gradual editor;
  • when you open a representation, and click the Properties button in the toolbar of the representation class editor or the representation value/gradual editor.

Representation tab:

Object type and name:

Representation type "representation name". When you click this flat button, the representation will be (re)opened.

Domain:

For your information, the domain to which this representation belongs is listed.

General tab:

Object type and name:

Representation type "representation name". When you click this flat button, the representation will be (re)opened.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this representation was last changed.

Object size:

Total size of the representation on disk in bytes (.RPR; in case of a representation class .RPR and .RP#).

Read only:

Select this check box if this representation should be read only. You are then not allowed to edit or delete the representation. Clear this check box if you want to be able to edit the representation.

Description:

Optionally, change the description of this representation. The description of a representation appears in the title bar when the representation is displayed. Further, the description is visible on the status bar of the Main window when moving the mouse pointer over the representation in the Catalog.

Used By tab:

Object type and name:

Representation type "representation name". When you click this flat button, the representation will be (re)opened.

Used by:

In a tree view, it is listed by which other objects this representation is used.

A representation is never a dependent object.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
