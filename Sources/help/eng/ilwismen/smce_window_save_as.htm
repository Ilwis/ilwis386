SMCE window

 File menu

Save a criteria tree as ...

 

Save the current criteria tree with a new name.

This dialog box appears:

  • when you choose Save As from the File menu in a SMCE window;
  • when you have not saved the criteria tree before, and you choose Save from the File menu or click the Save button in the toolbar of the SMCE window.

Dialog box options:

Save in:

If necessary, select another directory or folder where the criteria tree should be stored.

File name:

Type a new name for your criteria tree. It is not necessary to specify the extension.

Save as type:

You cannot select anything else than type Criteria Tree.

 

Note: When you try to save a criteria tree and assign a name that already exists, a question will be asked whether you wish to replace (= overwrite) the existing criteria tree with the current one.
