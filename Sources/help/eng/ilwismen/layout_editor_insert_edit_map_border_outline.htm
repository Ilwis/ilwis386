Insert/Edit Map Border

Neat line

Grid

Graticule

Outline

 

Insert or edit a map border, and/or ticks and/or coordinates for grid lines or a graticule for a selected map view in your layout.

You can insert or edit:

  • a neatline which exactly fits the selected map view;
  • grid ticks and/or coordinates along the map view in meters;
  • graticule ticks and/or coordinates along the map view in degrees, minutes, seconds;
  • coordinates for the corners of a map view;
  • an outline at a certain distance around a map view.

Use the tabs to go to a specific page.

On this Outline page, you can add or edit a border at a certain distance around your map view. To add a border which exactly fits the map view, use the Neat line tab. To obtain a border along the margins of the page of a layout, choose Page Border from the Edit menu.

Outline page options:

Outline:

Select this check box to obtain an outline around the selected map view.

Clear this check box if you do not wish to obtain an outline.

Outline offset (mm):

Type a value for the space in mm between the map view and the outline.

Line type:

Select a line type for the outline: Single, Dot, Dash, Dash dot, Dash dot dot, Blocked or Symbol (see below).

Color:

Select a color for the outline. To define your own color, select the 'Custom' item at the bottom of the list or click the create button; the Windows Color dialog box will appear.

Symbol button:

For line type Symbol: click the Symbol button to select a symbol type (= symbol font), a symbol, a symbol size, etc. You can use simple symbols or a symbol from any installed symbol font. The Line Symbols dialog box will appear.

Support line:

For line types Single, Dot, Dash, Dash dot, Dash dot dot and Symbol; select this check box when you wish to use an extra support line; the outline will then be drawn 'on top' of the support line.

Clear this check box when just the outline should be drawn.

Background color:

Select a color for a support line or select the second color for a blocked outline.

Line width (mm):

For line types Single, Double, Triple and Blocked: type a value for the width of the outline (both on screen and on printer). Line width 0 means: a width of 1 pixel on the screen, and a width of 1 dot on a printer, i.e. as fine as possible (hair line). Values larger than 0 are in mm.

Line types Dot, Dash, Dash dot, Dash dot dot will always be displayed with a width of 1 pixel on the screen, and printed a width of 1 dot on a printer, i.e. as fine as possible (hair line).

Block distance (mm):

For line type Blocked: type a value for the distance between the blocks (mm).

Symbol distance (mm):

For line type Symbol: specify a value for the distance between the symbols.

Examples:

Tips:

  1. If you size a selected map border in the Layout editor, by dragging one of the size handles of a selected map border, then the size or the scale of your map view will change. This depends on whether you used the Keep Scale button.
  2. To store annotation, you need to save the layout. Open the File menu and choose the Save or the Save As command. You can also click in the white space of your layout, and click the Save button in the standard toolbar.
