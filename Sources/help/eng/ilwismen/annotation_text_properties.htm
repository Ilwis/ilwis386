Annotation Text Properties

The Annotation Text Properties sheet lists all kind of important information on a selected annotation text object. In fact, all relevant information that is stored in the object definition file of the annotation text object is displayed. For more information, refer Basic concepts : Properties of objects.

The Annotation Text Properties sheet appears:

  • when you click an annotation text object in a Catalog with the right mouse button, and choose Properties from the context-sensitive menu;
  • when you select an annotation text object in a Catalog, and choose Properties from the Edit menu in the Main window.
  • when you display an annotation text layer, and choose Properties from the File menu in a map window.

General tab:

Object type and name:

Annotation Text "Name". When you click this flat button, the annotation text object will be (re)opened as a table.

Location:

The path and directory where this object is stored.

Object date and time:

Date and time when this annotation text object was last changed.

Object size:

Total size of this annotation text object (.ATX) on disk in bytes.

Read only:

Select this check box if this annotation text object should be read only. You are then not allowed to edit or delete the annotation text layer. Clear this check box if you want to be able to edit the annotation text layer.

Description:

Optionally, change the description of this annotation text object. Furthermore, the description is visible on the status bar of the Main window when moving the mouse pointer over the annotation text in a Catalog.

Used By tab:

Object type and name:

Annotation Text "Name". When you click this flat button, the annotation text object will be (re)opened as a table.

Used By:

In a tree view, it is listed by which map views this annotation text object is used.

Tips:

  • To print or to copy the contents of all Property pages to the clipboard, choose Print or Copy from the Control menu.
  • To print or to copy the contents of the currently active Property tab to the clipboard, click with the right mouse button in the white space of the Property page and choose Print or Copy from the context-sensitive menu.
