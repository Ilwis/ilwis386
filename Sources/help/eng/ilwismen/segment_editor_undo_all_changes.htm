Undo all changes   
 

 

Restore all changes in this segment map since the last Save.

The Undo All Changes command should be used to restore all recent changes. To undelete specific segments which were accidentally deleted before, use the Undelete Segments command on the Edit menu of the segment editor.

When using this Undo All Changes command:

  • all changes you made to the segment map since you last used Save will be restored;
  • if you have not used Save so far, all changes since you started editing the map will be undone.
