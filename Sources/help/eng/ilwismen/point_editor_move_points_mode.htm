Move Points mode   
 

 

Choose Move Points mode to change the position of a point with the mouse.

With the Move Points mode, you can move a point to another position in your map.

When Move Points mode is active:

  • the mouse pointer appears as a pair of tweezers,
  • an option bullet appears in front of this menu command, and
  • the Move Points mode button in the toolbar appears down.

To move a point:

There are three ways to move points to another position:

 

  1. Click the point you wish to move (the class name, ID, or value, etc. of the selected point changes color);
    • then move the mouse pointer to another position;
    • and click at the point's new position.

     

  2. Press and hold down the left mouse button on the point you wish to move (the class name, ID, or value, etc. of the selected point changes color);

    • then move the mouse pointer to another position;
    • and release the left mouse button to drop the point at its new position.

     

  3. Double-click the point you wish to move;

    • then type new coordinates in a dialog box.

 

Tip: You can also move points with the digitizer.
