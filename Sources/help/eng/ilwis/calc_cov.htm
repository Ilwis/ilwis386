COV( ) or COVAR() function

Returns the covariance of the values in ColumnValue a and in ColumnValue b.

Syntax

COV(ValueColumn a, ValueColumn b)
COVAR(ValueColumn a, ValueColumn b)

Input

ValueColumn a is a value column
Domain type: any Value domain
ValueColumn b is another value column
Domain type: any Value domain

Output

COVAR returns: a real value
Domain: system domain Value

Notes:

  • The COV and the COVAR function are aliases of each other.

  • The COV or COVAR function can also be performed through a dialog box, see Column Statistics.
  • When an input value is undefined, the record will be excluded from the calculation.

Tips:

  • Use the COV or COVAR function to determine the relationship between two data sets.
  • When the definition symbol = is used, a dependent output column is created; when the assignment symbol := is used, the dependency link is immediately broken after the output column has been calculated.

Usable in

TabCalc, TabCalc pocket line calculator
