How to create a table

How to create an attribute table

To create a table:

  • open the File menu in the Main window, choose Create Table,
  • in the Operation-tree, open the Create item, and double-click the New Table item, or
  • in the Operation-list, and double-click the New Table item.

The Create Table dialog box appears.

To create an attribute table:

There are several possibilities to create an attribute table:

  1. You can open the properties of a point, segment, polygon or raster map, and select the Attribute check box. Select an existing table (only tables with the same domain as the map are shown) or create a new table by clicking the create button. Properties of maps can only be edited when the map is not read-only.
  2. Another way to create an attribute table is to open the properties of a class domain or an identifier domain and select the Attribute check box. Select an existing table (only tables with this domain are shown) or create a new table by clicking the create button. Properties of domains can only be edited when the domain is not read-only.
  3. When you use the Create Table command from a File menu, then select for the domain of the table the same domain as is used by the map to which the attribute table should refer; this must be a class domain or an identifier domain.
