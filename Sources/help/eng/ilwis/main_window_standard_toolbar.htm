Main window

Standard toolbar

The standard toolbar of the Main window provides shortcuts for some regularly used menu commands for instance to open a new Catalog, to copy and paste objects from one Catalog into another, to delete objects from a Catalog, to view the Properties of an object, and to customize a Catalog.

The actions can be performed on Catalogs showing the contents of a directory as well as on opened map lists and opened object collections.

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating. Some of the actions that are available via the toolbar are also available via the context-sensitive menu.

The following actions can be performed:

 

New Catalog:

Open another Catalog. A dialog box will appear from which you can choose a directory; the selected directory will be opened as a new Catalog.

Open:

Open a raster, polygon, segment or point map in a new map window or open any other ILWIS object. The Open Object dialog box appears; you can select a map or any other object.

Open Pixel Info:

Open a pixel information window. When you selected one or more maps or coordinate systems in the Catalog, these objects will be added to the pixel information window automatically.

Copy:

Copy the selected object(s) to the clipboard. You can then paste them into another Catalog. When you copy one or more maps to the clipboard, you can also paste them into a map window. You can also use the Ctrl+C keys on the keyboard.

Paste:

Paste one or more copied objects into the current Catalog. You can also use the Ctrl+V keys on the keyboard.

Delete:

Delete the selected object(s) from the current Catalog. You can also use the Del key on the keyboard.

Properties:

Open the Properties of a selected object.

Customize Catalog:

Open the Customize Catalog sheet in which you can specify the object types that should be visible in the Catalog (Object selection tab), how the Catalog should look like in Details View (Details View tab), and/or choose or create a query to search for specific objects (Query tab).

List View:

Show the objects in the current Catalog as a list of icons and object names. You can select to view only certain object types, sort or order objects not only on their name, and/or use queries to search for certain objects (see the commands on the View menu).

Details View:

Show the details of the objects in the current Catalog. Object details include whether objects are up-to-date or not, calculated or not, dependent or not, object date, domain type, domain name, expression, description, etc. You can order the objects in ascending or descending order by clicking a column button. You can also drag columns to another position. Furthermore, you can select to view only certain object types, and/or use queries to search for certain objects (see the commands on the View menu).

cd..

Navigate to the parent directory.

The standard toolbar can be hidden or shown with the Standard Toolbar command on the View menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
