How to print a layout

with maps and annotation

Create a layout, do a Page Setup, and add map views and additional annotation according to your wishes, or open an existing layout.

In the layout editor:

 

  • open the File menu, and choose Print, or
  • click the Print button in the standard toolbar of the layout editor.

The Print dialog box will appear.

For information on installing a printer, refer to your Windows documentation.

Tip:

To select a printer, and to specify the paper size, and the orientation of the paper, see Page Setup.
