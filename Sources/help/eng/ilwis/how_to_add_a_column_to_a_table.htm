How to add a column to a table

To add an editable column to a table:

  • From the Columns menu, choose Add Column, or
  • from the Columns menu, choose Column Management; the Column Management dialog box appears; in this dialog box, click the Add Column button, or
  • in the table window, scroll to the last column and double-click in the gray area to the right of the last column header.

The Add Column dialog box appears.

The created column will be an editable source column.

 

Note: Even if you use this column later on as the output column of a calculation, the column remains a source column.

To add a column which contains the answers of a calculation:

  • Type a TabCalc formula on the command line of the table window, or
  • from Columns menu, choose use the Join, Aggregate, Cumulative, Least Squares, Semivariogram or Column Slicing command, and fill out the appearing dialog box of this column operation.

 

The Column Properties dialog box will appear; you can generally accept the defaults.

For more information on table calculation formulae, refer to Table Calculation.

The created column will be:

  • a dependent column when the column is created by a TabCalc formula on the command line in which you used an definition symbol (=),
  • a dependent column when the column contains the result of a join, aggregation, cumulative sum, least squares fit, semi-variogram or column slicing operation (when performed through a dialog box),
  • an editable source column when the column is created by a TabCalc formula in which you used an assignment symbol (:=).
