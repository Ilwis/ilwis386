How to change

the working or startup directory

When you start ILWIS, you find that the Catalog is located in a certain directory. When this is not the directory where your data is, you can change this startup directory as follows.

In ILWIS:

  • Close all Catalogs in the ILWIS Main window;
  • Open the File menu of the ILWIS Main window and choose Preferences;
  • In the Preferences, go to the Directories item, and for Default Startup Dir, type the path where ILWIS should start next time;
  • Click OK in the Preferences;
  • Close and reopen ILWIS.

Mind:

If you close ILWIS while any Catalogs are still open, these Catalogs will be reopened when you close and reopen ILWIS; thus overruling any specified startup directory.
