How to edit a script

A script is a sequenced list of ILWIS commands and expressions. By creating a script, you can build a complete GIS or Remote Sensing analysis for your own research discipline. Scripts are more or less equivalent to batch files in ILWIS version 1.4.

To edit a script:

  • in the Catalog, double-click a script, or
  • select a script in the Catalog, open the Edit menu in the Main window and choose Edit Object, or
  • in the Catalog, click a script with the right mouse button, and select Open from the context-sensitive menu.

The Script editor appears in which you can edit your script expressions.

Tips:

  • For more information on valid script expressions, see Appendices : ILWIS script language (syntax).
  • An example of a script is presented in Map & Table calculation : creating and running scripts.
