Table window

Introduction

A table window displays a table. In a table window, you can view and edit table data, and you can calculate with your table data. Furthermore, parts of the table window are used when a histogram, a cross table, or a two-dimensional table is opened, or when the georeference tiepoints editor or the coordinate system tiepoints editor is opened.

A table consists of columns, records and fields. Schematically, this looks like:

 

Besides the table viewer, in which you can view and edit the table, the table window consists of:

  • a statistics pane, at the bottom of a table window, to view the minimum, maximum, average, standard deviation and sum of values in a value column,
  • a record view (dockable window) in which you can display and edit table data record by record,
  • a command line, on which you can enter your Table calculation formulae,
  • a menu bar, a toolbar, and a status bar.

For an overview of the functionality of a table window, refer to the Table window : functionality topic.
