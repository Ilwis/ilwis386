Introduction

When you are new to Windows

When you are new to Windows you might have to learn some basic Windows terminology and techniques. To acquaint yourself with Windows, you can read some parts of the Windows Help. Open the Windows Help for instance from My Computer or from Windows Explorer, and use the Index tab to find topics on Introduction, Welcome, etc.

The following Windows terms and techniques should be familiar to you:

minimize, maximize, moving windows, sizing windows, title bar, menu bar, using scroll bars, using menus. You can read more about this in the following topics:
