Domain type String

The String domain can store free text. You can only use the String domain for columns in which you want to store for instance descriptions of units, see the example below. Some special TabCalc functions are available which enable you to work with String columns.

The String domain is available as:

  • system domain String.

You cannot create your own string domains and the String domain has no representation.
