Domain type Bit  (deprecated)

The Bit domain can only contain values 1 (True) and 0 (False). You can use the Bit domain only for raster maps. In most cases however it is advised to use the system Bool domain.

The Bit domain is available as:

  • system domain Bit (deprecated).

The Bit domain has no representation. You have to select colors for True values each time you display a map with the Bit domain in the map's Display Options dialog box.
