Domain type Color

The Color domain stores values that are actually colors which can be displayed on monitors configured for 16-bit or 24-bit display.

You can use the Color domain for instance for scanned color photographs (scanned with more than 256 colors) and for 24-bit color composites. The Color domain is also used for interactive color composites. The Color domain is only available for raster maps.
As the colors of a map using the Color domain are defined the values in the map, no representation object is needed or involved.

Note:

You can only use the Color domain when within Windows your monitor is properly configured to use more that 256 colors, for instance High Color 16-bit or True Color for 24-bit (see Display Settings in Windows' Control Panel).
