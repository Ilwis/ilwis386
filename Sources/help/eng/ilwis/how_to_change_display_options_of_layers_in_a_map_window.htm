How to change display options

of layers in a map window

Before a raster, polygon, segment or point map (or grid lines, a graticule, a map list, or an annotation text layer) is displayed in a map window, you have to answer some questions on how this layer should be displayed in the map window: you have to specify the display options. Also, once one or more layers are displayed in a map window, you might want to change the display options of a certain layer.

To change general display options of a layer:

When a raster, polygon, segment, point map (maps with any domain), a map list, grid lines, a graticule, or an annotation text layer is displayed in a map window:

  • open the Layers menu in a map window, choose Display Options, and select a layer from the cascading menu, or
  • click the right mouse button in a map window, choose Display Options from the context-sensitive menu, and select a layer from the cascading menu, or
  • in the Layer Management pane of a map window, double-click a layer.

A Display Options dialog box follows in which you can change the colors of the layer, line width, etc.

To open the representation of a map displayed in a map window:

When a raster, polygon, segment, point map is displayed in a map window and when the domain (class or value) of the map uses a representation:

  • open the Edit menu in a map window, choose Representation, and select the layer of which you wish to change the colors, or
  • in the Layer Management pane of a map window, double-click the word 'Legend'.

Depending on the domain used by the selected map, the representation Class editor, or the representation Value/Gradual editor will be opened. For more information, see also Map window : Edit representation.

Tips:

  • When a map uses a system domain and a system representation, the representation cannot be edited. You can create a new, custom representation for the map through the map's Display Options dialog box.
  • For maps with a Bool domain, you can set the default colors for True and False through the Preferences command on the File menu of the Main window.

To change specific colors of an item in a map with a class domain:

When a raster, polygon, segment or point map is displayed in a map window, and when the map has a class domain:

  • in the Layer Management pane of a map window, double-click a class.

Additional information:

In the Layer Management pane, all layers are preceded by their icon:

raster maps,
polygon maps,
segment maps,
point maps,
map lists,
grid lines,
graticules,
annotation text layers.

Furthermore, in the Layer Management pane, layers are preceded by:

a marked check box when the layer is displayed,

an empty check box when a layer is hidden, and

when the Info check box has been selected in a map's Display Options dialog box.
