How to edit a map list

A map list is a set of raster maps, for example the bands of a satellite image. All raster maps in the map list must have the same domain and georeference.

To open a map list:

  • double-click a map list in a Catalog, or
  • select a map list in a Catalog, and choose Edit Object from the Edit menu in the Main window, or
  • in a Catalog, click the map list with the right mouse button and choose Open from the context-sensitive menu.

The map list will be opened as a kind of Catalog, with some extra possibilities. This is also called a map list viewer.

To add maps to a map list:

  • Click the Add Raster Map button ;

    • A small dialog box follows in which you can select a raster map which must have the same domain and the same georeference as the other maps in the map list;

  • Drag one or more selected raster maps from a Catalog or from Windows Explorer into the opened map list.

To remove maps from a map list:

  • Select one or more raster maps in the map list, and
  • click the Remove Raster Map(s) button , or press the Del key on the keyboard.

To change the order of maps in a map list:

  • Select one or more raster maps in the list, and
  • drag and drop the selection to another position in the list.
