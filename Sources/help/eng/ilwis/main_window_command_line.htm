Main window

Command line

The command line of the Main window is by default located at the top of the Main window, just below the menu bar and the toolbar.

The user may type commands and expresssions on the command line to carry out calculations, operations and scripts.

The command line is used:

  • to perform calculations with maps and attribute tables,
  • to perform ILWIS commands: to display, edit, create or view the properties of ILWIS objects, and to obtain dialog boxes to start an ILWIS operation. This functionality is equivalent to typing an ILWIS 1.4 executable name on the DOS command line in ILWIS version 1.4.
  • to perform ILWIS expressions: to perform complete ILWIS operations. This functionality is equivalent to typing the ILWIS 1.4 executable name and all parameters required by this executable, on the DOS command line in ILWIS version 1.4.
  • to run scripts, which usually contain a sequence of ILWIS expressions. With a script, you can build a complete GIS and Remote Sensing analysis/application for your own research discipline. Scripts are more or less equivalent to batch files in ILWIS version 1.4.
  • to copy and delete objects, break dependency links of objects, etc. For more information, see ILWIS commands and ILWIS script syntax.

If you wish you can move the command line and dock it for instance at the bottom of the Main window; you can also make the command line floating. The command line can be hidden or shown by opening the View menu in the Main window, choose Command line.

Tips:

  • The command line has a history: use the Arrow Up key to retrieve previously used expressions and commands. The Arrow Down key can be used to scroll forwards again. You can also open the list of previously used commands and expressions by clicking the arrow at the right hand side of the command line.
  • The maximum length of an expression on the command line is 32000 characters. You can use the (Ctrl+) Left and Right Arrow keys on the keyboard to move back and forth in your formula. You can use the somewhat larger MapCalc dialog box if you want to type a MapCalc formula on multiple lines.
  • In ILWIS 3, object names comply with Windows long file names. On the command line however, object names must be enclosed in single quotes when they start with a digit, contain a space or any of the following characters ` ~ ! @ # $ ^ & ( ) - + = [ ] { } ; ' , .
    For more information, see How to use long object names.

Other functionality:

?expression

Performs calculations as a pocket line calculator.

For more information, see Appendices : ILWIS commands, section Other commands and expressions.

!command optional_parameter(s)
!!command optional_parameter(s)

 

Starts any Windows application, Windows shortcut, files with a registered file type, a batch file, a DOS application (with a .PIF file available), etc. Type the command directly after the exclamation mark (no spaces allowed).

Example: to start Notepad, type: !Notepad

For more information, see Appendices : ILWIS commands, section Other commands and expressions.
