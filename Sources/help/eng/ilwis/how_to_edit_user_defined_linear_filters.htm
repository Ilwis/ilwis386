How to edit ...

user-defined linear filters

You can define your own linear filters or view the contents of a linear system filter.

To edit a user-defined linear filter:

  • double-click a user-defined linear filter in the Catalog, or
  • use the right mouse button on a filter in the Catalog, and select Open from the context-sensitive menu, or
  • select a filter in a Catalog, and choose Edit Object from the Edit menu in the Main window.

The Filter editor will be opened.

Tip:

  • Linear system filters available in your ILWIS\SYSTEM directory can only be viewed in the Filter editor but not be edited.
