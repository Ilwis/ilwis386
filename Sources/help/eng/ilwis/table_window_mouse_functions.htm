Table window

Mouse functions

Moving the mouse pointer over:

  • Move the mouse pointer over column header:

    • the status bar gives a hint to double-click the column in order to edit the Column Properties. The status bar also shows the column name and its description.

     

  • Move the mouse pointer over a record button:

    • the status bar gives a hint to click the record button and show the record in Record View.

     

  • Move the mouse pointer over a field in the table:

    • the status bar gives information on the column name and the description of the column.

  • The Column Management button : the status bar gives a hint to click the button to open the Column Management dialog box.

Clicking:

  • Click a field: to Edit the field.

    Mind: Fields of dependent columns cannot be edited.

     

  • Click a column header: to select the column.

    • You can for instance use the context-sensitive menu (after selecting the column, click the selection with the right mouse button):

    • to Sort the table on this column in ascending or descending order,
    • to Copy the column into the clipboard, etc.

     

  • Click a record button: to show and optionally edit the selected record in Record View.

    • You can for instance use the context-sensitive menu (after selecting the column, click the selection with the right mouse button):

    • to Edit the fields in the record in a special dialog box,
    • to Copy the record into the clipboard, etc.

     

  • The Column Management button : to open the Column Management dialog box.

Double-clicking:

  • Double-click a column header: to open the Column Properties dialog box.

    • In case of dependent columns created by an earlier TabCalc expression, you can set or change the TabCalc expression to redefine and/or recalculate the column. You can also set or change the column's domain, value range, precision, etc. in this dialog box.

    Furthermore, for any column type, you can change the position of the column in the table, or set or change the width of the column. In case of a value column, you can also set or change the number of decimals shown.

     

  • Double-click to the right of the last column header: to add a new editable column. The Add Column dialog box will appear.

Dragging:

  • Drag the end of a column header to the right or to the left: to increase or to decrease the width of the column. When you are at the right position to start dragging, the mouse pointer changes to a double-sided arrow.

    •  

     

  • Drag a column header to another position in the table: to move the column to another position in the table. While moving the column, the mouse pointer changes into a grayish column icon.

    •  

     

  • Drag the mouse pointer over a number of fields or records: to select these fields or records.

    • Subsequently, you can:

    • edit the selected fields or edit the selected records, see Edit fields in a single column or Edit fields in multiple columns,
    • Copy the selection into the clipboard,
    • Paste table data from clipboard into the selection,
    • Clear or Delete the selection,
    • Print the selection.

    Tip:

    To select multiple columns: click the first column of a sequence of columns, press and hold the Shift key down, and click the last column of a sequence of columns.

Clicking the right mouse button:

  • Click a selection of fields, columns, or records with the right mouse button: to Copy, Paste, Clear or Delete, or Edit the selection (for edit, see Edit fields in a single column or Edit fields in multiple columns).

    •  

  • Click a selected single column with the right mouse button: to Copy, Paste, Delete, or Edit the selection, to Sort the table on this column in ascending or descending order, or to view/edit the properties of the column.
