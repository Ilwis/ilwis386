Map window

Functionality (advanced)

This topic deals with:

  • double-click actions on raster, polygon, segment or point maps (Class or ID domain) in a map window,
  • displaying the raster maps in a map list as a slide show in a map window.

Double-click actions

By double-clicking a mapping unit in a displayed raster, polygon, segment or point map, of which the Info check box in the layer's Display Options dialog box has been selected, you can:

 

  1. View and/or edit attributes of a double-clicked mapping unit:
    • the map should have a Class domain or an ID domain,
    • an attribute table should be linked to the map or to the domain of the map.

     

  2. Change the representation of a mapping unit:
    • the map should have a Class domain.

     

  3. Execute an action, i.e. show another map, a table, a GIF image, a bitmap, play a sound or a video, etc.:
    • the map should have a Class domain or an ID domain,
    • an attribute table should be linked to the map or to the domain of the map,
    • the attribute table should contain a column named Action, this column should use the String domain,
    • in this Action column, you have specified the file names inclusive path and extension of the (ILWIS) map, (ILWIS) table, GIF image, picture, sound or video that should be opened when a certain class or ID unit in the map is double-clicked, and
    • Windows should know which program should be used to open the specified files (registered file types, default 'Open' action).

    Executing an action on a double-click may give spectacular effects.

You must make sure that the Info check box of the layer you wish to use is selected. The Info check box is available in the Display Options dialog box of a layer.

You can select the type of double-click action by choosing the Double-click Action command on the Layers menu in a map window; select Attributes, Representation, or Execute Action from the cascading menu.

Upon a double-click, it is attempted to find the (defined) class name or ID of the unit that was double-clicked. When multiple layers are displayed, all layers will be searched from top to bottom. You can see which class name or ID will be found by pressing the left mouse button in the map window (Info).

Results when using:

  • Attributes: if, upon a double-click, a class name or an identifier is found, and an attribute table is available, then the attributes of this class or identifier will be shown in a separate window. You can also edit the attributes there.
  • Representation: if, upon a double-click, a class name is found, then an Edit Representation Item dialog box will appear in which you can select another color, etc. for this class. Changes will be directly applied in the map window.
  • Execute Action: if, upon a double-click, a class name or identifier is found, and an attribute table is found with a column Action in which, for this class or ID, a filename is found that has a known 'Open' action, then that file is opened (map or table is shown, audio or video is played, etc.).

For more information, see Double-click Action dialog box.

Displaying the raster maps in a map list as a slide show

A map window can show multiple raster maps like a slide show. A slide show is designed to present multi-temporal changes in maps. All maps that you wish to show need to be combined in a map list, then the maps in the map list are displayed one by one in a map window at a user-specified rate.

 

  • Create a map list of the maps that you want to show.
  • Open the map list, for instance:
    • by double-clicking the map list in a Catalog, then click the Open As Slide Show button in the Catalog displaying the contents of the map list, or
    • by using the right mouse button on a map list in a Catalog and choosing Visualization, as Slide Show from the context-sensitive menu, etc.
  • In a slide show, only one set of Display Options will be used; the defaults are based on the first map in the map list. The parameters as set for this map define the display options for all the other maps in the map list.
  • Then, in the Display Options - Map list as slide show dialog box, specify the number of images you want to see per minute in the map window.

For more information, see How to display a map list as a slide show.
