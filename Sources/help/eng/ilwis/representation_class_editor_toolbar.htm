Representation Class editor

Toolbar

The buttons in the toolbar of the Representation Class editor allow you to select a map type. This shows the representation as if this representation was used by that map type.

Every class name has a color. Each pixel, polygon, segment, point with that class name will use that color. Polygons, segments, and points have some other representation characteristcs as well.

 

The following actions can be performed:

Raster Map:

Edit the colors of this representation as used to display a raster map.

Polygon Map:

Edit the colors, hatchings or patterns of this representation as used to display a polygon map.

Segment Map:

Edit the colors, line types, line widths, etc. of this representation as used to display a segment map.

Point Map:

Edit the colors, symbols, symbol sizes, etc. of this representation as used to display a point map.

The toolbar can be hidden or shown with the Toolbar command on the Options menu.
