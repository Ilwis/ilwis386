How to create objects

There are several ways to create an object:

   

  1. From the File menu in the Main window, choose the Create command, and choose the type of object you want to create;
  2. In the Operation-tree, open the Create item, and double-click a New command;
  3. In the Operation-list, double-click a New command;
  4. In dialog boxes, you can click the little create button if available next to a drop-down list box;
  5. Type Create object abbreviation on the command line of the Main window. For more information, see Appendices : ILWIS commands.
