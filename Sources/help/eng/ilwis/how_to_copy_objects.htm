How to copy objects

To copy one or more objects to another directory, you can use one of the methods described below.

 

Mind: It is strongly discouraged to copy, delete or rename files using Windows Explorer or DOS.

Using the Copy Object To dialog box:

To copy a single object to another name or to another directory:

  • select an object in the Catalog, and
  • choose the Copy To command from the Edit menu.

 

The Copy Object To dialog box will appear. You may need to answer some additional Copy questions in case you are copying a dependent map to another directory or when the object to be copied already exists in the destination directory.

When you use the Copy Object To dialog box, you can only copy one object at a time.

Using drag and drop between two Catalogs:

By opening two Catalogs, you can drag and drop multiple objects between the Catalogs to copy the object(s).

 

  1. To open another Catalog:
    • click a directory in the Navigator with the right mouse button, and choose New Catalog from the context-sensitive menu, or
    • click the New Catalog button in the toolbar of the Main window, and select a destination directory; this will be shown in the new Catalog, or
    • click a directory in the current Catalog with the right mouse button, and choose Open from the context-sensitive menu.

     

  2. To copy objects between Catalogs:
    • select one or more objects in the original Catalog,
    • drag the selected object(s) to the destination Catalog,
    • answer additional Copy questions in case you are copying dependent maps or when the objects to be copied already exist in the destination directory.

 

When you use drag and drop, you can copy multiple objects simultaneously.

 

Tip: You can also drag the selected objects to a destination directory listed in the Navigator.

Using copy and paste:

To copy objects you can also use the Copy and Paste commands on a context-sensitive menu or on the Edit menu of the Main window.

 

  1. To Copy:
    • select one or more objects in your original Catalog,
    • either click the selection with the right mouse button, and choose Copy from the context-sensitive menu, or
    • choose Copy from the Edit menu in the Main window, or
    • press the Ctrl+C keys on the keyboard.

     

  2. Optionally, open a destination directory in a new Catalog or in the current Catalog:
    • click a destination directory in the Navigator with the right mouse button, and choose New Catalog from the context-sensitive menu;
      the contents of the selected directory will be shown in a new Catalog, or
    • click a destination directory in the Navigator;
      the contents of the selected directory will be shown in the current Catalog.
  3. To Paste:
    • click a destination directory in the Navigator with the right mouse button, and choose Paste from the context-sensitive menu, or
    • click in the white space of the destination Catalog with the right mouse button, and choose Paste from the context-sensitive menu, or
    • make the destination Catalog active, and choose Paste from the Edit menu in the Main window, or
    • make the destination Catalog active, and press the Ctrl+V keys on the keyboard.

 

When you use Copy and Paste, you can copy multiple objects simultaneously.

Using the command line:

Use the command line in the Main window. For more information, see Appendices : ILWIS script language (syntax).
