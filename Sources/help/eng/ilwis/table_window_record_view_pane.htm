Table window

Record View

Record View displays for a single record, all values, class names, IDs etc. of the fields in that record. It thus allows you to inspect and edit the values of a specific record in your table.

You can move the Record View window and dock it along an edge of a table window; you can also make the Record View window floating.

Figure 1 below shows the 'Laguna Totura' record of the Rainfall table in Record View. At the top, the name of the current record is listed; at the left side, the columns in the table are listed; to the right, you can view and edit the values for this record.

 

Fig. 1: Example of Record View.

To hide or show Record View:

  • Choose Record View from the View menu of the table window.

Record View is shown when a check mark appears in front of the menu command.

Upon closing a table, it is stored whether or not Record View is shown for the current table. This will then be used as the default setting to show or hide Record View when the table is re-opened later on.

To edit values, class names, IDs, etc. in Record View:

  • Click the field you wish to edit in record view.
  • If the field is editable, you can type another value, choose a class name, type an ID, etc.
  • If the field is not editable, it will appear gray.
  • After editing a field:
    • use the Enter key to confirm the value and stop editing, or
    • use the Arrow Down key to confirm the value and move to the next field.

As long as the 'Edit box' is open, you can still press Esc to undo an edit action. After you pressed the Enter key or an Arrow key, the edit action is completed and the change is stored.

Notes:

  • The contents of fields of dependent columns cannot be edited, as these 'values' are the result of a calculation or a column operation.
  • Furthermore, you cannot edit fields of read-only columns nor fields in dependent tables.
  • All changes are directly stored on disk. You CANNOT UNDO a completed edit action.

To show a next or previous record, etc:

  • Click the Next button in the toolbar of the table window, to display the next record in record view,
  • Click the Previous button in the toolbar of the table window, to display the previous record in record view.
  • Similarly, you can use the First Record button , or the Last Record button .
  • Click the Goto Record button , and select a record, to display that record.

Other functionality:

You can copy fields or the whole record to the clipboard.

  • Select the field(s) you wish to copy to clipboard, or
  • click the record name/record number at the top of record view to select the whole record,
  • then, choose Copy from the Edit menu, click the Copy button , press Ctrl+C on the keyboard, etc.

For more information, refer to Table window : Copy.
