Map window

Layer Management

The Layer Management is by default located along the left hand side of a map window. It lists the layers that were added to the current map window. For maps, the Layer Management pane offers a tree structure, to view the properties of the map and the objects used by the map, the legend of a map, etc.

Introduction

In the Layer Management pane, all layers are preceded by their icon:

raster maps,
polygon maps,
segment maps,
point maps,
map lists,
animations
grid lines,
graticules,
3D Toggle
background layer

For maps, the name of the map is displayed as well as the map's description.

Each layer is preceded by a check box which indicates whether the layer is displayed or not:

this layer is displayed in the map window.
this layer is not displayed in the map window.

A map is preceded by a small when, in the map's Display Options dialog box, the Info check box has been selected.

for this map you will obtain Info on a point, segment, polygon or pixel when you press the left mouse button in the map window.

 

For maps that were added to the map window, you will find the properties of the map in the Layer Management pane:

By clicking the + sign in front of the word 'Properties', the Properties will be expanded; you will see the objects that are used by the map.

 

For maps that were added to the map window, you will find the legend of the map in the Layer Management pane:

Depending on the domain used by the map, by default, the legend shows all class names and their colors (hatching, line type, symbol, etc.), or a limited number of values and their colors.

 

You can move the Layer Management pane and dock it along the right hand side or the left hand side of a map window; you can also make the Layer Management pane floating. The Layer Management pane can be hidden or shown with the Layer Management command on the Options menu. Some of the actions that are available via the Layer Management pane are also available via the context-sensitive menu.

Functionality of Layer Management

Layer Management allows the access to the display tools. Display tools are an extensible set of tools that allow you to change the visualization of a layer or the context in which the layers are drawn (e.g. 3D).

Which display tools are active and available is dependent on the context. Below is an example of how it looks like for a plygonmap plus a segment map.

 

Here we see a polygon map with several display tools : Attribute table, Colors, Boundaries, Areas, Areas and Transparency. There are also a set of display tools on a global level : 3D, Grid and Background Area.

Without going into detail. In many cases, double clicking on the tool or if it has sub tools, on the sub tool, will open a dialog box with which you can edit the display properties for that tool. E.g. if you fouble click on the "Line Style"
item you will get a dialog box with which you can change the properties of the line style of the boundaries.

 

By right clicking on the Display Tools item when can add or remove tools for that layer.

 

 

 

For layers that were added to the map window, the Layer Management pane allows you:

  • to show or hide a layer,
  • to change the display options of a layer,
  • to change the order in which layers are drawn,
  • to open the properties of a map,
  • to view the objects used by a map,
  • to add a layer,
  • to remove a layer,
  • to edit a feature map (for maps which are not dependent).

To show or hide layers:

  • Click the or the check box to toggle between showing or hiding a layer.

To change the display options of a layer:

  • Select one of the tools

 

To change the order in which layers are drawn:

  • Select a layer in the Layer Management pane and drag it to another position in the list.

The layer which appears at the top of the list in Layer Management will be drawn last in the map window (=on top).

To open the properties of a map:

  • Double-click the word 'Properties' of a map in the Layer Management pane.
  • Click a layer with the right mouse button in the Layer Management pane, and choose Properties from the context-sensitive menu.

A Properties sheet will be opened.

To view the objects used by a map:

  • Expand the Properties of a map in the Layer Management pane by clicking the + button in front of the word 'Properties'.

You can double-click the objects to open and/or edit them.

To open and/or edit the legend of a map:

  • Double-click the word 'Legend' of a map in the Layer Management pane.

Depending on the domain used by the map, the representation Class editor or the representation Value/Gradual editor will be opened.

 

Mind: When a map uses a system domain and a system representation, you cannot edit the map's representation. You can create a new, custom representation in the map's Display Options dialog box.

To edit a legend item (for maps with a class domain):

  • Double-click a class or a color listed in the Legend of a map in the Layer Management pane.

To add other maps to a map window:

To add an existing raster, polygon, segment, or point map to a map window, or to add a map list, or an annotation text layer:

  • Click with the right mouse button in the white space of the Layer Management pane, and choose Add Layer from the context-sensitive menu.

The Add Layer dialog box will appear.

You can also use the Add Layer button in toolbar of the map window; press the Ins key on the keyboard; or drag and drop multiple maps from a Catalog or from Windows Explorer into a map window.

To add grid lines or a graticule:

To add grid lines or a graticule to a map window:

  • Click with the right mouse button in the white space of the Layer Management pane, and choose Add Grid Lines or Add Graticule from the context-sensitive menu.

The Display Options - Grid Lines dialog box or the Display Options - Graticule dialog box will appear.

To remove a layer:

  • Click with the right mouse button on a layer in the Layer Management pane, and choose Remove Layer from the context-sensitive menu.

You can also select a layer in the Layer Management pane and use the Remove Layer button in toolbar of the map window.

To edit a map (for maps which are not dependent):

  • Click a map with the right mouse button in the Layer Management pane, and choose Edit Layer from the context-sensitive menu.

For more information, refer to Map window : Edit layer.
