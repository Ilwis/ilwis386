How to delete objects

Selecting objects in a Catalog and deleting them:

To delete one or more objects:

  • select one or more objects in a Catalog, and
  • choose the Delete command from the context-sensitive menu or from Edit menu, or press the Del key on the keyboard.

Using the command line:

Use the command line in the Main window. For more information, see Appendices : ILWIS script language (syntax).
