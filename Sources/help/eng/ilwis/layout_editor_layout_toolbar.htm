Layout editor

Layout toolbar

The Layout editor has two toolbars that provide shortcuts for some regularly used menu commands:

 

  • a Standard toolbar: for standard actions like zooming in and out, save and print;
  • a Layout toolbar: to insert annotation to a layout, to add or to remove a layer from a selected map view in the layout, to add grid lines or graticules to a map view, etc.

 

In this topic, the Layout toolbar is described.

 

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating. To add annotation to a layout, you can also use the Insert menu.

The following actions can be performed:

 

Map View:

Insert an existing map view into the layout.

The Insert Map View dialog box will appear.

Text:

Insert new text into the layout.

The Insert/Edit Text dialog box will appear.

Box:

Insert a new box into the layout.

The Insert/Edit Box dialog will appear.

Bitmap/Picture:

Insert an existing bitmap or picture (Windows metafile) from disk into the layout.

The Insert Bitmap or Picture dialog box will appear.

 

Legend:

For a selected map view in the layout: insert a legend for a certain layer in the map view into the layout.

When the selected map view contains multiple layers, first the Map Layer Selection dialog box will appear.

Depending on the domain and the type of the layer for which you wish to insert a legend, you will need to specify display options for the legend for instance in:

  • the Display Options - Legend (Class) dialog box when you add a legend of a class map;
  • the Display Options - Legend (Image/Value) dialog box follows when you add a legend of a value map.

For other types of legends, refer to Insert Legend / Map Layer Selection.

Map Border:

For a selected map view in the layout: insert a map border into the layout. With this option you can also add ticks and coordinates of grid lines and/or of a graticule available in the selected map view or coordinates of the corner of the map(s) in the map view. Furthermore, you can add an outline around the map view.

The Insert/Edit Map Border dialog box will appear.

North Arrow:

For a selected map view in the layout: insert a North arrow to the layout.

The Insert/Edit North Arrow dialog box will appear.

Scale Bar:

For a selected map view in the layout: insert a scale bar into the layout. The scale bar will reflect the scale on which the selected map view is displayed and will be printed later on.

The Insert/Edit Scale Bar dialog box will appear.

Scale Text:

For a selected map view in the layout: insert as text the exact scale figure on which the map view is displayed and will be printed later on.

The Insert/Edit Scale Text dialog box will appear.

 

Add Layer:

For a selected map view in the layout:

  • add a raster map to the current map view when the map view does not contain a raster map yet, or
  • add another polygon, segment or point map to the current map view, or
  • add grid lines, a graticule or an annotation text object to the current map view.

The Add Layer dialog box will appear and the layout editor will obtain a Layer Management pane like a map window. You can thus edit display options of the layers in the map view.

After you have added a layer, you need to redraw the map view:

  • select the map view and click the Redraw button in the standard toolbar, or
  • click the map view with the right mouse button and choose Redrawfrom the context-sensitive menu.

To save the changes in the map view, see the tip below.

Remove Layer:

When you already double-clicked a map view in the layout editor and when the layout editor has a Layer Management pane: select a layer in the Layer Management pane and click this button to remove the layer from the map view.

Click the Redraw button in the standard toolbar to view the grid lines in the map view. To save the changes in the map view, see the tip below.

Grid Lines:

For a selected map view in the layout: add grid lines to the map view.

The Display Options - Grid Lines dialog box will appear.

Click the Redraw button in the standard toolbar to view the grid lines in the map view. To save the changes in the map view, see the tip below.

Graticule:

For a selected map view in the layout: add a graticule to the map view.

The Display Options - Graticule dialog box will appear.

Click the Redraw button in the standard toolbar to view the graticule in the map view. To save the changes in the map view, see the tip below.

 

Tip:

To save a map view to which you added (or from which you removed) extra layers, grid lines or a graticule, or when you changed display options of a layer in the map view:

  • select the map view, and click the Save button in the standard toolbar, or
  • click the selected map view with the right mouse button, and choose Save View or Save View As from the context-sensitive menu.

The toolbar can be hidden or shown with the Layout Toolbar command on the View menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
