Properties of an object

The properties of an object describe the object and its relations to other objects.

The Property sheet of an object lists:

  • object type, object name, date and time of last change, read-only or not, total size of object on disk, object description,
  • objects used by the current object, for instance domain, georeference, coordinate system,
  • for maps and domains: whether an attribute table is linked to the map or to the domain and the name of the attribute table,
  • additional information on the object such as pixel size, number of lines and columns (raster map), number of segments (segment maps), etc.,
  • for dependent data objects: the object definition.

Furthermore, Property sheets are designed to handle the properties of dependent data objects, e.g. Make up-to-date, Edit Definition, Release Disk Space, etc. For more information, refer to Basic concepts : properties of objects. To view the properties of an object, refer to How to view and edit the properties of an object.
