Layout editor

In the Layout editor you can edit a layout. A layout is a virtual sheet of paper on which you can make a composition of one or more map views and annotation before printing.

In the layout editor, you can add to a layout:

  • one or more map views to be printed on a certain scale; each map view may include one raster map and/or one or more point maps, segment maps and/or polygon maps, annotation text layers, grid lines and/or graticules;
  • texts such a titles;
  • boxes;
  • pictures and/or bitmaps;
  • a page border.

For each map view in a layout, you can furthermore add:

  • a legend for a layer in a map view;
  • a map border, including coordinates and ticks for grid lines and/or a graticule;
  • a North arrow;
  • a scale bar and scale text.

You can also add or delete layers from a map view, or edit the grid lines or graticule(s) in a map view.

Finally, any picture or bitmap that was copied into the clipboard from another Windows application program can be pasted into a layout.

 

Tip: Multiple texts stored in an Annotation Text object can be edited in the Annotation Text editor.
