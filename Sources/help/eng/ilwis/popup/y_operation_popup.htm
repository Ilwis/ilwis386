Operations

An operation performs a certain task on input data, and stores the results in an output map or table. Contour Interpolation, Cross, Distance Calculation and Filter are examples of operations.

You can start an operation by selecting an operation from the Operation-tree or the Operation-list, from the Operations menu, by clicking an input object in the Catalog with the right mouse button and choosing an operation from the context-sensitive menu, by dragging an input object from the Catalog to an operation in the Operation-list, or by typing an ILWIS command on the command line of the Main window. Subsequently, fill out the appearing dialog box of that operation and click the OK button; this generates an ILWIS expression on the command line of the Main window.

Advanced users can type such complete ILWIS expressions on the command line of the Main window or create a script to execute a series of expressions.
