Representation Class editor

The Representation Class editor allows you to change the representation of any map with a class domain, a group domain, or a picture domain.

For each class as present in the domain to which this representation is linked, you can select or create a color. You can also assign a range of colors to multiple selected classes.

Furthermore, you can choose a hatching type or select or create a pattern for polygon classes; select a line type, line width, etc. for segment classes; and a symbol type, symbol size, etc. for point classes.
