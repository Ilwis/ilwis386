Graph window

A graph window displays one or more graphs.

From columns in a table, you can create a so-called graph from columns (displayed as points, bar, line, step, needle) or a rose diagram.

After you displayed a graph from columns, you can add graphs from formula, a least squares fit, or a semi-variogram model.

A graph window consists of a viewer which displays your graph(s), a Graph Management pane, menu bar, a toolbar and a status bar. Furthermore, a context-sensitive menu is available within the Graph Management pane.

For more information, refer to Graph window : functionality and Graph window : examples of graphs.
