Size handles

Size handles appear as 4 or 8 little black boxes at the edges of a selected annotation element. Size handles are used to change the size of an object.

To change the size of a text, legend, box, scale bar, North arrow, bitmap or picture in the Layout editor:

  • click the object that you wish to resize,
  • position the mouse pointer at one of the 4 or 8 size handles: the mouse pointer changes into a double-sided arrow,
  • drag the size handle to a new position.

Dragging one of the corner handles maintains the original height-width ratio (isotropic).

Dragging one of the middle handles resizes the annotation horizontally or vertically; the height-width ratio changes (anisotropic).
