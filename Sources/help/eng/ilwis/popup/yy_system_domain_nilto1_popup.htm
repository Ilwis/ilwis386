System domain NILto1

  • designed for use with Spatial Multi Criteria Evaluation (SMCE),
  • fixed value range from 0.00 to 1.00, and default precision 0.01,
  • during calculations or other operations, you may change the precision for the output object, but not the value range,
  • undefined values are allowed,
  • domain type: Value,
  • default representation: NILto1.
