Drag

By dragging you can for instance change the size a window, change the width of a column, and change the size of annotation objects in the Layout editor.

  • click an object to select it,
    • for windows: position the mouse pointer at the border of a window,
    • for columns: position the mouse pointer in between 2 column names,
    • for annotation: position the mouse pointer at one of the 4 or 8 size handles,
  • the mouse pointer usually changes into a double-headed arrow when you are at the right position to start dragging,
  • press the left mouse button, and hold the left mouse button down,
  • move the mouse pointer to a new position,
  • then, release the left mouse button.
