Table calculation

Table calculation is used to perform calculations with columns in a table. You can perform calculation on columns in attribute tables, but you can also perform calculations on columns of histograms and on columns of tables that are the result of an operation.

Type table calculation formulae on the command line of a table window. For complicated table calculations such as aggregations, join, cumulative values and least square fits, you can use the Columns menu of the table window.

The following list gives an overview of the functionalities:

  • column and record calculations using arithmetical, relational, logical, conditional, exponential, logarithmic and other operators and functions,
  • creation, storing and application of user-defined functions,
  • aggregation of values by using a group column,
  • calculation with columns of another table on disk (join),
  • least squares fit,
  • retrieval of data from raster maps based on coordinates columns in the table.
