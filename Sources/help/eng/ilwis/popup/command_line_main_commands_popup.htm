ILWIS commands on the command line

By using an ILWIS command on the command line, a dialog box will be opened and generally some user-interaction is required before an action can be continued.

You can for instance obtain dialog boxes to display, edit, create or view the properties of ILWIS objects, and dialog boxes to start an ILWIS operation.

When using an ILWIS command, you always have to fill out the entries in the appearing dialog box and click its OK button. For more information, see Appendices : ILWIS commands.
