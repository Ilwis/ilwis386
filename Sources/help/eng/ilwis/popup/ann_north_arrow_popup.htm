North arrow   

When one or more maps and optional grid lines, graticules and/or annotation text layers in a map window are saved as a map view, and when the map view is added to a layout, you can add a North arrow for each map view in the layout using the layout editor.

The North is determined from the georeference of the maps in the selected map view. The North is defined as the positive Y-direction.

In the Insert/Edit North Arrow dialog box, you can choose between eight types of North arrows. Furthermore, you can specify whether you want to obtain text(s) like N, E, S, W, etc.
