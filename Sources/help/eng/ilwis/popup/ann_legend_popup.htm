Legend   

In a map window, the legends of maps that were added to the map window are by default shown in the Layer Management pane of the map window.

To be able to print maps with legends, you need to save one or more maps and optional grid lines, graticules and/or annotation text layers in a map window as a map view. Then, you should create a layout to which you can add your map views and additional annotation.

In the layout editor, you can add a custom legend for each layer in a map view.

Legends are more or less automatically created from the domain of a map in a map view and from the representation used by that domain.

  • For a class map, you will get a legend consisting of color boxes and class names for selected items in the class domain.

    • For more information, see Display Options - Legend (Class) dialog box.

  • For an image or a value map, you will get a legend consisting of a vertical color bar with values along a side of the bar.

    • For more information, see Display Options - Legend (Image/Value) dialog box.

  • For other legend types, see Insert a legend.
