Sample statistics

The following sample statistics are calculated per band, for training pixels in a selected class:

  • mean,
  • standard deviation,
  • number of training pixels with the predominant value,
  • predominant value,
  • total number of training pixels of this class.

The same is calculated for the current set of selected pixels; this is displayed in the Lower half of the Sample Statistics window.

The first map in the map list is represented by Band 1, the second by Band 2, etc. Based on the sample statistics, you may decide to sample more training pixels of a certain class, merge classes, delete classes, etc.
