Table window

A table is displayed in a table window. In a table window, you can view and edit table data, and you can calculate with your table data.

A table window consists of:

  • a table viewer which displays the table data,
  • a command line on which you can type your table calculation statements,
  • a statistics pane, at the bottom of a table window, to view the minimum, maximum, average, standard deviation and sum of values in a value column,
  • a record view (dockable window) in which you can display table data record by record,

Furthermore, a menu bar, a toolbar, a status bar, and a context-sensitive menu are available.

Furthermore, parts of the table window are also used when a histogram, a cross table, or a two-dimensional table is opened, or when the georeference tiepoints editor or the coordinate system tiepoints editor is opened.
