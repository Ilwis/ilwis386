Context-sensitive menu Catalog

The context-sensitive menu that appears when clicking the right mouse button on an object in the Catalog offers:

  • short cuts to open or edit the selected object,
  • for data objects: a list of operations appears that can be performed on the selected object,
  • for service objects: short cuts to create other objects that should use the selected object,
  • for map lists, filters and scripts: a possibility to open the object in another way than normal,
  • a short cut to view the properties of the object,
  • short cuts to copy or delete the object.

When you use the right mouse button in the white space between objects in a Catalog, you can:

  • switch between List View and Details View,
  • paste the object(s) that were copied to clipboard into this Catalog,
  • specify the object types that should be visible and/or the sorting method of the objects,
  • open the Customize Catalog sheet.
