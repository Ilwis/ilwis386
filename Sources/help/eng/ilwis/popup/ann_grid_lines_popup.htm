Grid lines   

Grid lines can be drawn in a map window as lines which are parallel to the X- and Y-axes of the coordinate system used by the map window. You are allowed to add multiple grid lines to one map window.

When a map window displays:

  • a North-oriented map (e.g. raster map with a georef corners), the grid lines will be straight and exactly horizontal and vertical;
  • a raster map which has a georef tiepoints with a conformal or an affine transformation, the grid lines will be straight but not exactly horizontal and vertical;
  • a raster map which has a georef tiepoints with a higher order transformation, or a georef direct linear, georef orthophoto, the grid lines will be curved.

In the Display Options - Grid Lines dialog box, you can specify the spacing of the grid lines in meters. Furthermore, you can select a color for the grid lines.

You can save the map window as a map view; then you can add the map view to a layout.

In the layout editor, you can add ticks and XY-coordinates through the Map Border button in the layout toolbar, or through the Map Border command on the Insert menu of the layout editor.
