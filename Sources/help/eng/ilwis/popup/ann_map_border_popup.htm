Map Border   

When one or more maps and optional grid lines, graticules and/or annotation text layers in a map window are saved as a map view, and when the map view is added to a layout, you specify what should appear along the borders of the map view in the layout.

In the Insert/Edit Map Border sheet in the Layout editor, you can choose to add:

  • a neatline around the map view as a thin single line or as an alternating black and white pattern;
  • ticks for grid lines and optionally coordinates for the grid line ticks;
  • ticks for a graticule and optionally coordinates for the graticule ticks;
  • coordinates for the corners of the map(s) in the map view;
  • an extra outline around the map view.

When you move a selected map border to another position, both the map view and its border will be moved.

Tips:

  • To obtain a sizeable box around text or around a legend in a layout, and use the Box button or the Box command on the Insert menu.
  • To obtain a page border, choose the Page Border command on the Edit menu.
