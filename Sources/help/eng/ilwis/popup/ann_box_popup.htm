Boxes   

When one or more maps and optional grid lines, graticules and/or annotation text layers in a map window are saved as a map view, and when the map view is added to a layout, you can add one or more boxes to the layout.

Boxes can for instance be used to obtain borders around a legend or a title, etc.

In the layout editor, in the Insert/Edit Box dialog box, you can specify the line type and a color for the borders of the box. When you select line type Single, you can also specify the line width. By default, the inside of a box is transparent, but you can also fill the box with a color.

You can size a box by dragging one of the size handles of the box in the Layout editor.

Tips:

  • To obtain borders around a map view, select a map view in your layout and use the Map Border button or the Map Border command on the Insert menu; use the Neatline tab or the Outline tab.
  • To obtain a page border, choose the Page Border command on the Edit menu.
