Windows metafile / Picture   

A Windows metafile or picture is a standard Windows graphic file format (.WMF, .EMF) in 'vector' form.

A metafile consists of a set of Windows drawing instructions. The default extension for Windows metafiles is .WMF; the extension for enhanced Windows metafiles is .EMF.

Furthermore, the metafile format is one of the formats often used by the clipboard, i.e. when using Cut, Copy, Paste for graphics.

You can add one or more pictures, either from file or from clipboard, to a layout.
