System objects

System objects are the standard domains, standard representations, standard coordinate systems and standard filters that are installed during the setup of ILWIS.

Examples are for instance: system domain Value, system domain Image; the standard Gray and Pseudo representations; the standard LatLonWGS84 coordinate system; and the standard filters AVG3x3, DFDX, and Laplace.

System objects cannot be edited. System objects are by default located in your ILWIS\SYSTEM directory.
