Coordinate boundaries of the map

The coordinate boundaries of a map are the minimum and maximum XY-coordinates of the map.

  • With the minimum XY-coordinate, the XY-coordinate of the lower left corner of the map is meant;
  • with the maximum XY-coordinate, the XY-coordinate of the upper right corner of the map is meant.

The four boundary values determine the outermost coordinate values of a map.

This means that you can only add points or segments to the map within the map's coordinate boundaries.

  • However, while editing a point map or a segment map, you can change the map's coordinate boundaries: choose Boundaries of Map from the File menu of the point editor, or Boundaries of Map from the File menu of the segment editor.
  • You can furthermore change the coordinate boundaries of vector maps with the following operations: Sub-map of point map, Sub-map of segment map, Glue point maps, Glue segment maps.
