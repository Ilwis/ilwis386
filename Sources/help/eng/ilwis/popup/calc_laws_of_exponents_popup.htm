Laws of exponents

Negative exponents:

a-n = 1/an

thus a-3 = 1/a3

Fractional exponents:

am/n =nÖ(am)

thus a2/3 = 3Öa2

Multiplication:

anam = an+m

thus a3a2 = a3+2 = a5

Division:

an/am = an-m

thus a3/a2 = a3-2 = a1

Raising to a power:

(an)m = anm

thus (a3)2 = a3*2 = a6

Special cases

a1 returns a

a0 returns 1

1n returns 1

00 returns undefined
