Data layers

Maps that are displayed in a map window are called data layers.

In one map window, you can display one or more raster maps combined with one or more polygon maps, segment maps and/or point maps. Also three images of a map lists can be displayed in a map window (as an interactive color composite), and all maps in a map list can be shown as a slide show.

Other types of layers that can be added to a map window are: Annotation Text objects, which usually contain the labels of a point, segment or polygon map; grid lines and graticules.

You can save a map window as a map view and then add one or more map views to a layout. To a layout, you can add further annotation such as legends, map borders including ticks and coordinates for grid lines or a graticule, north arrows, scale bars and scale texts, texts (such as a title), boxes, bitmaps and pictures, and a page border.

 

Mind: In a layout, a map view may only contain one raster map.
