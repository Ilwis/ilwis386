System domain Value

  • designed for general use,
  • default value range from -9999999.9 to 9999999.9 (i.e. from almost -10 million to almost 10 million), and default precision 0.1,
  • during any operation or any calculation with MapCalc or TabCalc, you can adjust the value range and precision for the output map or column, so that they fit to the expected range of output values that you want to store for your output data; in most cases ILWIS will come up with a proper default,
  • undefined values are allowed,
  • domain type: Value,
  • default representation: Pseudo.

For the system domain Value, you can create a representation gradual but also a representation value.
