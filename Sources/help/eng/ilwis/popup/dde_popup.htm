DDE

By programming an application in Visual Basic, Delphi, or Java, etc. you can:

  • access and execute ILWIS commands, ILWIS expressions, ILWIS script commands,
  • retrieve or calculate simple information on maps and tables as single values, as using the ILWIS pocket line calculator,
  • retrieve coordinate(s) or values at a mouse click in a map window as using Pixel Information.

The process of exchanging data between two Windows programs is called Dynamic Data Exchange (DDE). A DDE conversation takes place between your own application (DDE client) and ILWIS (DDE server).

For more information, see Appendices : ILWIS as DDE server.
