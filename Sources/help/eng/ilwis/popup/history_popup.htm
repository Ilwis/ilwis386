History

The command line of the Main window and the command line of each table window have a history: previously used commands and expressions can be retrieved.

You can use the history for instance to repeat a certain calculation or expression and change input and output map names, input and output column names, parameters, etc.

To open the history of a command line:

  • click the arrow at the right hand side of the command line,
  • then, select a command or expression from the list.

You can also use the Arrow Up and the Arrow Down keys on the keyboard:

  • activate the command line;
  • press the Arrow Up key on the keyboard to retrieve previously used commands and expressions;
  • press the Arrow Down key to 'scroll' forward again.

The amount of commands and expressions that can be retrieved can be set in the Preferences of the Main window. The maximum number of commands or expressions that can be retrieved is 30. The commands and expressions are stored in the registry.
