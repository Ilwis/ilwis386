Representation Value/Gradual editor

The Representation Value/Gradual editor allows you to change the representation of any map with a value domain.

In general, in the representation Value/Gradual editor, you can:

  • insert, change and delete boundary values,
  • assign or change the color of a boundary value,
  • assign or change the coloring of the values in between the boundary values.

Colors in between boundary values are either stretched, this means gradual colors in a number of steps (you can increase or decrease the number of steps used to stretch), or are the same color as the boundary above or the boundary below.

For ILWIS 1.4 users:

Using the representation Value/Gradual editor gives the same results on the screen as applying a classify table and preparing a color lookup table for an output map, but the original values of the input map are retained.
