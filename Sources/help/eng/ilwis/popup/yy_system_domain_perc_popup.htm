System domain Perc

  • designed to calculate with percentages, for example in histograms,
  • default value range from 0.00 to 100.00, and default precision 0.01,
  • during calculations or other operations, you may change the precision, but it is not advised to change the value range,
  • undefined values are allowed,
  • fixed unit '%',
  • domain type: Value,
  • default representation: Gray.
