Map list   

A map list is a set of raster maps, for example the bands of a satellite image. All raster maps in the map list must have the same georeference and the same domain.

A map list is used:

  • for sampling and classification,
  • to display three maps in a map list as an interactive color composite,
  • to present multi-temporal changes in maps as a slide show,
  • to perform a Principal Components Analysis,
  • to apply the same MapCalc formula on all raster maps in the map list, or
  • to perform an operation on all raster maps in the map list.

A map list can be created from the File menu in the Main window or while starting an operation which requires a map list as input. You can include as many raster maps in a map list as you like.
