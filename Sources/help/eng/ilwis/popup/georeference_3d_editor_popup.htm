Georeference 3D editor

With the Georeference 3D editor, you can edit a georeference 3D and obtain a 3D view of your study area according to your wishes. To create a new georeference 3D, you need first of all an interpolated height map (Digital Terrain Model/Digitial Elevation Model), see also How to calculate a Digital Elevation Model.

To create a georeference 3D and to edit it, refer to Georeference 3D editor : functionality or to How to display Digital Elevation Models.
