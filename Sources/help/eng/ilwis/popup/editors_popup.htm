Editors

Pixel editor:

To edit class names, IDs, or values of pixels in a raster map.

Polygon editor:

To edit class names, IDs, or values of polygons,

to create labels from polygons, or apply labels to polygons, and

to extract segments.

Segment editor:

To add new segments or delete existing ones,

to edit class names, IDs, or values of segments,

to change the shape of segments,

to split and merge segments, and

to check segments and polygonize segments.

Point editor:

To insert new points and delete existing ones,

to edit class names, IDs, or values of points in a point map,

to move points to a new position.

Other editors are: Domain Class/ID editor, Representation Class editor, Representation Value editor, Georeference Tiepoints editor, Georeference 3D editor, Coordinate System Tiepoints editor, Sample Set editor, Filter editor, Function editor, Script editor, Annotation Text editor, Layout editor.

For more information, see Editors.
