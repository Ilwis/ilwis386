Domain Class/ID editor

The domain Class/ID editor allows you to add and edit names, codes and descriptions of classes in a class domain and of IDs in an ID domain. You can also add and edit upper boundary values, names, codes and descriptions of groups in a group domain.

Furthermore, you can set the sorting order of a class or ID domain, use Copy and Paste to add certain domain items of another domain into the current one, merge all items of another domain into the current one, change the prefix of IDs in an ID domain when available, delete domain items, and print all or selected items of a domain.
