Graticule   

A graticule draws parallels and meridians in a map window; the maps in the map window require a coordinate system of the type projection which contains projection information. You are allowed to add multiple graticules to one map window.

In the Display Options - Graticule dialog box, you can specify the spacing of the graticule in degrees, minutes and seconds. Furthermore, you can specify a color for the graticule.

You can save the map window as a map view; then you can add the map view to a layout.

In the layout editor, you can add ticks and coordinates (Latitudes/Longitudes) through the Map Border button in the layout toolbar, or through the Map Border command on the Insert menu of the layout editor.
