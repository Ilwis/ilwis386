System value domains

The system domains that can contain values are:

  • system domain Value,
  • system domain Distance,
  • system domain Count,
  • system domain Min1to1,
  • system domain NILto1,
  • system domain Perc,
  • system domain Byte,
  • system domain Image,
  • system domain NOAA,
  • system domain Radar.

For Boolean values, the following system domains are available:

  • system domain Bool, and
  • system domain Yes/No.
