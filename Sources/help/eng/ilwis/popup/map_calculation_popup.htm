Map calculation

Map calculation is used for the execution of most spatial analysis functions and modelling operations. It integrates spatial and tabular data. MapCalc enables you to perform queries and overlays on raster maps.

  • You can directly type map calculation formulae on the command line of the Main window;
  • You can also double-click the MapCalc item in the Operation-list or choose the Map Calculate command on the Operations menu in the Main window; then, type a formula in the appearing Map Calculation dialog box.

The following calculations can be performed:

  • manipulation of one or more raster maps using arithmetical, relational, logical, conditional, exponential, logarithmic and other operators and functions,
  • creation of attribute maps from raster maps that have an attribute table,
  • classification of value maps according to a domain group,
  • integration of two raster maps according to two-dimensional tables,
  • application of user-defined functions.
