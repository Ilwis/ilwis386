(Attribute) Table   

A table consists of records and columns. A table is an attribute table when the table stores additional information on elements in a map; i.e. extra tabular data which relates to mapping units, points, segments, or polygons in maps. To create an attribute table, see How to create an attribute table.

Raster, polygon, segment and point maps which use a domain type Class or ID can have attribute tables. The domain of the attribute table should be the same as the domain of the map to which it relates. An attribute table can be linked to a map or to a domain through the Properties of a map or a domain.

An attribute table can be edited when the table is displayed in a table window. When an attribute table is linked to a map or to a domain, and the map is displayed in a map window, you can also double-click the units in the map (see Map window : advanced functionality).
