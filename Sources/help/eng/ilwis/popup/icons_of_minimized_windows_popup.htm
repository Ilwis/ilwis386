Icons of minimized ILWIS windows

Windows can be reduced to an icon on the desktop if you want the window or application available for later use. To reduce a window to an icon click the minimize button, or open the Control menu and choose the command Minimize. An icon can be restored to a window by double-clicking it, or click the icon to open the Control menu and choose the command Restore.
