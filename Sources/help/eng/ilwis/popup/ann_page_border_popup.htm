Layout Page Border

When one or more maps and optional grid lines, graticules and/or annotation text layers in a map window are saved as a map view, and when the map view is added to a layout, you can add a page border along the margins of the layout.

In the layout editor, choose Page Border from the Edit menu.

  • When the Page Border command is active and marked with a check mark, the Page Border will be drawn in black and it will be printed.
  • When the Page Border command is not active, the paper margins will be drawn in gray for orientation purposes and the border will not be printed.

The Page Border automatically fits the paper margins as specified during Page Setup.

Tips:

  • To obtain borders around a map view, select a map view in your layout and use the Map Border button or the Map Border command on the Insert menu; use the Neatline tab and/or the Outline tab.
  • To obtain a sizeable box around text or around a legend in a layout, select a map view in your layout, and use the Box button or the Box command on the Insert menu.
