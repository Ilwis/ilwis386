Histogram of a polygon map   

A histogram is a special object which lists frequency information on values, classes or IDs in a map. A polygon histogram lists the number of polygons and the total perimeter and total area of polygons per class, ID, or value in the map. If the input polygon map uses a value domain, also the cumulative number of polygons, cumulative perimeters and cumulative areas are calculated.

In a histogram window, the histogram is presented as a table and as a graph.

A polygon histogram is obtained by performing the Histogram operation on a polygon map. For polygon maps that use a value domain and of which no histogram is available yet, a histogram is automatically calculated just before such a value map is displayed.

With the Histogram operation, you can also calculate histograms of raster, segment and point maps.
