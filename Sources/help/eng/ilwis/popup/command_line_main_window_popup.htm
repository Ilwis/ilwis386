Command line in Main window

The command line of the Main window is by default located at the top of the Main window, just below the menu bar and the toolbar.

The command line is used:

  • to type ILWIS commands to display, edit, create an object or view the properties of an object, and to obtain dialog boxes of ILWIS operations,
  • to type ILWIS expressions to perform map calculations and to perform complete ILWIS operations,
  • to run scripts which contain a sequence of ILWIS expressions.

The command line can also be used as a pocket line calculator.
