Windows bitmap   

A bitmap is a standard Windows graphic file format (.BMP) in 'raster' form.

In the layout editor, you can also save or export a complete layout (including map views and annotation) as a bitmap.

Bitmaps are stored in device independent bitmap (dib) format, i.e. bitmaps are not dependent on the graphic board. Bitmaps can use a 2, 16 or 256 color palette, or 24-bit colors. The default file extension for bitmaps is .BMP.

Furthermore, the bitmap format is one of the formats that is used very often by the clipboard, i.e. when using Cut, Copy, Paste for graphics.

You can add one or more pictures, either from file or from clipboard, to a layout.
