Display options

Display options are specified in a Display Options dialog box; this tells the system how a certain layer should be displayed for instance in a map window.

Display options for maps (data layers) mainly include: colors, line widths of segments and polygons boundaries, size of points, and, in case of images or map with a value domain, the stretch range.

Display Options have to be specified for each new layer that you want to display, but you can also change the display options of layers that are already displayed. For more information on the latter, see How to change display options of layers in a map window.
