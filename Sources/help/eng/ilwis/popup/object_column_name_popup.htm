Column name

In ILWIS 3, object names and column names comply with Windows long file names.

  • In object names and column names, any character may be used except \ / : * ? < > | "
  • The maximum length of an object name is 255 characters including the full path.

Furthermore, it is not allowed to use % in any ILWIS object name or column name.

Long object names and column names can be used freely in dialog boxes.

On the command line and in scripts however, object names and column names must be enclosed by single quotes when the object name or the column name:

  • starts with a digit from 0 to 9,
  • contains one or more spaces,
  • contains any of the following characters ` ~ ! @ # $ ^ & ( ) - + = [ ] { } ; ' , .

Object extensions should be left outside the quotes. If an object name or a column name already contains a single quote, this single quote must be duplicated.
