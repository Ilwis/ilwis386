Graph window

Toolbar

The toolbar of a graph window provides shortcuts for some regularly used menu commands for instance to add other graph layers, to remove a graph layer, to save a graph, or to print a graph.

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating. Some of the actions that are available via the toolbar are also available via the context-sensitive menu.

The following actions can be performed:

 

Open:

Open another graph in the current graph window.

Save:

Save the contents of the current graph window. When this is the first time you save the graph, the Save Graph As dialog box will appear. The option is not available when a graph has been saved already and has not changed since.

Print:

Print the contents of the current graph window. First, a Print Options dialog box will appear. The option is only available when the viewer part of the graph window is active.

Copy:

Copy the contents of the current graph window as a picture to the clipboard. Subsequently, you can paste the graph into an ILWIS layout or into any other Windows application, for instance into a word processor. You can also use the Ctrl+C keys on the keyboard. The option is only available when the viewer part of the graph window is active.

Add Graph:

Add another graph layer (from columns) to the graph window. If you press the arrow down button, you can also add a Graph from Formula, a Least Squares Fit, or a SemiVariogram Model.

Remove Layer:

Remove a selected graph layer from the graph window. First select the layer to be removed in the Graph Management pane of the graph window.

The toolbar can be hidden or shown with the Toolbar command on the Options menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
