Status bar

A status bar is located at the bottom of each table window. The status bar can be shown or hidden with the Status Bar command on the View menu.

The status bar gives brief explanations when the mouse pointer is located on:

  • a menu command,
  • a button in the toolbar,
  • a column header:

    • double-click to change column properties, also the column name and column description are shown,

  • a record button:

    click to view the record in record view and optionally edit the record there,

  • the Column Management button :

    • click to open the Column Management dialog box.

  • a field in the table:

    • column name and column description are shown.

  • a record button or field in the statistics pane.
