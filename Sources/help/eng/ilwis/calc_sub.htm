SUB( ) function

Returns a substring of a string, starting at a specified position and continuing for a specified number of characters.

Syntax

SUB(String, Value a, Value b)

Input

String is a string
Domain type: any Class or ID domain, or system domain String
Value a is the value that specifies the starting position in the text string, from where the substring is returned
Domain type: any Value domain
Value range: >=1
Value b is the value that specifies the number of characters that should be returned
Domain type: any Value domain
Value range: >=0

Output

SUB returns: a string
Domain type: system domain String

Notes:

  • If Value b is greater than the length of the input string the SUB function returns the entire input string.
  • If Value a and Value b are not integer values, they are rounded off.
  • When the input string is undefined, the output will also be undefined.

Tip:

  • SUB(String, 1, Value) is equivalent to LEFT(String, Value).
  • When the definition symbol = is used, a dependent output column is created; when the assignment symbol := is used, the dependency link is immediately broken after the output column has been calculated.

Usable in

TabCalc, Simple calculators

Examples

Pocket line calculator examples:

? SUB("Landuse",5,3)

returns use

? SUB("Industrial",2.5,3.6)

returns dust

TabCalc example:

OutCol = SUB(InCol,4,6)
