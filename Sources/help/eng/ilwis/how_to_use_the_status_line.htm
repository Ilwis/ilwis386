How to use the status bar

The status bar is located at the bottom of the Main window, a map window, a table window, the pixel information window, the SMCE window and in editors. The status bar gives a brief explanation on selected menu commands and on the functionality of buttons in a toolbar. Further, the status bar may give additional information when the mouse pointer is located in a Catalog, the Operation-tree, the Operation-list, a map window, a table window, a pixel information window, a SMCE window or an editor.

Information on menu commands:

Highlight a menu command by opening a menu and drag the mouse pointer along the commands on the menu; the status bar displays information that helps explain the task of the highlighted menu command. You can also use the UP ARROW and DOWN ARROW keys on the keyboard to move through the commands on a menu.

Information on buttons in the toolbar:

Move the mouse pointer over a button in a toolbar; the status bar gives a short explanation of the function of the button.

Information on objects and operations:

When you move with the mouse pointer over an object in a Catalog, or over an operation in the Operation-tree or the Operation-list, the status bar shows the description of the object or operation.

For objects in a Catalog, the status line will also display whether objects are dependent (D), calculated (C) and or up-to-date (U). Furthermore, the query used in the current Catalog is shown.

Information on coordinates and row and column numbers in a map window:

When you move with the mouse pointer in a map window, X and Y coordinates are displayed on the status bar. When a raster map is displayed in a map window, the row and column numbers appear as well.

Hide or Show the status bar:

You can hide the status bar by opening a View or Options menu and selecting the command Status Bar. The check mark to the left of the command Status Bar disappears, and the status bar will no longer appear in your window. When you choose this command again, the check mark and the status bar will reappear.
