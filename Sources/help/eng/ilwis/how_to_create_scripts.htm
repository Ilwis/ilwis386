How to create a script

A script is a sequenced list of ILWIS commands and expressions. By creating a script, you can build a complete GIS or Remote Sensing analysis for your own research discipline. Scripts are more or less equivalent to batch files in ILWIS version 1.4.

To create a script:

  • in the Main window, open the File menu and choose Create Script,
  • in the Operation-tree, expand the Create item, and double-click New Script, or
  • in the Operation-list, double-click the New Script item.

The Script editor appears in which you can type your script expressions.

For more information on valid script expressions, refer to Appendices : ILWIS commands, Appendices : ILWIS expressions or Appendices : ILWIS script language (syntax).

An example of a script to create slope maps in percentages and in degrees is presented in Map & Table calculation : creating and running scripts and in How to use parameters in scripts.

You can create your own Help for a script so that you can explain the meaning of parameters to other users of the script. For more information, see the Run Script dialog box.
