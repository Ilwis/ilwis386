How to customize a Catalog

List View and Details View:

When using List View, all objects in a Catalog are shown as a simple list of objects and icons.

When using Details View, all objects in a Catalog are shown in a table-like fashion with detailed information on each object, for instance whether objects are dependent or not, used domain/georeference/coordinate system, their description, expression, etc.

To switch between List View and Details View:

  • choose the List or the Details command from the View menu in the Main window;
  • click the choose the List button or the Details button in the standard toolbar of the Main window;
  • click the right mouse button in the white space of a Catalog, and choose the List or the Details command from the context-sensitive menu.

Show or hide certain object types:

  • choose the Show command from the View menu in the Main window;
  • use the object selection toolbar in the Main window; or
  • choose the Customize Catalog command on the View menu, or click the button in the standard toolbar of the Main window.

Sorting objects:

  • choose the Sort By command on the View menu in the Main window;
  • when in Details view: click a column header; columns can be sorted in Ascending order (delta up) and in Descending order (delta down);
  • click in the white space in a Catalog, and select Sort from the context-sensitive menu.

Using queries to search for certain objects:

  • choose the Catalog Queries command on the View menu of the Main window;
  • choose the Customize Catalog command on the View menu, or click the button and select the Query tab.

Tips:

  • To include system objects or non-ILWIS objects in a Catalog, e.g. DBF files and/or TIF files, see Customize Catalog, Selection tab.
  • To see only objects that start with certain characters, or which use a certain domain, georeference, etc.: use a predefined query or contstruct a query yourself.

Mind:

  • Settings on visible object types, sorting/ordering method and queries on objects are either applied to all new Catalogs that will be opened, or to the currently active Catalog only. This depends on whether you click the Use current settings for new Catalogs button or the Use default settings for new Catalogs button on the Customize Catalog, Details View page.
  • Only those objects will be shown which match the selected object types and a used query.

Other functionality:

  • To open another new Catalog, choose the New Catalog command from the Window menu in the Main window, or click the New Catalog button in the standard toolbar of the Main window.
  • To refresh the contents of a Catalog, choose the Refresh command from the Window menu, or press the F5 key on the keyboard.
  • To arrange Catalogs, choose the Arrange Catalogs command from the Window menu.
  • To close all Catalogs, choose the Close All Catalogs command from the Window menu.
