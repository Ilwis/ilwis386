Epipolar Stereo Pair - Creation window

Toolbar

The toolbar of the Epipolar Stereo Pair - Creation window provides shortcuts for some regularly used menu commands for instance to zoom in on maps, to pan, to mark the position of fiducial marks, transferred principal point, etc.

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating. Some of the actions that are available via the toolbar are also available via the context-sensitive menu.

 

Tip: For most actions you first need to use the Activate Left Pane or the Activate Right Pane buttons to activate the pane you wish to work in.

The following actions can be performed:

New Stereo Pair:

 

To create a new stereo pair. The Create Stereo Pair dialog box will appear.

Open Stereo Pair:

 

To open another stereo pair in the Stereoscope window. The Open Stereo Pair dialog box will appear.

 

Entire Map:

Displays the entire map in the currently active pane.

Redraw:

Redraws the map in the currently active pane.

Zoom In:

Allows you to zoom in on a selected spot or area in the map of the currently active pane. In the currently active pane, the mouse pointer will change into a magnifying-glass with a large plus inside, then:

  • in the active pane, click in the map to zoom in on that position with a factor 2, or
  • in the active pane, drag the mouse pointer in the from one corner of interest to another (any zoom in factor).

Tip: To zoom in, you can also use the overview window.

Zoom Out:

Allows you to zoom out from the map in the currently active pane.

In the currently active pane, the mouse pointer will change to a 'magnifying'-glass with a large minus inside, then:

  • in the active pane, click in the map to zoom out with a factor 2, or
  • in the active pane, drag a rectangle with the mouse pointer in the map; the smaller the rectangle, the more you will zoom out.

 

Select Fiducial Marks:

 

In the currently active pane:

  • Check that the cursor displays the number of the fiducial mark that you wish to enter, and that the row/col boxes of the fiducial mark that you are about to enter appear highlighted (by default yellow).
  • Position the cursor as exact as possible and click the location of the fiducial mark in the photograph, or type row/column numbers in the lower dialog pane.

Select Principal Point:

 

In case you don't have enough fiducial marks, or when you wish to mark the position of the principal point yourself:

  • Check that the cursor displays PP, and that the row/col boxes of the principal point appear highlighted (by default yellow).
  • In the currently active pane, position the cursor as exact as possible and click the location of the principal point in the photograph, or type row/column numbers in the lower dialog pane.

Select Transferred Principal Point:

 

In the currently active pane:

  • Check that the cursor displays TP, and that the row/col boxes of the transferred principal point appear highlighted in the lower dialog pane (by default yellow);
  • In the currently active pane, position the cursor as exact as possible and click the location of the transferred principal point in the photograph.

The position of the principal point of the left photograph has to be marked as the transferred principal point on the right photograph, and vice versa.

Select Scaling Points:

 

In case there are scale differences between the left photograph and the right photograph:

  • Check that the cursor displays SP, and that the row/col boxes of the scaling point that you wish to enter appears highlighted in the lower dialog pane (by default yellow);
  • Position the cursor as exact as possible and click the location of the scaling point in the photograph.

The position of a scaling point needs to be marked on the other photograph as well.

 

Activate Left Pane:

 

Activates the left-hand pane in the Epipolar Stereo Pair - Creation window.

Activate Right Pane:

 

Activates the right-hand pane in the Epipolar Stereo Pair - Creation window.

 

Exit:

Click this button to close the Epipolar Stereo Pair - Creation window:

  • The changes you made in the Epipolar Stereo Pair - Creation window will be stored.
  • However, the stereo pair and its output maps will not be calculated yet.

Exit and Show:

 

Click this button to close the Epipolar Stereo Pair - Creation window and show the resulting stereo pair:

  • All entered points will be saved.
  • The stereo pair and its output maps will be calculated.
  • The resulting stereo pair will be displayed, by default, in the Stereo Pair - Stereoscope window where you can view the stereo pair with a stereoscope.

The toolbar can be hidden or shown with the Toolbar command on the Options menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
