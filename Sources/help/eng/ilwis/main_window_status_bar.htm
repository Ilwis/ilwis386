Status bar

A status bar is located at the bottom of the Main window.

The status bar:

  • gives a brief explanation on highlighted menu commands,
  • gives a brief explanation on the effect of buttons in a toolbar when moving with the mouse pointer over a button,
  • displays the description of operations when moving the mouse pointer over an operation in the Operation-tree or the Operation-list,
  • displays the description of objects when moving the mouse pointer over an object in a Catalog,
  • displays whether objects are dependent (D), calculated (C), and/or up-to-date (U) when moving the mouse pointer over an object in a Catalog,
  • displays the query that is used in a Catalog.

Tip:

When during an operation no description is filled out for the dependent output object, the dependent output object will use its definition or expression as description.

For more information, see How to use the Status bar.
