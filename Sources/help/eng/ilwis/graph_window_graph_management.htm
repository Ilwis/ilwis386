Graph window

Graph Management

The Graph Management pane is by default located along the left hand side of a graph window. It lists the table from which the graph was created, the X-axis, Y-axes (left and right) and all graph layers that were added to the current graph window.

The upper part of the Graph Management pane contains information on the title of the graph, the legend of the graph, the table from which the graph was created, and the X-axis, Y-axis (left) and Y-axis (right). All items are editable, except for the table name.

Graph layers are listed at the lower part of the Graph Management pane. They are preceded by a check box to show or hide the layer (show or hide ). You can modify the name of each graph layer and the appearance of each graph layer.

In short, the Graph Management pane allows you:

  Title:

To edit the title of the graph.

  Legend:

To show or hide the legend of the graph.

  Table:

To open the table from which the graph was created.

  X-Axis, Y-axis:

To edit the appearance of the X-Axis, Y-axis Left or Y-Axis Right (including a description of the axes and grid lines).

  Axis Text:

To edit the additional text describing an axis.

  Min/Max value:

To change the minimum and maximum value used on an axis.

To show/hide a graph layer and to change the display options of a graph layer.

Individual graph layers can be switched on or off ; also the legend can be switched on or off.

The right mouse button can for instance be used to remove a graph layer, or to add another graph layer.

You can move the Graph Management pane and dock it along the right hand side or the left hand side of a graph window; you can also make the Graph Management pane floating. The Graph Management pane can be hidden or shown with the Graph Management command on the View menu. Some of the actions that are available via the Graph Management pane are also available in the graph window itself or via the context-sensitive menu.

Functionality

The functionality is described in three small sections:

1. Title, legend, table

2. Axes

3. Graph layers

You can furthermore refer to the Example below.

 

1. Title, legend, table

To edit the title of the graph:

  • Double-click the first Text item in the Graph Management pane, i.e. the item just below the Graph item.
  • The Title dialog box will appear.

Settings include text of title, font, font size, etc.

 

Tip: To edit the title of a graph, you can also directly double-click the title in the graph window itself.

To hide or show the legend in the graph window:

  • Click the show or hide check box in front of the Legend item (toggle).

 

Tip: To move the legend in a graph window, click the legend in the graph window and drag it to another position.

To open the table from which the graph was created:

  • Double-click the Table item.

 

2. Axes

To change the settings of the X-axis or Y-axis (left or right)

  • Double-click one of the following items:

    • to modify the X-axis; the X-axis dialog box will appear.

    to modify the Y-axis (left); the Y-axis (left) dialog box will appear.

    to modify the Y-axis (right); the Y-axis (right) dialog box will appear.

    Click the check box to show or to hide a Y-axis on the right of the graph in the graph window (toggle).

Settings include among others additional text for an axis, axis type (linear, logarithmic, or normal probability), minimum and maximum values used by an axis, interval, grid lines, font and font size.

Tips:

  • To change the settings of the axes, you can also directly double-click an axis in the graph window itself.
  • For examples of different types of axes, see Graph window : examples of graphs.

To change the values used on the X-axis or the Y-axis (left or right):

  • Double-click one of the values listed below the X-Axis item, the Y-Axis Left item, or the Y-Axis Right item.
  • You can type another minimum or maximum value to be used for the axis without dialog box.

To edit the additional text for an axis:

  • Double-click the text item listed below the X-Axis item, the Y-Axis Left item, or the Y-Axis Right item.
  • You can type another text describing the axis without dialog box.

 

3. Graph layers

To show or hide a graph layer:

  • Click the show or hide check box in front of a graph layer (toggle).

To modify the appearance of a graph layer:

  • Double-click a graph layer in the Graph Management pane.
  • Depending on whether you double-clicked a graph from columns, a graph from formula, a least squares fit, a semi-variogram model, or a rose diagram:
    • the Graph Options - Graph from Columns dialog box will appear.
    • the Graph Options - Graph from Formula dialog box will appear;
    • the Graph Options - Least Squares Fit dialog box will appear.
    • the Graph Options - Semi-variogram Model dialog box will appear,
    • the Graph Options - Rose diagram dialog box will appear.

Settings include name of the graph layer, symbol type, size, line type, color, etc.

To add a graph layer:

  • Use the right mouse button in the white space of the Graph Management pane, and
  • choose Add Graph from the context-sensitive menu.
  • Depending on whether you selected to add a graph from columns, a graph from formula, a least squares fit or a semi-variogram model:
    • the Add Graph from Columns dialog box will appear,
    • the Add Graph from Formula dialog box will appear,
    • the Add Graph Least Squares Fit dialog box will appear,
    • the Add Semi-variogram Model dialog box will appear.

 

Tip: To add a graph layer to the graph window, you can also use the Add Graph command on the Edit menu, or the Add Graph button in the toolbar.

To change the order of graph layers:

  • Drag a graph layer to another position in the list.

To remove a graph layer:

  • Use the right mouse button on a graph layer, and
  • choose Remove Layer from the context-sensitive menu.

 

Tip: To remove a graph layer from a graph window, you can also use the Remove Layer command on the Edit menu, or the Remove Layer button in the toolbar.

Example

The functionality of the Graph Management pane will be demonstrated by an example shown in the graph window below. The graph window shows:

  • points of columns 'val' (a column with values) and 'val2' (another column with values),
  • a least squares graph as a line.

 

 

Title: Double-click to edit the title of the graph, select a font, font size etc. for the title in the Title dialog box.

Legend: Click the check box to show the legend in the graph window, or to hide the legend from the graph window (toggle).

Table name: Double-click to open the table from which the graph was created, or click with the right mouse button and choose Open from the context-sensitive menu.

X-axis: Double-click to change the settings of the X-axis in the X-axis dialog box. Settings include among others axis text, type of axis (linear, logarithmic, normal probability), minimum and maximum values used by the axis, interval, grid lines, font and font size.

Text: Double-click to directly change the additional text describing the X-axis.

Minimum value: Double-click to directly change the minimum value for the X-axis.

Maximum value: Double-click to directly change the maximum value for the X-axis.

Y-axis (left): Double-click to change the settings of the Y-axis (displayed on the left of the graph) in the Y-axis (left) dialog box. Settings include among others axis text, type of axis (linear, logarithmic, normal probability), minimum and maximum values used by the axis, interval, grid lines, font and font size.

Text: Double-click to directly change the additional text describing the Y-axis.

Minimum value: Double-click to directly change the minimum value for the Y-axis.

Maximum value: Double-click to directly change the maximum value for the Y-axis.

Y-axis (right):

Click the check box to show or to hide a Y-axis on the right of the graph in the graph window (toggle).

Double-click to change the settings of the Y-axis (displayed on the right of the graph) in the Y-axis (right) dialog box. Settings include among others axis text, type of axis (linear, logarithmic, normal probability), minimum and maximum values used by the axis, interval, grid lines, font and font size.

Text: Double-click to directly change the additional text describing the Y-axis on the right.

Minimum value: Double-click to directly change the minimum value for the Y-axis on the right.

Maximum value: Double-click to directly change the maximum value for the Y-axis on the right.

 

 

Graph layer: Double-click to change the display options of this graph layer.

As this layer is a least squares fit, the Graph Options - Least Squares Fit dialog box will appear. Settings include among others name of the graph layer, line type, color, line width, and function used to calculate the least squares fit.

Columns: The name of the columns in the table from which this least squares fit was calculated.

 

 

Graph layer: Double-click to change the display options of this graph layer.

As this layer is a point graph from two columns in the table, the Graph Options - Graph from Columns dialog box will appear. Settings include among others name of the graph layer, graph type (line, bar, points), symbol type or line type, color, size, etc.

Columns: The name of the columns in the table that are shown as points in this graph layer.

Tips:

  • To add a graph layer to the graph window, you can also use the Add Graph command on the Edit menu, or the Add Graph button in the toolbar.
  • To edit the title of a graph, you can also directly double-click the title in the graph window itself.
  • To change the settings of the axes, you can also directly double-click an axis in the graph window itself.
  • To remove a graph layer from a graph window, you can also use the Remove Layer command on the Edit menu, or the Remove Layer button in the toolbar.
