Table window

Toolbar

The toolbar of a table window provides shortcuts for some regularly used menu commands for instance to copy and paste, to print, to view the properties of a column, and to go to the first, the previous, a specific, the next, or the last record of the table.

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating. Some of the actions that are available via the toolbar are also available via the context-sensitive menu.

The following actions can be performed:

 

Copy:

Copies the contents of the selected field(s), column(s) or record(s) to the clipboard.

Paste:

Pastes the contents of the clipboard into the selected field(s), or into the selected column(s) of the current table.

Delete:

Clears the contents of the selected fields, or of the selected record(s), or deletes the selected column(s).

Print:

Prints the whole table or a selection of the table. First, a Print Options box will appear.

Column Properties:

When a column is selected: the Column Properties dialog box will appear.

Update All Columns:

Updates the values of all dependent columns.

Graph:

Opens the Create Graph dialog box to create a new graph from columns in a new graph window. For more information on graphs, refer to Graph window : functionality.

Goto First Record:

Scrolls to the first record and slightly highlights this record. The record is furthermore shown in Record View.

Goto Previous Record:

Scrolls to previous first record and slightly highlights this record. The record is furthermore shown in Record View.

Goto Record:

Opens the Goto Record dialog box in which you can specify a record you wish to view.

After clicking the OK button, the table will scroll to the specified record and will slightly highlight it. The record will also be shown in Record View.

Goto Next Record:

Scrolls to the next record and slightly highlights this record. The record is furthermore shown in Record View.

Goto Last Record:

Scrolls to the last record and slightly highlights this record. The record is furthermore shown in Record View.

The toolbar can be hidden or shown with the Toolbar command on the View menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
