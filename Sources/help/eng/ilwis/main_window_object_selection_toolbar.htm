Object selection toolbar

The object selection toolbar of the Main window allows you to specify the object types that should be visible in the current Catalog.

 

  • All object types of which the button appears down will be shown.
  • The first button is used to hide all object types (after which you can click specific object buttons, to show only these object types), or to show all object types.

 

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the object that you are pointing to.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating.

Using the object selection toolbar:

Click the ILWIS button to switch between:

  • hide all object types; after this you can click specific object buttons, to show only these object types,
  • show all object types.

Click an object type button to switch between:

  • show the object type,
  • hide the object type.

Click the directory button or the drive button to switch between:

  • show subdirectories and the parent directory or drives,
  • hide subdirectories and the parent directory or drives.

Mind:

  • Settings are applicable to the currently active Catalog only.
  • In a Catalog, only those objects will be shown which match the selected object types and a used query.

The Object Selection toolbar can be hidden or shown with the Object Selection Toolbar command on the View menu.
