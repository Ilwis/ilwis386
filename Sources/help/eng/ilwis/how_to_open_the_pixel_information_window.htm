How to open

the pixel information window

There are several ways to open the pixel information window:

  

  1. In the toolbar in the Main window, click the Pixel Information button ;
  2. From the File menu in the Main window, choose Open Pixel Information;
  3. From the File menu in a map window, choose Open Pixel Information;
  4. Type Pixelinfo on the command line of the Main window.
