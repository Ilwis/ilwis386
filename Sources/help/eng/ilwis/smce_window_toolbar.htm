SMCE window

Toolbar

The toolbar of the SMCE window provides shortcuts for some regularly used menu commands for instance to open and save a criteria tree, to add groups, factors and constraints, to weigh and standardize items or to calculate output maps.

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also, the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

You can move the toolbar and dock it along an edge of the window; you can also make the toolbar floating. Some of the actions that are available via the toolbar are also available via the context-sensitive menu.

The following actions can be performed:

 

Open:

Open an existing criteria tree (*.smc file).

Save:

Save the current criteria tree.

Problem Definition:
 

Switch to Problem Definition mode:

  • The structure of the criteria tree can be created, i.e. goals, sub goals, groups, factors, constraints, and alternatives can be added.
  • The criteria tree can be filled with maps and attribute tables.
  • The map name for the main goal and the map names for possible sub goals can be entered.
  • Existing criteria can be renamed, other maps can be selected, or criteria can be deleted.

Multicriteria Analysis:
 

Switch to Multicriteria Analysis mode:

  • Individual factors and constraints can be standardized.
  • Groups of factors can be weighed.
  • Output maps can be generated.
  • The names of criteria and the selected maps cannot be changed.

The Multicriteria Analysis mode should only be selected after completing the Problem Definition mode.

Alternatives:

Change the number of alternatives and/or edit the name(s) of the alternative(s).

Insert Group:

Insert a new group, representing a sub-objective or sub goal:

  • under the main root of the criteria tree, or
  • under the currently selected group in the criteria tree.

Insert Factor:

Insert a new factor (i.e. a benefit or a cost) under the main root of the criteria tree, or under the currently selected group in the criteria tree.

  • Benefit: the more you have of such a criterion, the better it is. A benefit is not necessarily monetary.
  • Cost: the less you have of such a criterion, the better it is. A cost is not necessarily monetary.

Insert Constraint:
 

Insert a new constraint under the main root of the criteria tree.

Edit:

Edit the currently selected item in the criteria tree. The actual edit action depends on the mode (Problem Definition mode or Multicriteria Analysis mode) and the type of item selected.

Delete:

Delete the currently selected item from the criteria tree.

Standardize:

Standardize the currently selected factor or constraint.

Weigh:

Weigh the main root or the currently selected group of factors.

Generate Selected Maps:
 

Generate the currently selected map(s). Depending on the selection, one or more output maps will be calculated.

Generate All Maps:
 

Generate the composite index maps of all alternatives, and all intermediate maps.

Show:

Show the currently selected input or output map(s).

Histogram:

Calculate and show the histogram(s) of the SMCE output map(s).

Aggregate Values:
 

Aggregate the values of the SMCE output map(s). You can calculate Minimum, Maximum, Sum and Average.

Slice:

Classify the values of the SMCE output maps and obtain sliced SMCE output maps.

Show Sliced:

Show the sliced SMCE output maps.

Histogram Sliced:

 

Calculate and show the histograms of the sliced SMCE output maps.

Aggregate Values Sliced:

 

Aggregate the values of the sliced SMCE output maps. You can calculate the Shape Index and the Connectivity Index.

The toolbar can be hidden or shown with the Toolbar command on the View menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
