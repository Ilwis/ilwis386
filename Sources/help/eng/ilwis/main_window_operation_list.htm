Operation-tree and Operation-list

The Operation-tree and the Operation-list are located on the first two tabs in the Operations/Navigator pane, by default along the left hand side of the Main Window.

 

  • The Operation-tree provides a tree structure for all ILWIS operations, similar to the Operations menu in the Main window;
  • The Operation-list contains an alphabetic list of all ILWIS operations.

The icon by which an operation is preceded indicates the output data type of the operation.

The Operation-tree and the Operation-list are mainly used to start operations, bypassing the Operations menu. Furthermore, the Operation-tree and the Operation-list contain commands to create new objects, to open and/or edit objects, to view the properties of an object, to copy and delete one or more objects, etc. If you wish you can hide the Operations/Navigator pane.

For some short Help on an operation, you can click an operation with the right mouse button, and choose Help from the context-sensitive menu.

 

Tip: To extend the Operation-tree, the Operation-list and the Operations menu with scripts or external tools, see How to customize the Operation-tree, the Operation-list and the Operations menu.

Using the Operation-tree and the Operation-list:

The Operation-tree and the Operation-list are mainly used for the following actions:

  • move the mouse pointer over an operation:
    the status bar displays a brief description of that operation;
  • double-click an operation:
    the operation is started and the operation's dialog box appears;
  • drag a map or table from the Catalog to an operation in the Operation-tree or Operation-list:
    the operation's dialog appears and the input map (or table) name is already filled out in the dialog box;
  • drag a map or table from a Catalog to a text box or a drop-down list box in a dialog box of an operation:
    the object is filled out in the field of the dialog box;
  • click an operation with the right mouse button:
    a context-sensitive menu appears; start the operation by choosing the Run command, or get short Help on the operation by choosing the Help command.

Tips:

  • To start an operation, you can also type a command on the command line of the Main window (see also Appendices : ILWIS commands).
  • In all cases mentioned above, a dialog box is opened in which you can enter the input and output object names and other parameters necessary for the operation.

    • When you click the Show or Define button in the operation's dialog box, an ILWIS expression is generated on the command line of the Main window.

    Output objects that are obtained through an operation's dialog box are always dependent.

  • Advanced users can type complete ILWIS expressions on the command line of the Main window or create a script to execute a series of expressions. For more information, see Appendices : ILWIS expressions and Appendices : ILWIS script language (syntax).
