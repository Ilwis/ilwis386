Progress manager

Functionality

The progress manager lists all actions, calculations and operations that are currently performed by ILWIS and which will take some time. The progress manager shows the progress of each of these background processes.

For each operation that is running, the progress manager shows:

  • the name of the operation and the name of the output map or table that is being calculated,
  • the operation's progress indicator (if applicable),
  • the amount of completion (if applicable) as:
    • number of lines processed (in case of raster maps), or
    • number of points, segments, polygons processed (in case of point, segment or polygon maps), or
    • number of records processed (in case of tables),
  • additional information about the operation that is performed.

When multiple calculations or operations are performed at the same time, the progress manager will list these on multiple lines.

To interrupt an operation before it has finished:

When only a single operation is running:

  • Directly click the Stop button in the progress manager.

When multiple operations are running:

  • In the progress manager, select the operation you wish to stop, and
  • click the Stop button in the progress manager.

To view the Algorithm Help topic of an operation while the operation is running:

  • Select the operation in the progress manager, and
  • click the Help button in the progress manager.

Additional information:

  • The progress manager will only appear if a calculation or operation takes longer than 2 seconds.
  • If no calculations or operations are performed anymore, the progress manager will automatically disappear.

 

Tip: Optionally, you can use the old style progress indicators, i.e. progress indicators in separate windows for each operation; use the General entry in the Preferences.
