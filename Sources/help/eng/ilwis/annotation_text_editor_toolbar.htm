Annotation Text editor

Toolbar

The toolbar of the annotation text editor largely resembles the toolbar of a map window; there is an extra button to exit the editor.

When you leave the mouse pointer for a short period of time on top of a button, a tooltip will appear with the name of the button / the menu command that it will perform. Also the status bar at the bottom of the window may provide you with a short explanation on the effect of using the button.

The following actions can be performed:

 

Entire Map:

Displays the entire map in the map window.

Zoom In:

Zooms in on a selected spot or area in the map. The mouse pointer changes into a magnifying-glass. Click to zoom in on a spot of interest, or drag the pointer from one corner of interest to another (the mouse pointer changes into a little box).

Zoom Out:

Zooms out directly from the center of the displayed map with a factor 2.

Redraw:

Redraws the contents of the map window.

Exit editor:

Leaves the editor and keeps all changes.

The toolbar can be hidden or shown with the Toolbar command on Options menu. If the action of a certain button cannot be performed at a certain moment, the button appears gray.
