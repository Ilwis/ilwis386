Sub-map of segment map

Algorithm

The Sub-map of segment map operation copies a rectangular part of a segment map into a new segment map.

The user has to specify minimum and maximum XY-coordinates for the output segment map. When the input map uses a coordinate system of type latlon, the user has to specify minimum and maximum Latitudes and Longitudes for the output segment map. The specified minimum and maximum coordinates define the so-called clipping rectangle.

Then:

  • Segments that entirely fall within the clipping rectangle are copied to the output map without modifications;
  • Segments that have one intersection with the clipping rectangle, obtain a node at the intersection, and then the parts within the clipping rectangle are copied into the new output map;
  • Segments that have more than one intersection with the clipping rectangle, obtain nodes at each intersection, and then the parts within the clipping area are copied as multiple segments into the output map.
  • Segments that completely fall outside the clipping rectangle are not copied.
