Sub-map of point map

Algorithm

The Sub-map of point map operation copies all points within a user-specified rectangle into a new point map.

The user has to specify minimum and maximum XY-coordinates for the output point map. When the input map uses a coordinate system of type latlon, the user has to specify minimum and maximum Latitudes and Longitudes for the output point map.

Then:

  • all points of the input map that fall within the output coordinate rectangle are copied to the output point map;
  • points of the input map that fall outside the coordinate rectangle are not copied into the output point map.
