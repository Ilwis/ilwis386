Histogram

Dialog box

The Histogram operation calculates the histogram of a raster, polygon, segment or point map. Histograms list frequency information on the values, classes, or IDs in your map.

The results are shown in a histogram window as a table and as a graph.

Histograms of Value raster maps also contain summary information such as mean, standard deviation, and percentage intervals. The summary information is available in a histogram window as well as in the Properties of the histogram.

Dialog box options:

Input map:

Select an input raster, polygon, segment or point map. Open the list box and select the desired input map, or drag a raster, polygon, segment or point map directly from the Catalog into this box.

The output histogram always obtains the same name as the input map.

When you click the Show button, the dependent histogram will be defined, calculated and displayed in a table window. When you click the Define button, the histogram will only be defined; if necessary, it will be calculated later, for instance when the histogram is opened to be displayed.

Notes:

  • For raster maps that are stored using 1 or 2 bytes per pixel, a histogram will automatically be calculated when required. Thus, for such raster maps there is no need to perform the Histogram operation separately.
  • You can use the Histogram operation to calculate histograms for raster maps stored with 4 or 8 bytes per pixel, and to calculate histograms for polygon, segment or point maps.
