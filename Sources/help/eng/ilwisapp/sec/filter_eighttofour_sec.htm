CONN8TO4 filter

The CONN8TO4 filter is a standard binary filter which works in a 3x3 environment.

When true pixels are only 8-connected, this filter makes them 4-connected by adding a true pixel in some places. This filter must be used when you want to use line barriers in a distance calculation.
