Tunnel segments

Command line

The Tunnel segments operation can be directly executed by typing the following expression on the command line of the Main window:

  

OUTMAP

=

SegmentMapTunneling(InputSegmentMap, tunnel width, remove nodes)

where:

OUTMAP

is the name of your output segment map.

SegmentMapTunneling

is the command to start the Tunnel segments operation.

InputSegmentMap

is the name of the input segment map.

tunnel width

is the width of the tunnel in meters; real value >= 0; when a tunnel width of 0 is specified, no tunnelling is performed.

remove nodes

remove superfluous nodes or not:

yes | no

When the definition symbol = is used, a dependent output map is created; when the assignment symbol := is used, the dependency link is immediately broken after the output map has been calculated.
