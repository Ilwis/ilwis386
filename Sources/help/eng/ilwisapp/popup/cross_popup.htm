Cross

The Cross operation performs an overlay of two raster maps which have the same georeference. Pixels on the same positions in both maps are compared; the occurring combinations of class names, identifiers or values of pixels in the first input map and those of pixels in the second input map are stored.

These combinations give an output cross table and optionally an output cross map. The cross table contains the combinations of input values, classes or IDs in both maps, the number of pixels that occur for each combination, and the area of each combination.
