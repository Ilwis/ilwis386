Stereoscope

Stereoscope will show a calculated stereo pair in a Stereo Pair - Stereoscope window; you will be able to view the height differences in the stereo pair by using a stereoscope.

To create a stereo pair:

  • use the Epipolar Stereo Pair operation, or
  • use the Stereo pair from DTM operation.

 

Tip: A stereo pair can also be shown as an Anaglyph in a map window.
