Transform coordinates

The Transform coordinates dialog box allows you to:

  • select a certain input coordinate system,
  • type a XY-coordinate or a LatLon coordinate, using this input coordinate system,
  • select another target coordinate system,
  • then, the entered XY-coordinate or LatLon-coordinate will be shown in the selected target coordinate system.

In this way, you can interactively check whether a certain transformation is correct.

The Transform Coordinates dialog box can only be used when a transformation between the input coordinate system and the target coordinate system is possible.
