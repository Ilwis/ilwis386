Cumulative Hypsometric Curve

The Cumulative Hypsometric Curve script crosses a merged catchments map with your DEM. The script will ask for the ID of a single catchment that you wish to use in the Cross. In the resulting cross table, cumulative values are calculated for the Area of subsequent height values in pixels and in percentages. The output is stored in columns: cum_area and cum_percent.

With the output cross table and its newly calculated columns, you can create graphs:

  • use column cum_area or column cum_percent on the X-axis,
  • use the column with the height values on the Y-axis.

This will give you a so-called cumulative hypsometric curve.

As input, this script requires:

  • an output map of a previous Catchment Merge operation, and the ID of a catchment in this map,
  • a DEM.

For more information, refer to Cumulative Hypsometric Curve.
