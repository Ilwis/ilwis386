Apply 3D

The Apply 3D operation resamples an input raster map according to a georeference 3D. This enables you to permanently and quickly display the output raster map in three dimensions, i.e. as 3D view.

You can create a georeference 3D with Display 3D or Apply 3D; a Digital Elevation Model (DEM) is required and you have to specify 3D view parameters using the Georeference 3D editor.
