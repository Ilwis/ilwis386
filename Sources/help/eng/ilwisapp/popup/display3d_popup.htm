Display 3D

With Display 3D, you can create and edit a georeference 3D in order to obtain a three dimensional view of one or more maps. For the creation of a georeference 3D, a Digital Elevation Model (DEM) is required.

The DEM can then be displayed as 3D grid lines with or without a drape of a raster map. The georeference 3D can be edited with the Georeference 3D editor. When finished editing the 3D view, you can add point, segment and/or polygon maps and/or annotation to improve the 3D view.

With Apply3D, you can create a permanent raster map which uses your georeference 3D. In fact an input raster map is resampled to the georeference 3D.
