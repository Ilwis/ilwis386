Slide show

With Slide Show, you can display multiple raster maps that are combined in a map list as a slide show in a map window. The maps of the map list are displayed one after the other in the map window, at a user-specified rate. A slide show is designed to present multi-temporal changes in maps.

You can create a slide show:

  • to present temporal changes in maps: make a time-series by combining several maps of the same theme but of for instance different years into a map list and show the map list as a slide show,
  • to present temporal changes in for instance classified satellite images (of different months or years),
  • to display a number of 3D views one after the other.

For more information, see Show Map List as slide show dialog box or How to display a map list as a slide show.
