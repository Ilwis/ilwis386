Sub-map of raster map

Dialog box

The Sub-map of raster map operation copies a rectangular part of a raster map into a new raster map. The user has to specify row and column numbers or XY-coordinates of the input map to indicate the part of the input map that should be copied into the new raster map.

Dialog box options:

Input raster map:

Select an input raster map or an input map list. Open the list box and select the desired input map or map list, or directly drag a raster map or map list from a Catalog into this box. No special input map requirements.

When you select an input map list, the operation will be performed on all raster maps in the map list.

Lines and columns:

Select this option when you want to specify start line and column and the number of lines and columns that should be copied.

Corners:

Select this option when you want to specify start line and column and end line and column; the area in between will be copied.

Coordinates:

Select this option when you want to specify start the XY-coordinate and the end XY-coordinate; the area in between will be copied.

 

Lines and columns option:

First line:

Type the line number of the input map where copying should start (top row).

First column:

Type the column number of the input map where copying should start (left column).

Number of lines:

Type the number of lines of the input map that you want to be copied starting from 'First line'.

Number of columns:

Type the number of columns of the input map that you want to be copied starting from 'First column'.

 

Corners option:

First line:

Type the line number of the input map where copying should start (top row).

First column:

Type the column number of the input map where copying should start (left column).

Last line:

Type the line number of the input map where copying should end (bottom row).

Last column:

Type the column number of the input map where copying should end (right column).

 

Coordinates option:

First coordinate:

Type the X-coordinate and the Y-coordinate where copying should start. You may specify any corner where copying should start: the upper left, upper right, lower left or lower right corner.

Opposite coordinate:

Type the X-coordinate and the Y-coordinate where copying should end. This must be the corner opposite to the corner specified for 'first coordinate'.

 

Output raster map:

Type a name for the output raster map that will contain the copied pixels.

If you selected an input map list, then you can type a name for the output map list; all raster maps in the output map list will obtain this name followed by _1, _2, _3, etc.

Description:

Optionally, type a description for the output map or the output map list. The description will appear in the status bar of the Main window when moving the mouse pointer over the map in a Catalog, and in the title bar of a map window when the output map is displayed. If no description is supplied, the output map or the output map list will use its own definition as description.

When you selected a single raster map as input:

  • and you click the Show button, the dependent output map will be defined, calculated and shown.
  • and you click the Define button, the dependent output map will only be defined; if necessary the map will be calculated later, for instance when it is opened to be displayed.

When you selected a map list as input:

  • and you click the Show button, the dependent output map list and its dependent raster maps will be defined and calculated. Then, the output map list will be shown.
  • and you click the Define button, the dependent output map list will only be defined. When you open the output map list later on, the map list will be calculated, which means that the output raster maps will be defined and calculated.

Furthermore, a new georeference is automatically created for the output map(s); this georeference obtains the same name as the output map or the same name as the output map list followed by _1.
