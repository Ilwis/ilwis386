Aggregate map

Command line (2)

To perform the Aggregate Map operation on all raster maps in a map list, type one of the following expressions on the command line of the Main window:

  

OUTMAPLIST

=

MapListApplic(InputMapListName, MapAggregateAggFnc(##, Groupfactor, group))

OUTMAPLIST

=

MapListApplic(InputMapListName, MapAggregateAggFnc(##, Groupfactor, nogroup))

OUTMAPLIST

=

MapListApplic(InputMapListName, MapAggregateAggFnc(##, Groupfactor, group, RowOffset, ColOffset))

OUTMAPLIST

=

MapListApplic(InputMapListName, MapAggregateAggFnc(##, Groupfactor, nogroup, RowOffset, ColOffset))

OUTMAPLIST

=

MapListApplic(InputMapListName, MapAggregateAggFnc(##, Groupfactor, group, NewGeoref))

OUTMAPLIST

=

MapListApplic(InputMapListName, MapAggregateAggFnc(##, Groupfactor, group, RowOffset, ColOffset, NewGeoref))

where:

OUTMAPLIST

is the name of the output map list map; all output raster maps will be created when the map list is opened, the raster maps will obtain the same name as the map list followed by _1, _2, _3, etc.

MapListApplic

is the command to start an operation on all raster maps in a map list.

InputMapListName

is the name of the input map list which contains all raster maps on which the operation should be performed.

MapAggregate

is the first part of the command to start the Aggregate Map operation.

AggFnc

see Aggregate map : command line (1)

##

is an obligatory parameter in map list expressions; it indicates the individual raster maps in the input map list.

Groupfactor

see Aggregate map : command line (1)

group

see Aggregate map : command line (1)

nogroup

see Aggregate map : command line (1)

RowOffset

see Aggregate map : command line (1)

ColOffset

see Aggregate map : command line (1)

NewGeoref

see Aggregate map : command line (1)

When the definition symbol = is used, a dependent output map list is created. When the map list is opened and thus calculated, the dependent output raster maps will be defined and calculated.

When the assignment symbol := is used, the output raster maps will be defined and calculated, then the dependency links of the output raster maps will be broken.
