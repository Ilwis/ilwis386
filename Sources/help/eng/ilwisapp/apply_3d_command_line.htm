Apply 3D

Command line

The Apply 3D operation can be directly executed by typing the following expression on the command line of the Main window:

  

OUTMAP

=

MapApply3D(InputMapName,Georef3D)

where:

OUTMAP

is the name of your output raster map.

MapApply3D

is the command to start the Apply3D operation.

InputMapName

is the name of your input raster map.

Georef3D

is the name of an existing georeference 3D.

When the definition symbol = is used, a dependent output map is created; when the assignment symbol := is used, the dependency link is immediately broken after the output map has been calculated.
