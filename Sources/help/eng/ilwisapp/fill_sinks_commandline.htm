Fill sinks

Command line

The Fill sinks operation can be directly executed by typing the following expression on the command line of the Main window:

  

OUTMAP

=

MapFillSinks(InputMapName)

where:

OUTMAP

is the name of the output raster map.

MapFillSinks

is the command to start the Fill Sinks operation.

InputMapName

is the name of the input raster map (value domain); the input raster map is expected to be a Digital Elevation Model (DEM).

When the definition symbol = is used, a dependent output map is created; when the assignment symbol := is used, the dependency link is immediately broken after the output map has been calculated.
